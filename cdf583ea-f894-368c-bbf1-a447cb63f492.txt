　　前脚刚离开拍卖行，慕清霄就察觉到身后有人跟踪，却没有在意，而是带着小医仙回到了先前的客栈。
<br/>　　当日拍卖会结束后，黑印拍卖行所竞拍的东西自然也传了出来，七品丹药，阴阳玄龙丹自然在黑角域掀起了一阵轩然大波。
<br/>　　得知阴阳玄龙丹，最后被天蛇府拍获后，黑角域中某些不安分的修炼者们也都蠢蠢欲动起来。
<br/>　　毕竟，七品丹药，斗尊的名头诱惑力不可谓不大。
<br/>　　竖日清晨，旭日东升。
<br/>　　慕清霄抱着怀中柔软的娇躯，习惯性的睁开双眼，望着怀中慵懒的小猫咪，捏了捏她的鼻子，道：“仙儿，该起床出门办事了。”
<br/>　　小巧的鼻子皱了皱，小医仙狭长的睫毛颤了颤，渐渐睁开美眸。
<br/>　　简单的洗漱过后，两人就套上黑袍，径直离开酒店，瞥了眼四周，慕清霄嘴角牵起一丝冷笑，身影眨眼间就自原地消失。
<br/>　　两人凭空消失在大街上，瞬间就有几名大斗师围了上来，面面相觑，眼中充满了困惑。
<br/>　　“真是见鬼了，怎么凭空消失了。”
<br/>　　“不妙，赶快回去禀报范管事。”
<br/>　　......
<br/>　　......
<br/>　　茂密的丛林中，两道身影，悄无声息的出现在数十米外的巨树上，利用繁茂的枝叶，巧妙的隐藏其上。
<br/>　　望着相隔数十米，靠在巨树旁休息的身影，正是刚出城没多久的范凌等人，几人聚拢在一起，脸上挂着阴恻恻的笑容。
<br/>　　慕清霄嘴角牵起一丝笑容，道：“仙儿，接下来的事情，肯定让你大开眼界。”
<br/>　　片刻后，范凌几人貌似商议结束，前后两批，相隔着百米距离，朝着黑印城的西面方向掠去。
<br/>　　小医仙靠在慕清霄怀中，美眸中有些不解，道：“他们怎么突然改变路线了？”
<br/>　　“当然是去干坏事了，我们也过去。”
<br/>　　慕清霄神秘一笑，在前方山坳处，隐藏着一支队伍，其中有一名四星斗皇，与先前拍卖行中传出的气息相同，绝对是同一人。
<br/>　　当他降落在一处视线开阔的山坡上，小医仙也瞧见了隐藏在四周的身影，捂嘴惊呼道：“他们想干嘛？”
<br/>　　“其中一名是四星斗皇，估计是血宗的宗主，两名斗灵全都埋伏在这里，你说他们要干嘛？”
<br/>　　美眸闪烁的同时，小医仙心中不由的惊骇，道：“难道，他们真要抢夺拍卖物品？”
<br/>　　“这就是现实，七品丹药，阴阳玄龙丹，诱惑实在太大，而且这种拦路抢劫的事情，在黑角域屡见不鲜。”
<br/>　　说罢，慕清霄又道：“先前，我们刚出拍卖行的时候，可是有六名大斗师一路跟着我们到客栈。”
<br/>　　闻言，小医仙不禁有些后怕，同时心境也有了一番变化，望着下面的身影，目光变得冷漠无比。
<br/>　　此时，范凌已经进入埋伏范围，在平静的山坳中，隐藏的都是血宗之人，而其中的四星斗皇，自然就是血宗的宗主了。
<br/>　　在这簇小森林东面，是一道蜿蜒到尽头的小道，目光沿着西面扫过，隐隐能看见黑印城模糊的轮廓。
<br/>　　从地形上来看，此处是黑印城西面一条必经之路，而且容易隐藏，绝对是杀人越货的最佳地点。
<br/>　　诡异的气氛缭绕在周围，连鸟鸣声都没有，显得有些诡异，片刻后，耳中传来一阵马蹄声，慕清霄抬头望去。
<br/>　　只见在远处道路上，一群骑着快马的队伍，正追星赶月一般向小路另一边疾驰而去，沿途带起一阵尘埃。
<br/>　　片刻后，一支队伍映入眼中，领头的是一袭绿色裙袍的女子，正是天蛇府的青长老。
<br/>　　看到女子的瞬间，小医仙娇躯微颤了一下，已经确信，血宗隐藏在这里，就是为了劫杀天蛇府，目的自然就是阴阳玄龙丹。
<br/>　　“天蛇府在大陆上颇有名气，难道血宗是打算与天蛇府开战吗？”
<br/>　　“血宗宗主亲临，天蛇府队伍中连反抗的机会都没有，将这里的人全杀了，抹除痕迹，谁敢说是他们杀的？”
<br/>　　片刻后，只听见青长老骤然色变，厉声喝道：“都小心，其中有埋伏！”
<br/>　　“不愧是天蛇府的长老，这斗气凝蛇的手段，果然出神入化啊。”青长老厉喝声刚落，森林中就传出一道狂笑声。
<br/>　　随后一道雄浑气息冲天而起，血红影子自森林中爆射而出，最后稳稳的落在树顶上，泛着杀意的双瞳，盯着森林外的青长老一行人。
<br/>　　“范崂！”
<br/>　　见身披血色红袍的中年男子，青长老脸色骤变，旋即色厉内茬的喝到：“你这是何意？难道是要与我天蛇府开战？”
<br/>　　“哈哈，青长老说笑了，我只是对你手中的阴阳玄龙丹感兴趣。”虽是在笑，但范崂脸庞上挂的却是冷笑。
<br/>　　“快撤，所有人立刻撤离，只要有一人逃离，立刻向府主大人禀报此事！”
<br/>　　听到对方的目的是阴阳玄龙丹，青长老就知道事情不妙，根本没有商谈的余地，后者也绝不可能放过他们。
<br/>　　厉喝过后，青长老当下不敢迟疑，脚掌一踏马背，身影率先化作一道影子，向路两旁的森林中窜去。
<br/>　　就在天蛇府准备撤离之际，森林中骤然响起一片破风声，无数泛着寒芒的箭矢，铺天盖地的飞射而出。
<br/>　　在这般无间隙的箭雨下，饶是青长老都不得不减速退避，身影接连闪动，不断的躲避箭雨。
<br/>　　道路中央，原本四散开来的天蛇府弟子，被逼得再度聚拢在一起。
<br/>　　箭雨过后，四面八方涌出近百名身着红袍，手持血刀的血宗弟子，神色木然，犹如傀儡，但他们眼中，却充斥着阴森与暴戾。
<br/>　　“青长老，交出阴阳玄龙丹，本宗饶你性命，否则，死！”
<br/>　　站在树尖，范崂身形鬼魅一般，瞬间出现在包围网上空，阴森的厉喝声，在道路上空徘徊不散。
<br/>　　“杀！”
<br/>　　面对于范崂的阴森厉喝，青长老没有丝毫迟疑，脸色阴沉的爆喝一声，雄浑的斗气自体内狂涌而出，强横的气场，直接将周围的树木震的倒射而出。
<br/>　　青长老虽是斗王级别，又怎会是斗皇级别，范崂的对手，几回合就败下阵来，整条道路上，布满了血宗与天蛇府弟子的尸体。

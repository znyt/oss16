　　蹬蹬蹬！
<br/>　　在那股强悍的力量压迫之下，华云生心头竟然莫名其妙的生出一股怯意，不由得一连倒退四五步，方才站稳身形。
<br/>　　“不是都说，华夏武林被神秘强者重创，高手都已经死伤殆尽了吗？怎么竟然还有高手？而且还如此的年轻！”
<br/>　　华云生又惊又恐，望着那角落之中，依旧在缓缓翻看书籍的少年身影，顿时满脸的骇然之色。
<br/>　　那神秘少年从始至终，连眼皮子都没有抬一下，华云生原本以为他只是一个反应有些迟钝的普通学生，而且并没有从对方身上察觉到任何气息。
<br/>　　然而，他却怎么也没有想到，对方竟然是一个实力不弱于他的绝世高手！
<br/>　　不但将唐威从他的手中救下，甚至他都丝毫没有察觉到，对方是如此出手的？
<br/>　　简直太恐怖了！
<br/>　　“你——你到底是什么人？”
<br/>　　原本自信满满的华云生，一时间也有些惊疑不定。
<br/>　　叶尘却依旧在缓缓翻看着眼前书籍，始终没有抬头看华云生一眼，淡淡道：
<br/>　　“我，是你惹不起的人！不想死，快点滚！”
<br/>　　轰！
<br/>　　叶尘这话一出口，华云生顿时差点肺都要气炸了，
<br/>　　他活了这么大岁数，无论走到哪里，都被人当做神仙一般看待，这还是第一次被人如此轻视羞辱。
<br/>　　纵然对方的修为深不可测，此时愤怒之下，他也顾不得了。
<br/>　　“好大的口气！老夫倒要看看，你什么本事，敢口出如此狂言！”
<br/>　　华云生气极之下，再也按捺不住，当即大喝一声，再次朝叶尘扑了过来。
<br/>　　轰！
<br/>　　华云生不愧是神境强者，这遥空一拳砸出，真元居然凝聚成实质，犹如炮弹一般，直奔叶尘爆射而去！
<br/>　　这华云飞不但实力强悍，对真元的控制，也达到了登峰造极的地步，这一拳只针对叶尘，对周围建筑却丝毫无伤。
<br/>　　可惜，对方依旧连眼皮都没有抬，只是随手一拍，
<br/>　　彭！
<br/>　　华云飞这不弱于炮弹的一拳，瞬间被叶尘化解于无形！
<br/>　　“什么！！”
<br/>　　看到这一幕，不但华云飞彻底惊呆，其余众人同样个个一脸骇然。
<br/>　　尤其是杨之凡，他本以为有华云生出手，叶尘肯定不是对手，却万万没有想到，两人交手之下，叶尘居然稳占上风！
<br/>　　“不！这不可能！你到底是谁？怎么会这么强？！！”
<br/>　　华云飞见叶尘轻松接下自己这一拳，顿时都开始有些怀疑人生了。
<br/>　　后方的杨之凡和杨国强父子，同样脸色大变，
<br/>　　“这不可能！华老乃是神境强者，怎么会不是这小子的对手？”
<br/>　　杨之凡又惊又怒，实在无法接受这一事实。
<br/>　　而杨国强则眉头紧皱，
<br/>　　也不知道为何，他总感觉对方的声音，似乎有些熟悉，可是一时半会，却又想不起到底是谁？
<br/>　　就在三人惊骇之时，那道淡漠的声音，陡然再次响起，
<br/>　　“你也来接我一掌试试！”
<br/>　　那课本后面，一只如美玉般完美的修长手掌，缓缓朝华云生拍出。
<br/>　　这一掌，轻飘飘的，似乎没有任何力道。
<br/>　　然而，华云生却脸色狂变！
<br/>　　“你竟然是仙人之境……难道你就是……”
<br/>　　轰！
<br/>　　到了这个时候，华云生总算察觉到了，眼前这个神秘少年，竟然是仙人之境，在结合之前杨之凡的话，已经大致猜到了叶尘的身份。
<br/>　　可惜还没等他的话说完，一股无比霸道的力量，已经钻入他的身体之中。
<br/>　　华云生浑身一颤，如遭雷击，继而两眼瞪得老大，整个人伫立在教室中央的位置，一动也不动了。
<br/>　　过了片刻，华云生依旧站在那里，犹如雕像一般，不过从始至终也没有倒退一步。
<br/>　　“哈哈哈！”
<br/>　　后方轮椅上的杨之凡，先是微微愣神，继而得意的哈哈大笑，
<br/>　　“叶尘呀叶尘，你也不过如此！我就说嘛，华老乃是神境强者，尤其是你区区一个毛头小子……”
<br/>　　杨之凡的话还没有说完，
<br/>　　“彭！”
<br/>　　一直伫立在那里的华云生，突然毫无预兆的在爆炸了！
<br/>　　是的，整个人就这么突然爆了，化成了一团血雾，继而彻底消失不见……
<br/>　　“啊！！”
<br/>　　杨家的所有人看到这一幕，一个个吓得魂都要飞了，顿时连连倒退。
<br/>　　连神境强者华云生，都被对方一掌打爆，更遑论是他们？
<br/>　　“不！这不可能！不可能！”
<br/>　　杨之凡也吓坏了，一边不甘的吼叫着，一边想要拼命的从轮椅上站起来，可惜他双腿被废，身体才刚刚离开轮椅，便直接栽倒在了地上。
<br/>　　杨之凡双手并用，好像往后面去爬，唐威一个健步上前，一把将他拎了起来，生生扯了回去。
<br/>　　“爸！救我！快救我！”
<br/>　　到了这个时候，杨之凡终于慌了，当即对着杨国强大声呼救起来。
<br/>　　杨国强同样脸色煞白无比，不过眼看着儿子命悬一线，只好硬着头皮走上前来，冲着叶尘拱手道：
<br/>　　“阁下修为惊人，的确令人钦佩！我儿鲁莽，冲撞了阁下，我愿意替他向您道歉，能否看在我杨家的面子，饶他一命，我杨家愿意承受任何代价！”
<br/>　　叶尘冷冷一笑，一边继续缓缓翻看着手中的课本，淡淡道：
<br/>　　“杨家的面子？很值钱吗？”
<br/>　　杨国强顿时脸色越发惨白，
<br/>　　的确，对方可是连神境强者都能随手打爆的存在，又岂会在意区区一个杨家？
<br/>　　无奈之下，杨国强只好一咬牙，接着又道：
<br/>　　“我杨家的确算不上什么，但阁下可知道，我杨家的背后是何人？”
<br/>　　叶尘听到这话，不由得嘴角微微一扬，露出一抹戏谑之色，
<br/>　　“哦？说来听听！”
<br/>　　杨国强深吸了一口气，昂然道：
<br/>　　“我杨家的背后，乃是云州叶仙人！阁下可以不把我杨家放在眼里，难道连云州叶仙人也不惧吗？”
<br/>　　若非逼不得已，杨国强自然不敢随便扯出叶仙人的大旗，可是眼下为了救自己的儿子，他也管不了那么多了。
<br/>　　唐威三人听到这话，顿时全都脸色大变，忍不住凑到叶尘跟前，小声道：
<br/>　　“尘哥，看来传言是真的，杨家背后的靠山，果然是那位叶仙人，这可如何是好？”
<br/>　　叶尘顿时笑了，继而不由得摇了摇头，
<br/>　　他本不想暴露自己的身份，没想到这个杨国强，竟然敢打着自己的旗号，来威慑自己。
<br/>　　“杨国强，睁大你的狗眼，好好看看我是谁！”
<br/>　　叶尘放下面前的书本，缓缓站起身来。

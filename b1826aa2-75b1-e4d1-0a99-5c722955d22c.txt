　　这一尊人物，不止是样貌跟东宫绝相似，此人也是站在了荒古门无数弟子的最前方，为这一次荒古门一方的领头人物。
<br/>　　看样子，便是跟东宫绝，脱不开关系。
<br/>　　东宫世家，是荒古门之中的第一大家族，也是相当于荒古门的皇族。
<br/>　　东宫世家的人，成为荒古门的领头人物，并不奇怪。
<br/>　　这一次剿灭剑道门，七大宗门可是动了真正的杀心，所拍的人，可都是七大宗门之中真正的强者。
<br/>　　站在此人身边的人物，许多人的一身实力，也不在此人之下。
<br/>　　“这剑道门，不值一提，真正要死的，是那楚风眠，这楚风眠，只怕不止是得到了曾经剑道门的剑术，那么简单，许多古老的隐秘，此人都知道，此人才是我们七大宗门，真正的心腹大患。”
<br/>　　站在那东宫世家强者身边，一尊老者，冷笑连连道。
<br/>　　“东宫厉，我知道这小子杀了你们东宫世家的人，等攻破了剑道门，那小子的命，我可以让给你，只要你放弃那小子身上的机缘便是。”
<br/>　　这一尊老者，手中握着一把灵剑，这一把灵剑，看似腐朽，上面却蕴含这极为恐怖的剑意。
<br/>　　这老者，也看似苍老无比，如同迟暮老者，可没有人敢小看这老者，因为世人都知晓，这老者的强大。
<br/>　　上剑宗，枯叶。
<br/>　　寻常剑修，都是一股惊天剑意，一眼便是足以看的出来，剑修所修的，便是剑锋所致，一切斩灭。
<br/>　　可这枯叶却截然不同，可他却依然是上剑宗之中，最为恐怖的太上长老之一。
<br/>　　这种老古董，可都是经历过万年之前大战的人物。
<br/>　　平日里都在宗门圣地之中沉睡，在寻找冲击半步天人境的机会。
<br/>　　可今日却都是纷纷出关，为了围攻剑道门而来。
<br/>　　“枯叶，那东宫绝，本来就是我们东宫世家的废物，要不是万年之前围攻剑道门的时候出了点力，根本不值得培养，他死在楚风眠那小子的手中，也就死了，算不得什么！”
<br/>　　那东宫厉冷哼一声开口道。
<br/>　　“至于那小子身上的机缘，我们七大宗门平分，我当然要分一杯羹，怎么你想独战？”
<br/>　　“不用吵了，从这剑道门之中传递出来的气息来看，楚风眠那小子只怕是在冲击境界，让此人突破境界，可就麻烦了！”
<br/>　　又一尊身影，从虚空之中走了出来，这是一尊中年男子，从他从虚空之中走出来的一刻，一颗颗星辰，都闪耀在他的周围。
<br/>　　星象之道，已到大成。
<br/>　　“怕什么，那小子之前不过是区区八劫古帝的境界，哪怕是突破了境界，才不过是一尊九劫古帝，我们联手之下，可还怕一尊九劫古帝？”
<br/>　　听到这一尊身影的话，东宫厉不屑的说道，同时他的目光也看向了走出来的这一尊人物，开口道。
<br/>　　“仲皇，天枢殿，这一次怎么会是你带队？星家的人呢？怎么攻打剑道门这么重要的事，星家打算不出手？”
<br/>　　天枢殿的第一家族，便是星家，星家在天枢殿之中的地位，等于皇族。
<br/>　　可这一次天枢殿的领头之人，居然不是星家之人，其中更是没有星家武者，这让那东宫厉，都是眉头一皱。
<br/>　　“星家的那一群老古董，不知道是听到了什么消息，不愿意出手，不过由我在，杀了那小子，不成问题。”
<br/>　　仲皇冷哼一声，语气之中也有些不满的说道。
<br/>　　“不过你们可不要小看那小子，此人在八劫古帝的时候，就可以杀了东宫绝，据说盘石，屠人间，纯剑长老，都死在了此人的手中。”
<br/>　　“八劫古帝的境界，就有着如此的实力，这小子一旦是踏入九劫古帝的境界，将会更加恐怖。”
<br/>　　听到仲皇的话，那东宫厉这一次倒是没有出言反驳，而是眼神之中也露出几分凝重。
<br/>　　他虽然狂傲，可仲皇的话，他却不得不听。
<br/>　　楚风眠以八劫古帝的境界，就足以轰杀盘石，屠人间这等九劫古帝之中堪称巅峰无敌的人物。
<br/>　　这种战绩，已经是太过惊世骇俗了。
<br/>　　东宫厉，虽然比起他们要强横几分，可毕竟都是九劫古帝，想要杀了盘石，对他都并不容易。
<br/>　　“先打破了这剑道门的门户再说！”
<br/>　　东宫厉看向这剑道门门户。
<br/>　　在无数七大宗门弟子联手轰击之下，这剑道门的门户之上，已经是布满了无数裂痕。
<br/>　　突然之间，由他率先出手，荒古大道在东宫厉的脚下浮现而出，一股股的荒古之力凝聚在他的拳风之上，向着剑道门门户，发起了猛攻。
<br/>　　轰！
<br/>　　东宫厉只是第一个出手的人。
<br/>　　在听到了仲皇的话后，在场七大宗门的强者，也都不敢小看楚风眠。
<br/>　　尤其是荒古门，上剑宗，圣王宗，六道魔门一方的强者。
<br/>　　他们更是吃过楚风眠的亏，有着强者陨落在了楚风眠的手中。
<br/>　　随着东宫厉出手，他们也已经是纷纷凝聚灵力，同时轰击在剑道门门户之上。
<br/>　　在场一共十五位堪称帝尊无敌的强者，同时凝聚灵力，轰击在剑道门门户之上。
<br/>　　轰隆隆！
<br/>　　这剑道门门户之上，在这一股力量之下，被突然打破。
<br/>　　哪怕是楚风眠重塑的剑道门门户，也不可能抵挡得住十五位帝尊无敌强者的联手轰击。
<br/>　　更何况在这十五位帝尊无敌强者的背后，还有着足足数百位古帝相助。
<br/>　　这爆发出来的力量，哪怕是一尊半步天人境的强者，位于在这无数强者联手的面前，都要被无情轰杀。
<br/>　　随着剑道门门户被破，这剑道门的护山大阵，黑玄剑阵，都被毁灭。
<br/>　　剑道门，无数宫殿，云羽天宫，映照在他们的目光之中。
<br/>　　“云羽天宫！这是曾经称霸九域的势力，居然是被楚风眠那小子得到了！”
<br/>　　“这阵法，居然是千把天级灵剑，凝聚在一起，好大的手笔！”
<br/>　　“这剑道门的财富，绝不会少，杀进去，将这些剑道门弟子都杀了！这些财富都属于我们！”
<br/>　　剑道门门户被破，无数七大宗门弟子，都如同饿狼一般，冲了进去，疯狂的要掠夺起来。

　　察觉对方的气息，战狂瞬间面色一变。
<br/>　　千年僵尸，道行已经非常恐怖了，只要有所机遇是可以变成旱魃的，旱魃就是一种灾难，有旱魃的地方，就会大旱，水井里没有水，天上不下雨，可以说是影响到天地法则的逆天僵尸王的存在了。
<br/>　　“这小子身边怎可能又这样的存在？这小子到底是谁？”
<br/>　　战狂心中居然出现了一抹恐惧。
<br/>　　他甚至都开始怀疑，林峰是不是也是异能者了。
<br/>　　跑！
<br/>　　此时此刻。
<br/>　　战狂心中直接出现了一个念头，那就是掉头就跑。
<br/>　　然而千年僵尸可不会让他跑，一蹦过去就是追上了他。
<br/>　　“滚开！”
<br/>　　战狂怒吼一声，他手中的战刀居然刷的一下冒出了一层火焰。
<br/>　　刷。
<br/>　　这一刀落在千年僵尸身上顿时把对方给击退了。
<br/>　　战狂松了口气就是要跑。
<br/>　　“这小子刚才居然没有施展全力？还有他的那个刀居然会自己冒火？这家伙到底什么异能？几个异能？？”
<br/>　　林峰心中一惊，不过他刚才可没有看戏，而是同样追了上去，抬手就是几枚飞镖。
<br/>　　叮叮叮
<br/>　　战狂就好像背后长了眼睛一样直接把飞镖全部弹开。
<br/>　　而这时，林峰的飞镖也算是彻底的消耗完毕了。
<br/>　　林峰直接又丢出去几枚麻痹飞镖，对方也全都弹开。根本无法碰到对方的本体。
<br/>　　不过因为追风靴的缘故，林峰和战狂的距离再不断的拉近。
<br/>　　“你小子的速度这么快？你也是异能者？？”
<br/>　　战狂彻底的震惊了。他自然不知道林峰脚上穿的那双鞋子是他捡回来的灵靴可以增加速度的存在。
<br/>　　刷。
<br/>　　而这时。
<br/>　　之前被击退的千年僵尸再次一蹦一跳的追了上去。
<br/>　　“滚开！”
<br/>　　战狂怒吼着一刀又是劈了过去。而也就在这时，林峰也是掏出了一张符箓。
<br/>　　“就你会玩火？火焰符箓！”
<br/>　　刷。
<br/>　　瞬间一道火龙卷从符箓之中喷射出来。
<br/>　　“什么？你到底什么异能！妈的！！”
<br/>　　战狂显然没有想的林峰还有这么一手，他立即面色大变就要闪躲。林峰吃惊对方的路数，这战狂也同样震惊林峰的手段，双方都是拿出了不同的招式套路。
<br/>　　“跑？我看你怎么跑。”
<br/>　　林峰拿出沙漠之鹰砰砰砰的就是连射。
<br/>　　这战狂非常的灵活居然可以躲子弹。
<br/>　　但是躲得了一颗躲不了连射。
<br/>　　林峰几十颗子弹全部都射了出去，此时此刻，也根本不是去顾忌消耗的时候了。
<br/>　　火焰符箓加上子弹的双重攻击，顿时让这战狂无处可躲。
<br/>　　“啊啊啊啊。”
<br/>　　对方被火焰符箓吞没，只剩下了一声惨叫，很快，就变成了一个烧焦了的黑炭。
<br/>　　林峰松了口气，擦了擦头上的汗水。
<br/>　　“还真是够难缠的。”
<br/>　　战狂虽然死了。
<br/>　　但是林峰也算是底牌全出。
<br/>　　飞镖射完了。
<br/>　　沙漠之鹰子弹没了。
<br/>　　火焰符箓也因为能量耗尽，全部消耗完毕了。就是武士刀都裂了。
<br/>　　“回来吧。”
<br/>　　林峰把千年僵尸收入物品栏，来到这战狂尸体边。
<br/>　　对方身上东西都烧没了，唯独那把刀还依然完好如初。
<br/>　　“好在有把刀可以补偿我一下。”
<br/>　　林峰捡起这把刀，比武士刀要重。
<br/>　　“斩灵刀？”
<br/>　　刀身上刻着三个字。
<br/>　　“不错，以后就是我的了。”
<br/>　　林峰拿着手中挥了几下，手感很不错。
<br/>　　而就在这时。
<br/>　　一道流光忽然从战狂的尸体上飞出落在了林峰身上。
<br/>　　“什么东西？”
<br/>　　林峰根本无法反应顿时被流光击中。
<br/>　　“小子，你等着吧，要不了多久，我战门的师兄弟就会替我报仇的啊！中了我战门的追魂索命蛊，你逃到天涯海角都没用！！哈哈，我在下面等着你！！”
<br/>　　耳畔响起战狂那阴冷怨毒的声音。
<br/>　　“追魂索命蛊？”
<br/>　　林峰面色变了，他万万没想到还有这么个后手。
<br/>　　“该死。”
<br/>　　林峰心情非常的不好，这个战狂虽然死了，但这家伙只是个外门弟子，那么肯定门派内有更强的。
<br/>　　“算了，木已成舟，再怎么样担心也是无用，兵来将挡水来土掩，我能够杀的了这个战狂，就能杀的了其余人！不过关键就是空投！这个东西是一切的来源！要想变强，就靠空投！”
<br/>　　林峰深吸口气，他一路狂奔回到了学校。
<br/>　　一夜无话。
<br/>　　翌日。
<br/>　　空投再次刷新。
<br/>　　林峰出去找空投，苏云老师陈勾吴三他们则继续出去猎杀丧尸，顺便找物资。
<br/>　　几个人的实力也提升的非常迅速，陈勾四阶，李广四阶，苏云也是四阶，唯独吴三差了点，目前只是三阶。
<br/>　　然而物资越来越难找了，尤其是一些药剂。。
<br/>　　在末世，许多人不是死于丧尸，而是死于一些疾病，寻常的感冒咳嗽，都会变成的非常严重，因为没有药物的治疗，往往都最后演变成了死亡。
<br/>　　“有没有好心人给我孩子一盒退烧药啊。”
<br/>　　“我给你们磕头了，给我一点吃的吧。”
<br/>　　“我两天没喝水了，求求你们，只要一小瓶盖的水就可以了。”
<br/>　　一些抱着希望来找物资的人，当看到超市内已经空空如也的东西，全都剩下了绝望。
<br/>　　“没有了，什么都没了，我已经找了十七家超市，什么都没了。”
<br/>　　“死定了，我们就算不被丧尸吃了，也要活生生渴死啊。”
<br/>　　“不能在坐以待毙了，城市里呆不下去，这么多天也没有看到军警，我们去郊区去农村，那些地方人少，丧尸少，而且大山里还有食物，怎么样都比呆在这里好。”
<br/>　　“没错，去农村！去乡下地方！”
<br/>　　林峰从旁边路过去找空投。
<br/>　　“去农村，的确是个不错的选择。”
<br/>　　林峰暗自点头，实际上他一开始也有这个想法。
<br/>　　农村的确是非常适合避难的地方。
<br/>　　首先他人没有城市这么密集，其次就是他不少地方都是自供自足的，蔬菜田地里种，鸡鸭鱼肉都是自己养的。
<br/>　　比起城市，农村显然值得去试一试的。
<br/>　　不过林峰不会去农村里当野人，他要去曙光基地。
<br/>　　半天下来。
<br/>　　今天刷新的五个空投全部拿到了手中。
<br/>　　总共是五瓶农夫山泉，五包饼干，一罐八宝粥外加一罐猪肉罐头，以及一包辣条。
<br/>　　“快看，这个家伙在吃辣条。”
<br/>　　“哇，这都吃得起。”
<br/>　　“好羡慕，我也想吃。看得我口水都涌出来了。”
<br/>　　“别做梦了。这年头辣条可是奢侈品了。”
<br/>　　“厉害，这家伙肯定是大佬，辣条都吃的起。”
<br/>　　林峰吃着辣条走在外面不少幸存者看到了他都羡慕嫉妒恨，一群人更是舔嘴巴。口水都留下来了。

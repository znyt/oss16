　　第365章
<br/>　　崔弄海觉得卸掉程大雷的剑后，程大雷对自己便没有威胁。很显然他搞错了这件事，没理由说会用剑的程大雷，在力气和速度上便不如他。
<br/>　　程大雷活动着手腕，向崔弄海勾了勾手指，做出一个挑衅的姿势。
<br/>　　崔弄海发出桀桀的怪笑，在稍犹豫片刻后，瞬间向程大雷扑去。二人几乎是同时撞向对方，拳对拳，脚碰脚，身体撞击向身体。
<br/>　　崔弄海没有任何技能，这是程大雷早就发现的。而打了这么久，程大雷甚至怀疑，崔弄海没有接受过任何武学训练。但他的速度足够快，力气足够大，而在战斗中掌握这两点，几乎已经立于不败之地。
<br/>　　更何况，他的凶残是人所不及。
<br/>　　骨头，肌肉，甚至牙齿都能成为他的兵器。他以躯体碰撞，牙齿撕咬，利爪袭击……人和人打架很少会如此，狗咬狗却是常见。
<br/>　　如果换一个其他人，即便强过崔弄海，怕单单这副气势，就会令他气馁。
<br/>　　不过好在，程大雷也是一个狠人。
<br/>　　程大雷也没有任何拳脚上的训练，他只是力气够大，速度够快，脑海中有太多对敌经验。心中的愤怒被勾惹得无法遏制，无比渴望的想摧毁眼前这个人，不管是身体还是灵魂。
<br/>　　剑，乃是堂堂之器，讲究体面，讲究威严。但当弃掉剑，以拳头战斗时，便无所谓体面和威严。
<br/>　　心中那头猛虎被程大雷释放出来。
<br/>　　他是第一次不再去遏制这种力量，双脚立地，肌肉绷紧，眼睛布满血丝，喉咙发出古怪的声音，像从地狱中爬出来的魔。
<br/>　　樊梨花是这场战斗唯一的观众，她正在给自己止血，眼神却又不肯错过任何一个场景。
<br/>　　此时，她恍然意识到一个事实：崔弄海这怪人定然是可怕的，可很显然，程大雷才是更加可怕的存在。
<br/>　　程大雷已完全压制住对方，脚步向前顶，膝击对方裆部，肘击对方小腹，同时身体也承受着对方的攻击，步步向前压，绝对不让对方离开他三拳的距离。
<br/>　　程大雷抓住了他的耳朵，渗血的嘴角发出一抹冷笑，猛地发力，活生生的撕扯下对方半边耳朵。
<br/>　　“啊！”
<br/>　　崔弄海发出一声杀猪般的惨叫，这是裹夹着恐惧的声音。紧跟着，程大雷拳头重重捶在对方耳朵上，双风贯耳，崔弄海两眼泛白，直接晕了过去。
<br/>　　程大雷倒是没有着急杀他，这有些太便宜他了。捡起剑挑了他的手筋脚筋，崔弄海吃痛站起来，以恶毒的眼神盯着程大雷。
<br/>　　可手筋脚筋被挑断的他，此刻已经成了一个废人，也只能用恶毒的眼神看看程大雷。
<br/>　　程大雷还剑入鞘，走向樊梨花，微微蹲下身，手搁在樊梨花肩膀上。
<br/>　　“你没事吧？”
<br/>　　樊梨花激灵一下，下意识就想躲避，只因为程大雷如今的样子实在恐怖。
<br/>　　他浑身浴血，脸上挂着青肿，伤口还在往外浴血，浑身上下散发着一种血煞味道。
<br/>　　樊梨花像是第一次见到程大雷，有些不敢相认。
<br/>　　忽然，程大雷头一歪，身体跌进樊梨花怀里。
<br/>　　樊梨花一怔，发现程大雷已经昏了过去，乱糟糟的头发压着自己手掌。樊梨花有些不知该如何是好，她本来就伤得不轻，身上的伤还是其次，关键是这一夜的经历对心灵的冲击实在太大了。
<br/>　　她倒在地上，还正等着求安慰，怎么程大雷就噗通一声，跌进自己怀里晕了过去。
<br/>　　本来受伤的樊梨花是最弱的一个，可如今崔弄海手筋脚筋被挑断，程大雷力竭昏迷，樊梨花反而成了此间最清醒的一个。
<br/>　　但她也失去了行动能力，既不能推开程大雷，也无法带着他回琴川关，只能撑在这里，以期待琴川关的人能找过来。
<br/>　　程大雷派出许多搜查队，日夜在琴川关附近搜寻，只不过此地距离琴川关已经远了，不知他们什么时候能找到这里。
<br/>　　及到夜尽天明，远处过来一队人马，领队的是高飞豹。看到这一幕的他大惊失色，忙带人奔了过来。
<br/>　　“大当家，你没事吧？”高飞豹。
<br/>　　程大雷此刻已经清醒过来，却依旧是手足无力，他摆摆手道：“没什么大事，把人带回去吧。”
<br/>　　高飞豹目光落在崔弄海身上，试探着问：“大当家，他就是我们要抓的那个人……被大当家抓到了。”
<br/>　　程大雷点点头。
<br/>　　“让我杀了他！”高飞豹大声道。
<br/>　　“瞎激动什么！”程大雷道：“现在杀了他岂不是太便宜他了，带回山寨处理。”
<br/>　　“那就让他多活一回儿，我真恨不得现在就杀了他！”
<br/>　　“你废话说完没完，能不能先送我回去，你想害死我么。”
<br/>　　“大当家你不说没事么？”高飞豹一头雾水。
<br/>　　“我特么这个样子像没事么。”
<br/>　　“哦哦哦。”
<br/>　　高飞豹恍然大悟，直接让人回城去拉一辆马车，把程大雷和樊梨花抬进车里，现在不管是程大雷还是樊梨花都没有骑马的能力。
<br/>　　至于崔弄海，他被用麻绳绑了，大棍一串，抬进了琴川关。
<br/>　　来来往往，消息已经传到琴川关，大家都说程大雷已生擒那恶人，即将押着回来。
<br/>　　这件事对高飞豹这些人而言，不算小事，却也算不上什么太大的事。真正激动的是迁到琴川关附近的流民。
<br/>　　那十三人的惨状大多人都见过，如一片阴云笼罩在众人头顶，既然来到这里，大多数人还是想安稳过日子的。大家担心，那些残酷的景象会不会发生在自己身上。甚至有人传言，人就是琴川关杀的，为的就是惩罚那些不愿缴税的人。
<br/>　　今日程大雷生擒崔弄海，却是可以令谣言烟消云散，大家都围在城门口，堪称夹道欢迎。
<br/>　　手中的鸡蛋、菜叶、石头等丢向崔弄海，如果不是被拦着，怕崔弄海已经被活活砸死。
<br/>　　“诸位，诸位！”吴用被推了出来，他高声道：“此獠已经被生擒，是大当家亲手抓来的。等大当家醒来，必会亲手制裁他。我们琴川关从不畏惧任何人，我们只让别人畏惧！”
<br/>　　
<br/>(本章完)

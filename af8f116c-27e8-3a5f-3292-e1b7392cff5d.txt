　　第32章
<br/>　　空气好像突然安静，枯叶被风吹落，砸在地上的声音都无比清晰。
<br/>　　程大雷往后退了一步，自己脸皮都有些发烫，想也知道，苏樱现在一张脸肯定已经是血色。
<br/>　　却发现苏樱眨了眨眼睛，困惑问：“什么意思？”
<br/>　　“呃……”
<br/>　　程大雷无语，头顶好像有乌鸦飞过，发出嘲讽的笑声。程大雷觉得自己说出一句很唐突，很冒犯的话，却忽略了一件事。苏樱这个年纪，对男女之间的事可能还停留着亲亲嘴就能怀孕的程度。
<br/>　　他无奈的挥挥手：“走吧。”
<br/>　　苏樱困惑的颦起眉头，依旧想不明白程大雷这句话是什么意思：“等我回去后问问别人。”
<br/>　　“不要！”程大雷崩溃的伸出手，说：“以后如果有机会，我亲自讲给你听。”
<br/>　　扬起的马鞭落在马身上，哒哒马蹄声响起，马车在程大雷的视线中渐渐消失。
<br/>　　程大雷心里是十分不舍的，他其实谈不上对苏樱有些喜欢，可人总有好美恶丑之心，何况是苏樱这种水灵的姑娘，总是有些占有欲在作祟。程大雷自然能用暴力手段留下对方，孩子生出几窝也不算什么问题。
<br/>　　可苏樱今年不过才17岁，当然，在这个时代女子十三四岁成婚是常态，十七岁都能够得上大龄未婚女青年。可在程大雷眼里，她不过是个十七岁的孩子，放在自己前世，可能高中还没毕业。程大雷再低格，也不能强迫一个17岁的孩子。
<br/>　　想到这里，程大雷心中突然咯噔一下，意识到自己想错了什么。
<br/>　　对了，苏樱才十七岁，她没有那么多见识，也不该这么冷漠，少女情怀总是诗……也可以是湿，古人说得好，哪个少女不怀春，自己说了那么多的话，做了那么多的事，虽然有些看上去很蠢，很可笑。但苏樱的心难道真的是一道深渊，投入一座大山都没有回音嘛？
<br/>　　自己可能被骗了。
<br/>　　嘟，完成抢夺压寨夫人第二阶段，获得奖励：重弩1 。
<br/>　　嘟，条件不足，无法触发第三阶段任务。
<br/>　　长长山道之上，行着一辆马车。车厢内的小蝶擦掉额头汗珠：“小姐，真没想到他竟然会放我们走，可吓死我了。”
<br/>　　苏樱眉头微颦，似乎想着什么心事。半晌她才问：“小蝶，假若一个男人对你说他……硬了，你知道什么意思么？”
<br/>　　小蝶睁大眼睛：“心肠硬？”
<br/>　　……
<br/>　　送走苏樱后，程大雷整理心情。先在蛤蟆寨堆积杂物的仓库中，找到那架系统奖励的重弩。
<br/>　　蛤蟆寨重建后，原先的杂物都堆在仓库中，程大雷确认过，当时收拾东西时，绝对没有看见这架重弩。可如今它就这样无端端出现，而且还配了十枚重箭。
<br/>　　果然是重弩，弩身长有两米，宽有三米，程大雷确认过，整个蛤蟆寨，能拉开这张重弩的，除了秦蛮，就是自己。秦蛮是信手拈来，而程大雷是使劲全身气力，才勉强将它拉开。至于徐神机和徐灵儿，根本连一丝丝都拉不动。
<br/>　　程大雷决定将这张重弩放在瞭望台上，算是给山寨添一防御设施。不过，程大雷明白，这架重弩的真正意义，就是给了自己复制生产的可能。可惜现在山寨没有木匠，这种事想也白想。
<br/>　　安置好重弩，程大雷开始研究辣椒的培育工作。有了辣椒种子，如何种还是个问题。程大雷记得辣椒好像是苗植，不是直接把种子埋在土中，辣椒就蹭蹭的往外长。程大雷没有种地的经验，不过他曾经在宿舍种过小盆栽，这倒不是太大的问题。
<br/>　　先将辣椒密植在塑料大棚中，等着幼苗生在出来，在移植到田中。蛤蟆寨自然不可能有塑料大棚，但这也不算什么问题。
<br/>　　蛤蟆寨有的是空屋子，这也屋子都是用土坯搭成，上面盖着茅草，倒是极好的暖棚。
<br/>　　程大雷将种子埋在细土中，上面浇透水，只等着幼苗生长出来，便移植到天中。
<br/>　　山寨只有六个人，这些事情做得很慢，这就是一个很慢的世界，想要前世那种高效率的生活，根本是不可能的事情。
<br/>　　程大雷忙碌完这一切，就安心等着，等到林少羽和黄三元回来，山寨就能多两个干活的人。
<br/>　　而也就在这个时间段，蛤蟆寨迎来它第一个客人。
<br/>　　那一天程大雷正在山下值班，也就是打劫。辣椒要种，员工要招，等林少羽和黄三元回来，猪也要养，鸡也要养，而打劫的日常工作也是必须要做的。
<br/>　　不过，这条路不知怎么会事，最近都没什么人经过，没有人经过，业绩自然就差。
<br/>　　程大雷百无聊赖的坐在大树下乘凉，正这个时候，一个女人的声音响起：“程当家，这么热的天，还亲自下山打劫？”
<br/>　　“敬业嘛，我一向都是如此。”程大雷随口答了一声，循声音扭过头来，忽然一楞，诧异道：“小白狼！”
<br/>　　距离程大雷不远的，正是杏花岭小白狼。这女人也不知道怎么长得，皮肤像棉花一样白，在这样炎热的天气，看得人垂涎欲滴。
<br/>　　“这么热的天，白当家怎么有时间过来，难道是找我的？”程大雷从树下站起来。
<br/>　　“怎么，不可以。”小白狼捂嘴一笑：“我还真是过来找你，有些事想和大当家商量。”
<br/>　　“走，山上说话。”
<br/>　　程大雷立刻的痛快的道了一声，没有任何一个男人可以拒绝小白狼这样的美女，程大雷更加不能。
<br/>　　一个浑身上下都是荷尔蒙洋溢的女人，就像这天气一样，热得让人喘不过气来。
<br/>　　小白狼随着程大雷走入蛤蟆寨，山寨里到处晒满的辣椒让她有些发呆。
<br/>　　“这是辣椒，从山坡上摘来的，白当家你可以拿走一些到山寨种。”程大雷也是忽然有这个念头，他将小白狼引到聚义厅，没有什么水果，也没有什么茶水，就用井水待客：“白当家找我有什么事？”
<br/>　　小白狼目光环视一圈，压低声音，显得很小心。
<br/>　　“最近有一笔大生意，杏花岭一个人吞不下，不知程当家有没有兴趣？”
<br/>

　　第454章
<br/>　　程大雷双腿一夹身下战马，调整马姿，摘下身后的斧头，亮在帝国军面前。
<br/>　　“来！”程大雷指着刚才与林少羽激战的六将。
<br/>　　林少羽骑马立在程大雷身后，程大雷一出现，他整个人显得拘谨起来。
<br/>　　“大当家……”
<br/>　　“先将身上的伤处理一下，待会儿我们兄弟再说话，身体还撑得住么？”程大雷没有回头。
<br/>　　“还撑得住。”林少羽见到程大雷，心中便有了底气，一振身上的颓废之势。
<br/>　　程大雷目光扫过帝国的万员兵千员将，最后目光落在刚才被自己击退的六将身上。
<br/>　　“来来来，且让本大家领教你们马上武艺。”
<br/>　　这六员将都认得程大雷，即便没见过程大雷，也听过他的名字。抵挡戎族，是值得让人佩服的，而大闹长安，恶名却也传满了天下。
<br/>　　这是一个比林少羽更棘手的敌人。
<br/>　　三军交头接耳，议论着程大雷的来历。章螂头晕脑胀，竟有些完全不知身在何处的感觉。自己竟将这等煞星，看做憨傻之辈，当真是心智被猪油蒙住了。
<br/>　　尉迟离感受到了军心的动摇，身在大撵之上，微微抬头。
<br/>　　“无非一介莽夫而已，来人，取他项上头颅。”
<br/>　　刚才激战林少羽的六将，重新围住程大雷，摆好了攻击的阵势。
<br/>　　“青州焦孟雷，想请程当家指教。”一用锤的大将道。
<br/>　　“豫州司徒墨，向程当家请教。”司徒墨是名留着长髯的用枪之将。
<br/>　　“益州秦雪……”
<br/>　　“京州赵……”
<br/>　　“扬州……”
<br/>　　六员将在马上通名报姓，将程大雷围在当中，不留半点缝隙。
<br/>　　“好说，好说。”程大雷将斧头扛在肩头，道：“我与你们无冤无仇，不过两军阵前，不是你死便是我亡，本当家出手狠些，你们可也别记恨。”
<br/>　　“这个自然，能与程当家交手，是焦某生平夙愿。”
<br/>　　这焦孟雷说得诚恳，想也没有撒谎。程大雷当初在青牛山挡住戎族，还是很得人心的。当然，这夙愿也是杀了程大雷的夙愿。
<br/>　　“来来来！”程大雷在马上招手。
<br/>　　焦孟雷拍马冲刺，铜锤向程大雷砸来。程大雷挥手抡起斧头，两者撞在一起，咚地一声巨响，焦孟雷直接被砸到马下。
<br/>　　于此同时，司徒墨的长枪被程大雷空手抓住，伸臂捣了一下，将对方推到地上。
<br/>　　秦雪的刀，赵立山的棍……眨眼之间，六将的攻击被程大雷一一迫去。他们的刀棍枪锤，全部被程大雷击落，而从始至终，程大雷只不过递出去一斧而已。
<br/>　　而且，他的匹夫剑从未出鞘，不然的话，六将被程大雷击落的，就不仅仅是兵器而已。
<br/>　　六将狼狈倒在地上，站起来后，抬头仰视着程大雷。在军中，有些人是佩服程大雷的，也有些人想踩着程大雷扬名天下。
<br/>　　焦孟雷的口中的『夙愿』，便是这个意思。然而，他们和程大雷的差距，却也只有碰上了才知道。
<br/>　　程大雷扛着斧头，问：“帝国军中，是否还有人要与我交手？”
<br/>　　尉迟离沉默下来，不得不对程大雷的实力重新估计。他既然能出现在这里，很显然老兵组并没能留住对方。
<br/>　　帝国上千武将，当真无一人能在程大雷马前走上那么几回合么。
<br/>　　“既然你们不着急动手，就缓我片刻，容我与我兄弟说几句话。”
<br/>　　尉迟离本不愿让程大雷如此猖狂，但确实也没办法奈何他，不过，等下大军压过去，千军万马，程大雷也逃不掉。
<br/>　　李行哉在马上打了个哈欠，微微眯起眼睛，假装没有看到程大雷。
<br/>　　程大雷回身面向惨败的起义军，目光顺着义字旗往下看，看到一群丢盔弃甲的败兵。林冲和林少羽立在军前。
<br/>　　林少羽避开程大雷的目光，低头行礼：“大当家，少羽给你丢人了。”
<br/>　　“嗳，也就是做事莽撞些，谈不上丢人。”程大雷摆摆手：“手下还有多少人？”
<br/>　　“两三千老弱病残而已。”林少羽叹口气道。
<br/>　　程大雷扫到那条拦路的大河，也叹了口气：“退无退路，进有重兵，不过不怕，我带你们出去。”
<br/>　　“大当家，有法子冲出去么？”林少羽问。
<br/>　　程大雷怔了怔，其实他也没什么法子。眼前确实已陷入绝境，进退不得。不过程大雷接受现实的能力是很强的，无非就是带人杀出去而已。
<br/>　　“出不去了。”林少羽叹了口气：“少羽这一路行来，杀人太多，掉落的人头已为我铺了一条黄泉路。此地风景甚好，离青牛山也尽，正适合埋骨。”
<br/>　　程大雷无言以对，虽然他表现得很有信心，但也无非试一试而已，具体突围的法子，他也并没有想到。
<br/>　　或者，根本没有。
<br/>　　“我一直想见大当家，此时此刻大当家既然来了，我有两物正要献给大当家。”
<br/>　　程大雷奇怪的皱起眉头，都到现在了，林少羽要给自己什么。
<br/>　　林少羽近前两步，来到程大雷马前，往程大雷袖子里塞了一样东西，口中低声道：“我从长安逃走时，卷走一批金银宝物，本是为了日后东山再起时用，眼下看来已是用不上了。大当家，那藏宝之地，就在图上，日后大当家如果用得上，可以按图取出。”
<br/>　　程大雷沉默的点点头。
<br/>　　“还有一物，便是正义教的教众。眼前这三千兄弟，都是随着我一路杀过来的，大战小战，早已将他们煅成精兵，本想去凉州送给大当家。但今日，怕他们都要死在此地。不过全天下，还有我正义教的信徒，日后大当家若是用得着，自然一呼百应。”
<br/>　　林少羽回头冲起义军道：“此人便是我正义教圣主，我若死了，便由圣主大人带领大家，重见光明。”
<br/>　　“参见圣主！”
<br/>　　三千残兵同时俯首。
<br/>　　程大雷在马上看着这一幕，道：“别说这些有的没的废话，拿起兵器，与我放手一战。”
<br/>　　“少羽一路行来，所做的事情颇多，但最后其实什么都没做成。就留下这些东西，希望大当家日后用得上。”
<br/>　　林少羽一拜再拜。
<br/>(本章完)

　　第606章
<br/>　　野原火的目光沉重宛若实质，从踏上这条路开始，他便有准备与所有人为敌。包括本来的敌人，自己的手下，甚至身边的谋士。而对于这个突然冒出来的白无双，他自然不敢掉以轻心。
<br/>　　白无双一揖到地，道：“老朽特来道喜，恭喜大王成为北蛮之主。”
<br/>　　“我似乎并未请过先生。”野原火道。
<br/>　　“我此来还为另外一件事，想问阁下一个问题。”
<br/>　　野原火眯起眼睛，目光仔细打量着对方，到现在他还未下跑，一见事情不对，赶紧溜之大吉。
<br/>　　“什么事？”
<br/>　　“能否借一步说话？”白无双看到野原火警惕的模样，笑了笑道：“大王放心，我如果有歹心，不用等到现在。”
<br/>　　说着，白无双向周围扫了一眼，见不远处一棵碗口粗细的杨树，他走过去抬起右手，一掌拍在树干上。只听得咔嚓咔嚓声响，碗口粗细的杨树竟然硬生生折断。
<br/>　　野原火倒抽一口凉气，这等功夫绝不是战场上的武将能有的，怕早已到神通的地步。的确，如果对方想要杀自己，就算自己脚底生风也逃不掉。
<br/>　　如此，野原火倒是放松了警惕，毕竟也没什么用。他从马上下来，一手扯着缰绳，另外一支手向前一指：
<br/>　　“先生请。”
<br/>　　二人一前一后，缓步而行，这里渐渐静下来，恍惚什么都不曾发生过。
<br/>　　过了不知多久，原先倒在地上的一具『尸体』竟挣扎着坐起来。他睁大眼睛，勉力适应了周围的黑暗，看到同伴的尸体就倒在身边。
<br/>　　这人正是李舜时，他天生心脏偏右，倒是阴差阳错躲过白无双的要害攻击。在地上枯坐良久，心中恐惧，愤怒，迷茫……诸样情绪终究没有什么用处，他挣扎着站起身，一手拖着剑，一手捂着胸口，慢慢向南边走去。
<br/>　　……
<br/>　　避风的草坡下，燃起一堆篝火，枯枝在火堆中噼啪作响，冒出红色火焰。
<br/>　　野原火和白无双隔着火堆，互相都看不太清楚彼此的脸。野原火折了截枯枝掉在火堆中，心里仍想不清楚白无双突然找到自己是为了什么。
<br/>　　“敢问大王，征服北蛮部之后，下一步准备做什么？”白无双问道。
<br/>　　野原火奇怪的看着对方，自己下一步，自然要打呼延力的主意。灭掉呼延力，便有机会统一戎族草原。不过，这种事怎么能说给你听，就连野原火身边也没几人知道。莫非，这白无双就是过来打探自己的心思，瞧出自己心思不对，便一掌击毙。估计，自己的脖子也没有杨树硬。
<br/>　　不过，如果他要杀自己，倒也不用等到现在。如此一想，野原火心里更加糊涂。
<br/>　　只听白无双道：“草原上数百年来，一乱再乱，虽然呼延跋曾试图统一草原，但也没能完成这件壮志。老朽夜观天象，草原上将会降生一位圣主，以气吞天下之雄心，一统戎族诸部落，建立万世不移之基业。”
<br/>　　野原火眨了眨眼睛，他因为曾经的经历，对于这种怪力乱神的事倒是不怎么相信。
<br/>　　“喔，还有这种事。”
<br/>　　“大王觉得，草原上谁会是这位天降之主？”
<br/>　　“那我怎么知道。”野原火随口道了一声：“莫非是戎族王城哪位？”
<br/>　　白无双摇摇头，心中莫名有些苦涩。他倒是想辅佐呼延力成为一代雄主，只不过对方明显没有这份心思。不然的话，他也不用大半夜跑到这个鸟不拉屎的地方来。
<br/>　　“草原上倒是有几位英雄人物。”野原火轻轻颔首，除了呼延力以外，又提了几人，却被白无双一一摇头否决。
<br/>　　野原火这倒是有些糊涂，他道：“还请先生赐教？”
<br/>　　白无双透过火光看着野原火，道：“大王下一步打算做什么？”
<br/>　　野原火一激灵，白无双又一次问出这个问题，他如果再不懂对方的言外之意，就真的是个傻瓜了。
<br/>　　“先生，该不会是说笑吧？”野原火慢吞吞道：“我才不惊人，艺不出众，怎么会是先生所说的天降之主？”
<br/>　　“大王在草原上出世的时间并不长，但已搅出不少风雨，只要有眼睛的都看得出来，一个小小的北蛮部自然无法满足大王的野心。”白无双道：“老朽斗胆问一句，大王的野心究竟在什么地步。”
<br/>　　野原火笑看着对方，笑而不语。
<br/>　　“不知吞掉王族呼延氏能否满足大王的野心？”
<br/>　　野原火心中一发狠，吐出两个字：“不够。”
<br/>　　说罢他就感觉耳边清风拂过，似乎黑暗中有什么声音在嘲笑自己。以野原火现在的势力，说出这两个字，的确有些狂妄了。
<br/>　　“那么戎族诸部呢，够不够填满大王的野心？”
<br/>　　“不够。”
<br/>　　火光飘忽，衬得白无双一张脸通红，他眼底仿佛也有火光燃烧，整个人像是突然年轻了几岁。
<br/>　　“如此，我便明白了。草原无法填满大王的野心，大王定是想要马踏中原，饮马江南了。”
<br/>　　野原火什么也没有说，垂下头来拨弄火堆。
<br/>　　“老朽也斗胆说一句，若大王想吞掉王城，老朽愿助大王一臂之力，这件事当不算困难。”
<br/>　　野原火心中一惊，他自然有吞掉呼延氏的野心，可也知道自己现在的斤两还不够与呼延氏厮杀。按他的计划，至少要等两三年之后，倒时候能不能顺利达到目的，他也并没有什么把握。
<br/>　　但如果有白无双从中相助的话，可以加快这件事的进度，而成功的概率也会变得很大。
<br/>　　不过，对方为何要帮自己呢。这会不会是一场阴谋，引得自己大军攻击，然后对方一举将自己消灭。
<br/>　　到现在这个地步，野原火并不敢轻信于人。
<br/>　　“你愿意帮我？”野原火问。
<br/>　　“不瞒大王，良禽择木而栖，贤臣择主而事，那呼延力目光短浅，心胸狭窄，并非王霸之才。如果大王看得起老朽，老朽愿为大王称霸天下效犬马之劳。”
<br/>　　“仅仅靠这句话，我怕是不能相信你。”野原火慢慢道：“你想要我相信你，总要拿出些真东西来。”
<br/>(本章完)

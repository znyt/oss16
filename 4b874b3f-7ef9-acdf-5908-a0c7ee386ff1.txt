　　第680章
<br/>　　真正的问题也并非浩甲、武威、金城三座城池，而是该用怎样的态度面对他们。
<br/>　　程大雷以迅雷不及掩耳之势拿下凉州城，像那蛤蟆张开巨口，妄想吞掉月亮。结果，只能是被月亮撑破肚子。
<br/>　　程大雷的胃口太大了，可他并没有吞下整个凉州的实力。
<br/>　　蛤蟆城、朔方城、凉州城这是程大雷目前占有的三座城池，但他的总兵力不过三万出头。如今蛤蟆城留守的兵力不过五千，朔方城也差不多是如此，凉州城刚刚被占领，压了两万兵马在此地。
<br/>　　手中握着三座城池，程大雷不得不分兵，这便注定他防御的力量不足。
<br/>　　假若武威、浩甲、金城三座城池回过神来，举大兵攻打程大雷，程大雷手中的三座城便都岌岌可危。
<br/>　　而扩充兵力也绝非一朝一夕的事情，程大雷必须度过刚开始这段不稳定期。
<br/>　　当然，如果能度过这段不稳定期，程大雷站稳脚跟，便可再继续打武威、浩甲、金城的主意，让自己实实在在成为凉州城的主人。
<br/>　　占领凉州城后，必须要通知其他三座城的城主。他们现在当然已经得到消息，估计也在揣测程大雷的态度。
<br/>　　凉州城对程大雷来说，是个开始，还是个结束。他是否会继续动兵，意图攻占其他三座城，还是会按兵不动，守住现在的基业。
<br/>　　程大雷的态度，将会决定他们的态度。而他们的态度，反过来也会影响程大雷的态度。
<br/>　　现在双方坐在牌桌两端，各自风轻云淡，却都在揣测对方口袋中的筹码。
<br/>　　吴用立在程大雷身边，看他紧锁眉头的样子，也能猜到他头疼什么。
<br/>　　“大当家，不若给他们三人去信一封，说明我们并无战意，愿与他们和平共处？”
<br/>　　程大雷摇摇头，心中忽然打定了主意：“不可，我们绝不能示好，示好就等于示弱，你给他们去一封信，要他们老实些，否则，本当家平得了宋伯康，也自然平得了他们。”
<br/>　　“这……”吴用稍稍迟疑：“这是否太过狂妄了些。”
<br/>　　程大雷长叹一口气，他心中何尝不明白自己在铤而走险。如果对方回过味来，联手攻打自己，那么自己一个都防不住。现在只能期盼对方能够给自己喘息的时间。
<br/>　　当然，寄希望于对方，是十分愚蠢的事情。可程大雷也确实没有别的办法，只能如此做。
<br/>　　在如今这个局面，他不能有丝毫示弱，即便囊中空空如洗，也要做出腰缠万贯的样子。
<br/>　　无论如何，这第三件事算是解决了，至于结果如何，那也只能听天由命。
<br/>　　现在，和珅如安抚凉州城的居民，刘悲去处理降兵，吴用面对虎视眈眈的其他三座城池。
<br/>　　这三件事是要紧事，但目前要解决的不仅仅是这三件，还有诸多事情迫在眉睫，它们可能不是很重要，却必须去处理。
<br/>　　“宋伯康去了哪里？”程大雷问阿喜。
<br/>　　“据属下得到的情报，他有极大可能逃往了长安。”阿喜道。
<br/>　　“去了长安就好，去了长安就好。”程大雷松了口气，他最担心的是宋伯康会逃到凉州其他三座城池。若是他纠集兵力，企图复仇，自己的确没有抵抗的力量。
<br/>　　昨夜，宋伯康的表现是很令程大雷失望的。宋伯康并不缺乏智慧，只是太过怯懦，昨夜若他能鼓起勇气，号召士兵反抗，程大雷就算不败，也必定要付出惨重的代价。
<br/>　　但对方竟然撒丫子逃了，不得不说，这让程大雷省了很多力气。
<br/>　　对方，当真不是一个称职的对手呐。
<br/>　　“对了，你着人写封信递到长安城，给帝国天子。”程大雷忽然想起一件事。
<br/>　　“信中写什么？”阿喜道。
<br/>　　程大雷沉吟片刻，道：“就说宋伯康残暴无道，意图谋反。在他的治下，凉州百姓活在水深火热中。我，本当家程大雷受明帝遗命，持匹夫剑斩天下不臣之贼。所以，取而代之，向如今天子讨一个封号。”
<br/>　　阿喜眼睛一亮：“这是真的？”
<br/>　　程大雷白他一眼：“假作真时真亦假，真为假时假也真。不管真的假的，咱们都要摆出是真的态度。”
<br/>　　阿喜仍旧一头雾水，不明程大雷话中的含义。
<br/>　　所谓名不正言不顺，言不顺则事不成。程大雷假若仍旧窝在蛤蟆城，那就无所谓当个山贼，看好自己的一亩三分地。但他既然已经踏出这一步，就必须给自己安排个名份了。
<br/>　　名份这种东西，不是为了糊弄帝国的聪明人，恰恰是用来糊弄帝国那些糊涂人。
<br/>　　阿喜遵命离开，程大雷直起身，长舒臂膀，骨节发出噼啪响声。
<br/>　　他感到了自己的人手严重不足，不缺武将，缺能处理各种事物的文官。和珅现在要处理凉州城的事物，朔方城的发展就得停下来，而蛤蟆城目前也仅仅是维持状态而已。
<br/>　　缺文官，缺兵，程大雷缺的东西太多了。
<br/>　　站起身时才恍然想起，经历昨夜一番大战，一直持续到现在，自己还没歇过片刻。而小蝶柳芷都在蛤蟆城，凉州城是百废待兴，自己身边连个端茶递水的丫鬟都没有。
<br/>　　程大雷自己倒了杯水，润了润发干的嗓子。虽然疲惫，虽然眼前还有诸多困难，但征服这件事还是给他带来了一定的快感。
<br/>　　只要稳住局势，自己就是凉州城的主人，掌中剑，身后的兵就是自己说这话的底气。
<br/>　　宋伯康不足为患，凉州城应该也不会有什么乱子。最大的不确定因素，还是其他三座城池的态度。
<br/>　　如果他们敢打来，自己要如何办？
<br/>　　程大雷深吸一口气，心中忽地生气一股豪横。他们若来了，自己便放手与他们一战，怕他什么。
<br/>　　他端着杯子，杯中只是普通的白水，但给人的感觉却是如饮佳酿。程大雷明白，这不是水的滋味，这是权力的甘甜。
<br/>　　怪不得古往今来，诸多英雄好汉都愿意勾心斗角，争名夺利。或许并不是因为他们在意名利，而仅仅是他们喜欢赢。
<br/>　　而自己，也慢慢开始喜欢这个字眼了。
<br/>(本章完)

        “为什么？”童筱颖不解，顿了一下，才反应过来。
<br/>
<br/>        小爱的狗牌上至今都还写着：小卫爱筱颍。
<br/>
<br/>        所以，大哥这是……吃醋呢？！
<br/>
<br/>        “冷先生，您想在狗牌上写什么？”女服员微笑着问道。
<br/>
<br/>        傅夜沉不假思索地回答道：“青蛙爱天鹅。”
<br/>
<br/>        “……”童筱颖只笑不语。
<br/>
<br/>        这算不算是只属于她和大哥之间的小暗号？
<br/>
<br/>        “那电话号码写谁的？”女服员莞尔一笑，又问道。
<br/>
<br/>        童筱颖这才搭上话：“写我的吧！”
<br/>
<br/>        “好的！两位请稍等。”女服员登记了后，拿着傅夜沉挑选好的狗牌去了后台制作间。
<br/>
<br/>        小爱的护理和体检都做完后，童筱颖抱着小爱，傅夜沉双手提着两大袋的东西，一起从宠物店里走出来的时候，小爱突然从童筱颖的怀里跳了出去，朝着左侧的道路跑去。
<br/>
<br/>        因为没拴狗绳，童筱颖不得不去把小爱追回来。
<br/>
<br/>        傅夜沉见状，只好先将手里的东西提去车子的后备箱，再去尾随童筱颖。
<br/>
<br/>        当童筱颖追到小爱的时候，小爱已经被一个西装革履的男人温柔地抱在了怀中。
<br/>
<br/>        几日不见，他整个人看上去要憔悴了许多。
<br/>
<br/>        两人彼此四目相视的那一刻，时间恍若仅在她和他身上静止，而周遭来来往往的人群却在加速行走。
<br/>
<br/>        傅昼景惊怔地看着童筱颖，看着她披散着一头直亮仿若瀑布般倾泻的长发，穿着时尚简约型米色长裙，以及脚上那双限量款的高跟鞋，忽然间明白了什么。
<br/>
<br/>        童筱颖淡定自若地朝他走过去，将他怀里的小爱抱了回来，刚想不打招呼就转身离去时，却被傅昼景突然抓住了手臂。
<br/>
<br/>        其实，她和他之间，在上次一别之后，就注定彼此是“最熟悉的陌生人”。
<br/>
<br/>        她不想再跟他有任何感情纠葛，更想早点从他的阴影里走出来。
<br/>
<br/>        而这个时候，傅昼景却突然疾言厉色道：“你压根就没有怀孕！”
<br/>
<br/>        童筱颖尴尬地笑了笑：“我当时例假推迟了，我以为怀孕了，结果去医院复查才知道，是个乌龙。”
<br/>
<br/>        傅昼景却眉心微蹙，当他无意间看到她的身后，大哥傅夜沉疾跑的步伐忽然停下时，他心口猛地一阵抽痛。
<br/>
<br/>        眉头紧锁时，脑海里的回忆历历在目。
<br/>
<br/>        那次筱颖撞见他和季思妍在婚纱馆里挑礼服，大哥在筱颖的身边；上次在外婆家，来寻筱颖的人，仍旧是大哥……
<br/>
<br/>        让他最放心、最值得信任的大哥，竟然抢了他最爱的女人！
<br/>
<br/>        “筱颍，你为什么要骗我？”傅昼景手下的力度倏地一紧。
<br/>
<br/>        童筱颖能感觉到从手上传来的痛楚，却没有反驳他的话，而是坦然面对：“我谎称自己怀孕，只是为了……”
<br/>
<br/>        “我指的是大哥！”傅昼景瞪大了黑眸，痛心疾首地打断了童筱颖欲要说话的话，“你明明是跟我大哥在一起了，为什么还要找其他男人来骗我？”
<br/>
<br/>        童筱颖身心一怔，下意识地转过身去，才看到站在不远处止步不前的傅夜沉。
<br/>
<br/>        傅昼景这一生，最羡慕的人是大哥傅夜沉，最嫉恨的人也是大哥傅夜沉。
<br/>
<br/>        她跟傅昼景在一起的这三年里，在某些方面，她比任何人都要懂他。
<br/>
<br/>        所以，她不可以说是大哥强娶了她，她不能破坏他们兄弟两的感情，更不能让昼景去恨大哥。
<br/>
<br/>        那么她现在该怎么做？
<br/>
<br/>        纸是包不住火，她不可能瞒昼景一辈子。
<br/>
<br/>        “筱颍，我可以接受你跟其他的男人度过余生，但是大哥不可以！只有大哥他不可以！”傅昼景忍不住歇斯底里起来，他的手指，隔着一层轻纱，深深地嵌入到她手臂的肉里。
<br/>
<br/>        童筱颖痛得皱起了柳眉，忽然间想到了什么，故作傲气地微微扬起嘴角：“是我故意的！”
<br/>
<br/>        “你什么意思？”傅昼景惊愕地看着童筱颖的眼睛。
<br/>
<br/>        童筱颖耸了耸肩，云淡风轻地回答：“为了报复你啊！”
<br/>
<br/>        “我不信！”傅昼景反驳道。
<br/>
<br/>        童筱颖装出一副得意洋洋的样子，接着说道：“你有什么不相信的？既然是你出轨在先，还跟小三有了孩子，并且还让我净身出户，还说要我等你之类的可笑允诺。你觉得，是你渣，还是我傻？噢！对了！我怎么能就这样甘心地放过你呢？不如，让你叫我一辈子的‘大嫂’，每天膈应着你，岂不是更爽？为了报复你，我冒充苏漫雪去故意勾引大哥，反正我跟苏漫雪有着一张一模一样的脸不是吗？大哥只不过是被我赶鸭子上架了，不得已才对我负责的。”
<br/>
<br/>        “你不是这样的女人！”傅昼景笃定道，心痛的滋味让他胃里开始在翻滚。

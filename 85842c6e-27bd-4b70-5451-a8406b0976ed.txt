　　北邙学院，不比秦皇国度，武胜国度那等小地方，是真正的天才齐聚之地。
<br/>　　诺大的北邙学院之中，每一尊外门弟子，都是需要经过考核，百人之中，脱颖而出，才可以的。
<br/>　　可以说每一尊弟子，放在外面去，都是天才，是外面各大势力宗门争夺的对象。
<br/>　　在这里面的竞争，非常之大，楚风眠现在只是一个外门弟子，之上还有内门弟子，种子弟子，甚至是最为神秘的核心弟子。
<br/>　　其中一些种子弟子的实力，都足以跟生死境抗衡，地位甚至都跟一些长老接近了。
<br/>　　至于核心弟子，每一尊可都是一方巨头了，在学院之中，只怕是地位都要超越许多长老。
<br/>　　以现在楚风眠的实力，还根本远远无法跟他们抗衡。
<br/>　　楚风眠必须要提高自己的实力，才能真正跟这些大人物去抗衡，没有实力，早晚会被人捏死。
<br/>　　两日时间，楚风眠便是一直坐在洞府之中，一动不动，全然是在缓缓修炼。
<br/>　　楚风眠如今的境界，已经达到瓶颈，实力也已经无法再度提升。
<br/>　　所以现在楚风眠要做的，却是在参悟这关于空间的法则，吞噬了无数圣者精血，楚风眠对法则的领悟，只怕已经是达到了随时可以突破的地步。
<br/>　　但是楚风眠现在却还是不能着急突破。
<br/>　　因为楚风眠察觉得到，以他的资质，现在以御风境的境界，便是就足以抗衡生死境强者，楚风眠的资质，绝对是九域罕见。
<br/>　　这样的资质，想要突破，也是千难万难，从御风境到神力境之间，力量便就是一个质的突破。
<br/>　　寻常御风境武者，在突破神力境的一刻，都需要消耗掉无数灵力才行。
<br/>　　而楚风眠所需要消耗的，更是他们的千倍，万倍，如今贸然突破，楚风眠可能会直接失败。
<br/>　　毕竟资质越高，突破起来也就越难，所以楚风眠必须得到一个万全的准备才行，现在的一切，都是在准备。
<br/>　　如今楚风眠得到了天元神丹，倒是增添了一份助力，但是这还不够。
<br/>　　“若是我可以在圣山上的洞府中修炼，便是就可以借助灵泉的力量，令我一跃踏入到生死境之中去。”
<br/>　　楚风眠心中暗自想到。
<br/>　　圣山，便是距离灵泉最近的那一方山脉，这山脉，不属于任何弟子。
<br/>　　而是单独出来的一部分，在北邙学院中，只要有着足够的功勋值，便是就可以兑换一次从圣山上修炼的机会。
<br/>　　北邙学院中的不少弟子，都是会选择积攒功勋值，在圣山上修炼突破，楚风眠的目的，也是如此。
<br/>　　“按照拓跋鼎的说法，想要在这圣山上修炼，少说也要一万贡献点才行，看来必须要想个办法，去积攒一些贡献点了。”
<br/>　　楚风眠心中暗自想到。
<br/>　　在北邙学院中，积攒贡献点最为简单的方式，便是去门派中接取任务，完成任务，就可以得到贡献点。
<br/>　　“必须要等待几日，至少要等到完成入门任务之后，才可以自由接取任务。”
<br/>　　楚风眠低声自语道。
<br/>　　所有进入北邙学院的外门弟子，都必须要完成一次任务才行，这任务是学院强制要求的，不可以拒绝，为的就是要考核弟子的忠诚度。
<br/>　　只有完成了这一次任务，才算是真正的加入到了北邙学院之中去，才可以真正的自由行动。
<br/>　　楚风眠必须要再等几天才行，现在着急，也着急不来。
<br/>　　“等到积攒完贡献点，可以去圣山洞府上修炼，我就可以以此突破神力境，倒时候便是真正的一飞冲天了。”
<br/>　　对于神力境，楚风眠也是充满着期待。
<br/>　　圣者之下皆为蝼蚁，这句话便是预示着神力境跟御风境的截然不同，虽然只有一重差距，但是却宛如天沟，无法跨越。
<br/>　　以楚风眠的资质，等他真正踏入了神力境之后，才会真正的一飞冲天，金鲤化龙。
<br/>　　就在楚风眠的思考之际，在楚风眠的洞府之外，七道身影，却是突然降临下来，来到了楚风眠的洞府外围。
<br/>　　这一群人，皆是身着外门弟子的衣服，但是在他们的肩膀上，都有着大大的风云二字，表露了他们的身份。
<br/>　　这突然出现的动静，引得周围不少武者也都是探出头来，观察着，想要看看到底是发生了什么。
<br/>　　当看到这七人肩膀上的风云二字的一个个，一个个的眼神，却是骤然大变。
<br/>　　“风云会的人！”
<br/>　　“果然，风云会的人，不会善罢甘休。”
<br/>　　这上面的风云二字，已经是暴露了这群人的身份，楚风眠打伤了牛峰，马允二人，更是直接骂了武风云。
<br/>　　这风云会的人，果然来了。
<br/>　　“方师兄，这里就是那小子的洞府，就是他打伤了我们，更是辱骂武师兄。”
<br/>　　这七人之中的两人，便是牛峰，马允二人。
<br/>　　其中牛峰，便是指着楚风眠的洞府，献媚的对身旁的方师兄开口道。
<br/>　　“他现在就在洞府之中。”
<br/>　　那名为首的方师兄，听到牛峰的话，向着洞府中看了过去，一眼便是看见了楚风眠，打量了一眼楚风眠的境界，他的脸上露出几分不屑道。
<br/>　　“你们两个废物，居然被一个御风境的蝼蚁打败了，根本不配做我们风云会的人，从今日起，你们二人便是被逐出风云会了。”
<br/>　　“是，是。”
<br/>　　牛峰跟马允，急忙的附和道，他们二人已经是废人了，被逐出风云会，他们也无力反抗。
<br/>　　“我们二人的事我所谓，但是此人敢辱骂武师兄，简直是狂傲的没变，罪大恶极，必须要惩罚。”
<br/>　　牛峰开口，声泪俱下，简直是一脸忠诚。
<br/>　　他现在已经成了一个废人，再也没有任何希望了，但是他也不想要让楚风眠好过，他必须要让楚风眠，也成为一个废人。
<br/>　　所以才会将风云会的人招来，目的就是为了要对付楚风眠。
<br/>　　“这小子，居然敢辱骂武师兄，真是活的不耐烦了，一个小小的新人，居然敢挑衅我们风云会的威严！简直放肆！”
<br/>　　方师兄冷冷的开口道。
<br/>　　“今日不给他一个教训，我们风云会还如何在外门立足。”

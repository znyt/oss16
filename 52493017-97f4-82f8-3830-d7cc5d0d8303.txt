　　谁都料想不到，在这大庭广众下，萧家九大长老齐聚，萧若狂竟还敢如此狂妄，当众偷袭萧逸。
<br/>　　这忽然的偷袭，让所有人都是一惊。
<br/>　　而萧逸在不使用升龙的情况下，不过是凡境九重，与后天一重足足差了一个大境界，若是被打中，必定重伤，甚至有生命之危。
<br/>　　幸好，萧逸前世身为顶尖杀手，谨慎是他无时无刻都保持着的。
<br/>　　“嗯？”萧逸几乎在大长老暴喝一声前，已经感受到后背一阵浓烈的危机感。
<br/>　　“哼，小废物，不使用那些奇怪的招式，我看你如何挡我这一掌；只要把你杀了，我看谁又能拿我怎么样！”
<br/>　　萧若狂极想杀掉萧逸，更是对自己这次的偷袭有足够的把握。
<br/>　　“既然你卑鄙，那就被怪我狠辣了。”萧逸目光一凝，眼中尽是杀意，没有转过身去抵挡那一掌，而是堪堪歪了歪脑袋。
<br/>　　正是他这看似不经意的歪脑袋，竟让萧若狂的一掌打空，从他耳边擦过。
<br/>　　“躲过了？怎么可能！”萧若狂顿时大惊，他本想一掌轰碎萧逸的脑袋，万万没想到萧逸能躲开。
<br/>　　“哼，没什么不可能的。”萧逸冷哼一声，瞬间使用了升龙，而后一个虎形握住了萧若狂的手掌。
<br/>　　而后一个倒摔，将偷袭的萧若狂甩到了自己面前。
<br/>　　咔嚓一声。
<br/>　　只见萧逸的虎爪一用力，轻松将萧若狂的手臂给废掉。
<br/>　　“阿。”萧若狂痛呼一声，偷袭失败后，论正面攻击，他只有被萧逸碾压的份。
<br/>　　“小废物，快给我滚开。”萧若狂怒喝一声。
<br/>　　萧逸充耳不闻，面容冷酷，又是咔嚓一声，将萧若狂的另一条手臂废掉。
<br/>　　“小废物，你想做什么，我乃玄火门弟子，你...”
<br/>　　他的话还未说完，已经感到腹中一疼，无力谩骂。
<br/>　　“你...你想废了我？”萧若狂忽然意识到萧逸想做什么，顿时大惊。
<br/>　　他明显感觉到，萧逸刚才打向腹中的一拳，位置正是丹田之处。
<br/>　　丹田，是武者储存和调动真气的最重要位置，一旦被毁，将会修为全失。
<br/>　　“萧逸，给我住手。”五长老大喝一声，准备救援萧若狂。
<br/>　　三长老眼疾手快，脚步一动，拦住了他面前，冷声道，“五长老，还是给我老实看着吧。”
<br/>　　萧若狂刚才意欲偷袭并杀掉萧逸的行为，已经让三长老无比愤怒。若五长老还敢从中作梗，他绝对会暴走。
<br/>　　长老席上，七、八、九三位长老一向以五长老马首是瞻，见五长老被拦住，顿时一急，准备出面支持五长老。
<br/>　　孰料这时，大长老微微地咳嗽了两声，“咳咳”。
<br/>　　这两声简单的咳嗽声，听在七、八、九长老耳中，却如遭雷击，顿时不敢动弹，老老实实地站在原地。
<br/>　　大长老看向萧逸的目光，已经不能用满意来形容了，而是极为赞赏。
<br/>　　就连二长老也赞叹道，“好小子，反应竟如此敏锐，不愧是个出色的武者。”
<br/>　　大长老也笑了笑，道，“反应灵敏，临危不惧，又是炼药天才，此子日后必成大器。”
<br/>　　比武台上，萧逸连续轰了数拳，正准备毁了萧若狂的丹田，让他彻底失去修为。
<br/>　　忽然，三道火红色人影疾速略上比武台，速度极快，竟抢先在萧逸手中救走了萧若狂。
<br/>　　三道人影，乃是这次陪同萧若狂回萧家的玄火门执事，皆是后天境强者，实力极强。
<br/>　　萧若狂身为门主亲传子弟，乃是玄火门有名的天才，自然有门中强者陪同并保护。
<br/>　　只是，这三人救走了萧若狂也就罢了，其中一人竟还意图趁机攻击重伤萧逸。
<br/>　　“哼。”一道怒喝声响起。
<br/>　　早有准备的大长老，岂会让他们得手。
<br/>　　“嘭、嘭、嘭”
<br/>　　大长老仍旧在长老席上，不见他有什么动作，但那三位玄火门执事包括萧若狂，却忽然吐血被击飞。
<br/>　　大长老萧离火的实力，强悍如斯。
<br/>　　“好强。”萧逸也吃了一惊，暗道，“论实力，大长老怕是比三长老和二长老还要强。”
<br/>　　极少有人见过大长老出手，但其实，他才是如今萧家的第一强者。
<br/>　　“噗。”萧若狂猛地吐出一口鲜血，而后满脸不可置信地站起来，愤怒地问道，“大长老，你竟然打我？你疯了吗？”
<br/>　　萧若狂状若癫狂，怒声道，“论天赋，我乃是紫云城最出色的天才；论身份，我父亲是萧家五长老，我师傅是玄火门门主；假以时日，我将极有可能成为玄火门门主，甚至是一名出色的炼药师。”
<br/>　　“我将是整个萧家的骄傲，大长老你竟然为了萧逸这个小废物打我？”
<br/>　　一旁的萧逸不禁翻了个白眼，暗道，这萧若狂确实狂妄，而且狂妄得接近白痴了。
<br/>　　萧若狂，从小都在无数赞扬和天才光环下长大，深受所有长辈的溺爱，早已养成了目空一切的性格。
<br/>　　如今，萧逸无论是武道修炼还是炼药天赋都胜过了他，连族中长辈也帮着萧逸，这让他无比愤怒。
<br/>　　“大长老，你可想清楚了，他日我修为有成之时，整个萧家获得的好处，远不是萧逸这个废物能给予的。”萧若狂竟开始直接质问大长老。
<br/>　　“放肆。”二长老怒喝道，“萧若狂，何时轮到你来质问大长老。”
<br/>　　大长老摆了摆手，示意二长老不用多言，而后直视萧若狂，道，“萧若狂，你觉得你自己很厉害，觉得萧家需要以你为荣。”
<br/>　　“但，你可曾想，以往若无家族帮助，你如何能展露自己的天赋，成为一名修炼天才。”
<br/>　　“你又可曾想，若玄火门不是看在萧家的份上，你如何能在玄火门中得到玄火门长老乃至门主的青睐和帮助。”
<br/>　　“罢了。”大长老叹了口气，道，“如此数典忘本之人，不配做我萧家子弟，日后若无允许，不得再踏入萧家半步。”
<br/>　　大长老对萧若狂失望至极，但看了眼萧逸后，又忽然老怀大慰，顿感无比满意。
<br/>　　“你...你会后悔的。”萧若狂大声骂道，“老不死得，待我回玄火门禀报门主，你今日将我等打伤，就等着我玄火门的报复吧。”
<br/>　　“混账。”大长老瞬间怒了，没想到萧若狂竟还是如此不知悔改，凌空一巴掌扇了过去。
<br/>　　萧若狂直接被扇飞了数米，口中狂吐鲜血。
<br/>　　大长老威严地说道，“萧若狂，回去问问你口中所谓的门主，老夫当年成名之时，他算个什么东西？还有，告诉他，日后不用再来找老夫炼丹。滚吧。”
<br/>　　“噗。”萧若狂再次吐了一口鲜血，彻底晕了过去。
<br/>　　三个玄火门执事，连忙抱起了萧若狂，远遁逃离。
<br/>　　萧家执事刚想去追，大长老摆了摆手，道，“不必追。”
<br/>　　......
<br/>　　这次的比武，没想到会以这样的情况结束。
<br/>　　本是五长老刁难萧逸，意欲让萧若寒上位，没想到反而彻底奠定了萧逸的地位。萧若寒重伤，萧若狂更是变相地被逐出萧家。
<br/>　　在族人们众星拱月般的目光中，萧逸离开了比武场，回到了家中。

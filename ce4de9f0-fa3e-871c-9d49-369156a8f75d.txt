　　当叶帝他们走入现场后，自然是迎来无数粉丝的欢呼。
<br/>　　经历过去年荣耀战队的辉煌，以碾压的战绩拿下冠军，叶帝他们就已经是积攒下来了无数的粉丝。
<br/>　　将一个电竞薄弱的国家一举带上了超级电竞大国的地位，这份荣誉感和归属感是其他战队比不了的。
<br/>　　虽然在战队拆分过后，有一阶段双方的粉丝是有一些争执的，但是随着两支战队的优秀发挥，众多粉丝渐渐的还是接受了这种情况，而且更加坚定的支持其了他们。
<br/>　　到了今天，那些粉丝已经是非常坚定的相信当初叶帝许下的那个承诺了，那就是荣耀蓝白战队一定会创造出属于他们的传奇时代，今年他们将毅力到最后。
<br/>　　而且今年的世界赛赛制已经是重新规划了一遍，在世界赛上将会只有32个战队参加比赛，不再是像往年那样有几百支队伍出现的局面了。
<br/>　　而在这32支队伍中，其中华夏、韩国还有美国以及法国这些拥有着王者俱乐部的国家的名额是不会被取消的。
<br/>　　也就是说华夏、韩国这些个国家依旧是会有3个出线名额，至于其他的国家则是回去共同争取那最后剩下的20个名额。
<br/>　　而且今年还取消了外卡赛，所有的外卡赛队伍全部都将被分配到正规的赛制当中，由600多支队伍来争取这20个名额，战况还是非常惨烈的。
<br/>　　不过这样的优点就是世界赛的时间不会在像往年那么长了，仅有32支队伍参赛，时间会节省下来很多。
<br/>　　重新回到这个舞台，叶帝几人都是有些激动，尤其是站在舞台上的时候，当台下的那些观众热情欢呼的时候，叶帝他们心中的那种归属感也就越发的强烈了起来。
<br/>　　“夏季赛揭幕战正式开始！”经过一系列的活动后，当米粒高喊出这句话后，现场彻底的沸腾了起来，欢呼声直冲天际。
<br/>　　看的出来，经过这漫长的一个月的等待，这些观众也是有些迫不及待的想要看他们的比赛了。
<br/>　　“今天的一场比赛将会有荣耀蓝队对阵雷霆战队，第二场比赛则是会有皇族战队对阵王朝战队，明天则是会有荣耀白队的比赛，让我们拭目以待，比赛马上开始。”在欢呼声结束后，林雅楠连忙跟着开口说道。
<br/>　　而在两人说话的时候，场上双方已经是开始了banpick。
<br/>　　本来大家以为这场比赛雷霆战队会给蓝队这边造成一些阻碍，甚至是赢下来，然而谁也没想到，两场比赛荣耀蓝队这边都是火力全开，每场比赛几乎都是在35分钟的时候结束。
<br/>　　“蓝队的实力变的更强了，老秦、蒋念还有楚南包括飞刀和林萧，感觉五个人的实力都有了一个质的提升，这个真的太可怕了。”比赛结束后，陆伟一脸震撼的在那里说着。
<br/>　　“全场比赛都被对方掌控着，我这边连gank的机会都没有，他们究竟是怎么办到的，为什么一个月的时间他们的实力还会进步，我原本以为他们春季赛的时候已经是巅峰了呢。”小刀也是同样震撼。
<br/>　　解说席上，
<br/>　　“我的天，太不可思议了，这场比赛给我的感觉就是完全不是一个量级的比赛，但是要知道雷霆战队这可是被官方评为世界****俱乐部之一的存在的，蓝队这是要逆天么，看到这一幕，我感觉今年的冠军要被蓝队收入囊中了。”米粒大声的喊着。
<br/>　　“确实太厉害了，赛前谁也不会想到雷霆战队居然会被压着打成这样，毕竟雷霆战队可是有着王者伟神还有小刀这个顶尖分段的选手，加上他们的下路和上单同样不弱，而且上个赛季他们还赢过蓝队一局对战，大家对他们的期望还是非常大的，今天这个真的是太不可思议了，蓝队再一次给我们带来了惊喜。”林雅楠跟着开口说道。
<br/>　　“雅楠你在荣耀基地内，对于蓝队的实力提升的这么快，你能透露一些消息么？”米粒好奇的问道。
<br/>　　“其实我也不知道，之前半个月他们都在备战王者战，接下来都是正常训练，唯一知道的一点就是暗夜和霄神变的比以前更加的强大了。”林雅楠苦笑着说道。
<br/>　　“那就对了，这两场比赛我们看到基本上都是蓝队这边中野辅三人在带着节奏的，辅助飞刀肯定是听霄神的指挥的，说实话我现在有些期待起明天白队的比赛了，不知道大魔王会变的有多强。”米粒开口分析着。
<br/>　　“是的，明天白队的比赛可是非常有意思的，而且最关键的是白队会带给大家一个意想不到的惊喜，至于是什么到时候大家就知道了。”林雅楠卖了个关子，而经过林雅楠这么一说，无数的粉丝对于荣耀白队的比赛倒是更加的期待了起来。

<br/>　　“蓝队真的是火力全开啊，两场比赛雷霆战队都被压的根本没法反击，真的牛逼。”
<br/>　　“感觉今年的冠军非我们荣耀蓝队莫属了，六六的！”
<br/>　　“我们白队肯定不会差的，我们这边也是有着两个王者呢，你们不要高兴的太早了。”

<br/>　　“皇族这边虽然有着白衣和司马贼这两个天赋型选手，但是其他三个位置实力差的还是有些大，而且他们还都太年轻了，论老道和雷霆战队还是要差一些，如果我猜的不错的话，这最后一个名额可能是雷霆的了。”叶帝跟着开口说道。
<br/>　　通过观看今天皇族和雷霆战队这两支战队的比赛，叶帝的心里已经是有了一个大致的推测。

　　狼牙战团到达西南地带之后，就第一时间在靠近山脉的地方建立了一座本部，这是他们从未有过的新创举，此处的环境严寒，到处都飘着雪，很多地方更是冰冻三尺，与这个季度的正常气温完全不一样。
<br/>　　“好在气温低，食物能储存很久。”刘玉定穿着军大衣，与何峰并肩行走。
<br/>　　“恩。”何峰只是点了点头，这几天他都很沉默。
<br/>　　刘玉定也没有深问，每个人都有秘密，而狼牙内部的很多机密，他也并非事事能打探。
<br/>　　于是刘玉定也只是就着当下的问题与何峰探讨：“我们这次带着八万人过来，任务艰巨，一路上也避开了那些丧尸的视线，算是完成了第一步计划。”
<br/>　　“就是苏行不在，这本部建造的有些糙。”一旁的路冰泽满脸嫌弃。
<br/>　　刘玉定白了他一眼：“很不错了，依靠山脉不说，前方在推进的时候，后方的山间隧道打通也只是需要时间而已。”
<br/>　　“既然到了就事不宜迟，明早就率领第一批队伍出发吧。”何峰忽然开口，让所有人都一愣。
<br/>　　“这么急？”徐锋皱起眉。
<br/>　　“时间不等人。”何峰解释道：“楚涵不在这里，万一出现什么变故，我们跑这么远的计划泡汤怎么办？”
<br/>　　江凌若说出了不同意见：“南方的埋骨之地，可是连神秘家族都不敢踏足的地方，我们虽然有勇气过来，不代表可以没有计划就只能鲁莽冲入。”
<br/>　　何峰看着她：“你不就是那个从埋骨之地杀出来的人么？”
<br/>　　江凌若苦笑：“那是两年前。”
<br/>　　“我相信两年后的你依然可以。”何峰说着这样的话，而后又在众人都迟疑的时候补了一句：“据我和楚涵的推测，丧尸王并不在埋骨之地，所以在尸潮攻击华夏南部边境的时候，就是我们切入丧尸后方老巢的最佳时机。”
<br/>　　何峰留下这句话后就大步离开，显然已经做好了立即出发的准备。
<br/>　　其余人则被这一信息所惊，一个个都待在原地大眼瞪小眼，丧尸王不在埋骨之地这一说法，他们的确是第一次知道。
<br/>　　“你们，怎么看？”刘玉定难得不摆出军法官的架子开口。
<br/>　　江凌若苦思道：“我连丧尸王到底诞生没有都保持怀疑，楚涵与何峰到底是怎么推理出那家伙不在埋骨之地的？”
<br/>　　“老大的想法你别猜。”路冰泽说着拍了拍江凌若的肩膀，语重心长的劝说道：“否则你会变痴·呆的。”
<br/>　　“早说丧尸王不在啊，靠！”徐锋则是翻了个大白眼，手中金枪‘刷刷’的挥舞，似乎有使不完的力气：“既然不在还担心个啥，明天就启程把埋骨之地给屠了！”
<br/>　　陈少爷一直没吱声，他很想说老大现在还昏迷着呢，但望着众人都毫不知情又信心十足的样子，最终还是没有说出口。
<br/>　　何峰也并非真的弩定丧尸王不在南方，只是在与楚涵深讨这一计划时，两人都决定冒一次险，埋骨之地总要有人过去攻打，长期依靠边境线的防守与丧尸对抗根本不是有效办法，只有将埋骨之地彻底掀翻，才是真正解决南方的这批恐怖丧尸。
<br/>　　那里的丧尸与华夏不一样，更与全世界的其他地方不一样，那里是丧尸这一种族的中心，如果不及时制止它们的发展，早晚有一天会变成最大隐患。
<br/>　　到时候别说南方那么大地理范围内的丧尸，甚至连蜗居在华夏各个城市里的丧尸，都会受到埋骨之地的召唤，开始走出城市慢慢聚集。
<br/>　　分散的丧尸不足以造成威胁，但如此多的数量一拥而上，才是最可怕的事！
<br/>　　狼牙战团的行动快速又坚定，一部分人留在山脉边上继续建造本部，一批最强的先行部队则在第二天清晨就出发，徐锋率领着杀羽战队首当其冲，担任着冲锋和开路的职责，蒋天庆的侦查队也不甘示弱，游走在杀羽战队两旁不断扩大探索范围，致力将他们走过之地的所有情况都了然于心。
<br/>　　其余高战力队伍都跟在这两批队伍的后方，步步朝着南边推进……
<br/>　　在狼牙战团不声不响朝着埋骨之地进发的时候，以段氏基地为中心的南方边境防守线，也经历了第二次大战，依旧是百万尸潮的强烈攻势，在某天夜里朝着另一处小基地突如其来的就发起了进攻。
<br/>　　好在这时候的联盟军已经全员到达这条防守线，驻扎在两旁的队伍接到消息后第一时间就往事发地赶，回到战场的段江伟拎起那柄紫色太刀就带着人出发，金泽宇也从另一方向赶去。
<br/>　　好几股队伍在半路相遇，短暂商议后分为两批，一方以金泽宇为首继续前进，支援受到尸潮进攻的小基地进行防守战。
<br/>　　段江伟则是带着另一批人绕后，从尸潮攻来的方向反向进攻，给它们来了个前后夹击。
<br/>　　比不上狼牙战团那么勇猛的战斗方式，但说起打仗联盟军中也不缺人才，在经历过一次又一次的尸潮后，这批人杀起丧尸来也得心应手。
<br/>　　于是第二次的尸潮就这么被顺利挡下，很多刚参军不久的新兵也从一开始的害怕渐渐拥有了信心，文奇胜也从这两次战役中摸索出了一些规律，哪怕尸潮接二连三的出现，也能应付的得当。
<br/>　　艰苦又顺利的防守战就这样每天进行着，随着日子一天天的过去，很多人也都习惯了这样的每日战斗和疲惫，待在华夏各处基地内的幸存者安详的过着日子，从远方传来的战报里详读着当今世上的大事。
<br/>　　在这样按部就班的日子又过去了半个月后，忽然爆发的一则消息，又一次震惊了所有人。
<br/>　　“报！”一人匆忙的跑进鲁初雪办公室，浑身是汗满脸苍白。
<br/>　　鲁初雪手中的笔‘啪’的一声落在地上，自从段江伟把这些杂事都扔给她自己逃到战场上后，这些天她都从未见到汇报人员这么慌张过，最严重的事也无非是某次防守战我方联盟军死伤数量超出预估。

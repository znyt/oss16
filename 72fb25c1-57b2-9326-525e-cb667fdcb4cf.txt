　　杜正伦一脸深受启发的样子，八十万户逃民，三百多万人口，如果能把这些人全部找出来登记造册，那么对于大唐来说，完全可以迎来一个井喷式的发展。
<br/>　　三百万人口可以开垦多少土地？又可以缴纳多少赋税？大唐律规定，唐人四岁为小，十六为中，二十一岁为丁，六十为老。
<br/>　　这些人里面凡丁及男十八岁以上这，皆有八十亩口分田，以及二十亩永业田，老者为四十亩，寡妇、妻、妾则为三十亩，凡户主加二十亩永业田。
<br/>　　这样新开出来的土地按三百万人，人平均五十亩算，那就是一万万五千万亩地。
<br/>　　杜正伦感觉自己已经快要疯了，扳着手指算来算去怎么也算不明白这些人到底会缴纳多少赋税。
<br/>　　而且人多了并不全是赋税的问题，大唐执行的是租庸调，这里涉及到的事情远比单单一个赋税的问题要复杂得多，比如服兵役的问题，比如出劳力修桥修路的问题。
<br/>　　想到最后，杜正伦魔症了一样，竟然连告辞都忘了，踉踉跄跄的离开李承乾位于丽正殿的书房，念念有词的离了东宫，去找与他一样留在长安的虞世南、姚思廉研究逃民的问题去了。
<br/>　　“殿下，这杜正伦未免太过无礼了吧？”因为夜魅与杨雨馨的离开，李承乾将自己的跟班换成了苏猛，这家伙性子直，看不惯的东西就要说，杜正伦进来不通报，离开不请辞的行为在他眼中自然是极不顺眼。
<br/>　　“收起你的嘴脸，去外面守着，不经允许任何人不得进来。”李承乾瞪了苏猛一眼，最讨厌他这种马后跑，真要是忠心护主，丫刚刚就应该直接把杜正伦给踹出去。
<br/>　　李承乾开始怀念夜魅了，如果那姑娘在的话，这些‘闲杂人等’如何进得了自己这书房。
<br/>　　“喏！”吃了李承乾的排头，苏猛不敢多话，答应一声退了出去，不过这家伙太直，纵然被赶出去，也依旧不知道自己错在哪里。
<br/>　　而李承乾则是注视着苏猛离开的背影，最后将目光定在门框上，思绪又回到最开始杜正伦来之前的那一段。
<br/>　　从来到大唐的那一刻一直到现在的贞观八年，李承乾所有的行为都是秉承一种随心所欲的态度，直到与禄东赞谈过一次之后，他才意识到，自己必须要有一个为人处世的态度。
<br/>　　小市民的心态必须调整，随着手下的人越来越多，疑心病太重只会让自己众叛亲离，而且就目前来说，他这个太子当的似乎太过和气了，大多数时候都说话都是含糊其词，给人模棱两可的感觉。
<br/>　　比如禄东赞，事实上面对这个家伙的时候，他完全可以用太子的身份向其施压，顺我者昌，逆我者亡，不为我所用尽为吾之对手，必除之而后快。
<br/>　　这样的处世方式才是一个上位者应该有的行为，闪烁其词只会让对方更加瞧不起他，甚至会在心里生出某种异心。
<br/>　　李承乾呆呆的坐在椅子上，默默的想着自己的心事，直到长乐的一张俏脸出现在他的面前：“哥，你又想什么呢？叫了你半天都不应声。”
<br/>　　“你怎么来了？苏玫她们呢？”李承乾低垂的眼睑缓缓抬起，在长乐的脸上扫了一圈之后，淡淡问道。
<br/>　　“不就在你身后么，哥，你今天很奇怪啊，到底发生什么事情了？”长乐指指李承乾的身后，示意他要找的人就在那个位置，然后说出自己的新发现。
<br/>　　“想通了一些事情。”李承乾扭头看看站在自己身后的苏玫与孔雯，淡笑着开起玩笑：“怎么来了也不叫我，我又不会吃人！”
<br/>　　结果不等苏玫等人开口，长乐就撇嘴说道：“得了吧，我们进来的时候就看到你咬牙切齿的表情，除了我谁还敢吱声。”
<br/>　　“这次算是我的错，下次一定不会忽视你们！”李承乾没有理会表情夸张的长乐，只是拉起苏玫柔软的小手，轻轻的拍了几下。
<br/>　　“无碍的，殿下公务繁忙，我等本不应该来打扰的。”李承乾第一次在人前做出如此亲昵的举动，苏玫脸色微微有些发红，连声道歉。
<br/>　　李承乾摇摇头，并没有与苏玫争论什么，将目光转向孔雯：“你们用过晚膳了没有？若是没有便在这里吃些吧。”
<br/>　　有了一下午的思考，李承乾的处世态度转变了许多，对待苏玫几人一改往常和光同尘的态度，变得积极主动了许多，但也强势了许多。
<br/>　　食不言，寝不语，有长乐这个妹妹监督，李承乾数次想要开口聊点什么都被瞪了回去，直到几人安安静静的吃完东西净了口之后，李承乾这才有机会开口。
<br/>　　“小玫，小雯，你们两个回去之后好好想想，看看自己有什么特长，有什么想要作的事情，想好了告诉我。”面对静静等着他开口的苏玫等人，李承乾一边品茶一边说道。
<br/>　　“哥，你又想干什么？”长乐在一边听出事情有些不对头，忍不住接过的话头。
<br/>　　李承乾将茶杯放到一边，抬手止住长乐接下来的话，半开玩笑的说道：“连你也一样，学医这么长时间了，难道你就没有想过以后要做些什么？”
<br/>　　找到了自己的缺点，自然就要改正，长乐与苏玫等人此时面对的李承乾正在完成自己最后的蜕变，至于最后能否蜕变成一个什么样的人，那就要看李承乾自己的选择是什么。
<br/>　　长乐明显发现了李承乾的不同，看着自己这个有着‘智妖’诨号的大哥眉头越皱越紧，一时间不知应该如何判断他的目的。
<br/>　　“不用想了，你是我妹妹，自然有选择自己人生的权力，而她们是我的女人，我身为她们的丈夫，为她们撑起一片天空也是该当。”李承乾并没有让长乐考虑太久，便给了她一个答案。
<br/>　　只是这个答案的背后要付出多大代价却是李承乾无法想象的，不过他已经做好的迎接一切挑战的准备，就像高尔基的文章中那只海燕一样。

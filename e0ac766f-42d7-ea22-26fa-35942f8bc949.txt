　　第473章
<br/>　　战斗过后，两支队伍聚在一起。
<br/>　　众人打扫战场，清点伤亡，相比鱼字军的严肃，张肥和高飞豹的手下就放肆很多。他们和他们的指挥者一样，好勇斗狠，逞强好胜。因为秉性相投的缘故，两支队伍的关系也最好，所以这次才会配合行动。
<br/>　　“将军，敌军已经被击溃，另还缴获一批辎重粮草，两位将军看怎么处理？”
<br/>　　“还能怎么办，把能用的都带上，带不走的全部烧掉，烧掉也不能留给其他人。”
<br/>　　“是！”
<br/>　　手下人立刻开始行动，不多时便火光冲天。
<br/>　　高飞豹盯着熊熊升起的烈火，道：“三爷，咱们该去下一个地方了吧？”
<br/>　　“不用走。”张肥道：“这里出事，消息肯定会传出去，我们守在这里，打他们的援兵。”
<br/>　　草原上的夜，出来行动的不仅有猎食的野兽，食尸的豺狗，还有一个个人。
<br/>　　关鱼、张肥、高飞豹、秦蛮、赵子龙五支队伍都开始自己的行动，他们时而配合作战，时而分开各自为战。五路军成五条线扫荡过草原，一旦遭遇敌人，便如发现猎物的野兽般扑上去。遇到数量庞大的敌人，便一点点将敌人蚕食。
<br/>　　……
<br/>　　大帐之中，崔相一张脸黑得像锅底一般，随着一份份战报呈到他的案前，他的脸色变得更加难看。
<br/>　　三日前，青州徐龙田部被击溃，全队三千余人，死伤过半，残存兵马丢盔弃甲逃回。
<br/>　　两天前，相府的运粮队遭遇敌人埋伏，所有粮草被付之一炬。
<br/>　　昨日，敌军攻占黄沙城，守军死伤惨重。
<br/>　　……
<br/>　　这都是这两三日内发生的事情，事情一旦发生，就第一时间递到崔相案前。现在累积下来，已经有十几封之多，全部都是战败的消息。
<br/>　　被击败的不仅是相府的队伍，还包括将军府，扬州李星的队伍，其他各路诸侯，只要敢在草原上落单，再接到他们消息，定就是他们全军覆没的消息。
<br/>　　五支队伍，像是五头饿狼，在草原上席卷而过，无人能敌。
<br/>　　“他们想干什么，他们想干什么！”便是以崔相的涵养，此刻竟也是勃然大怒：“帝国军难道是一群酒囊饭袋么，就算是一群猪，也知道哼唧两声，他们连猪都不如。”
<br/>　　李行哉坐在帅案后，相比崔相的愤怒，他倒是表现得很平静。坐在那里老神自在，在崔相拍桌子的时候，他将茶杯端了起来。
<br/>　　崔相注意到了他，强压下心中的怒火：“六殿下……”
<br/>　　“啊……”李行哉像是才回过神来，睁开眼睛道：“崔相息怒，息怒，要知道气大伤身，现在三军系于您一人，您还是要照顾好自己的身体。”
<br/>　　“六殿下，对于这件事怎么看？”
<br/>　　“哈，依我看呐，崔相也不要怪帝国军无能，实在是敌人太狡猾。”李行哉抿了口茶。
<br/>　　“殿下曾在蛤蟆城住过，对于他们的行事作风很了解，定能看出许多我们看不到的东西。还要向殿下请教？”
<br/>　　“这个么，我也就是随便说说，说得也未必对。”李行哉道：“这几场帝国军战败的战斗，所参与的都是后勤队伍，帝国主力部队并未参战。蛤蟆军利用他们的高机动性，打得过就打，打不过就骚扰，等着队伍合拢后才一举出击，将我方击溃。”
<br/>　　“也就是说，帝国军的战斗力未必弱于蛤蟆军？”崔相问。
<br/>　　李行哉点点头，又摇摇头：“当然不能完全这么说。我在程大雷身边待了三年多，这三年他可是一天都没有闲着。一柄刀磨了三年，今朝头一次出鞘，自然锋利得紧。”
<br/>　　“言之有理。”崔相缓缓点点头：“那据殿下看来，他们是打算做什么呢？”
<br/>　　除了愤怒帝国军的无能，崔相另外困惑的就是这一点。蛤蟆军从草原上席卷而过，抢到粮食便就地焚烧，对城池似乎也没有兴趣。目前也仅仅是占领黄沙城一座城池而已。
<br/>　　“这很简单呐，林少羽死了，他自然要做些什么。”李行哉。
<br/>　　“你是说，他不计代价，不计伤亡……”崔相顿了顿：“这仅仅是一场单纯的报复行动。”
<br/>　　李行哉重重的点点头。
<br/>　　……
<br/>　　黄沙城是一座废城，因为战争的缘故，此地已经十室九空。原先，这里被扬州李星部占领，但在不久前，程大雷攻陷占领了这里。
<br/>　　持续这些日子，不断有战报传来，多是胜利的消息。在某地，歼灭某部，歼敌多少多少人，又抢到了什么东西。
<br/>　　大部分物资被运回黄沙城，以战养战，黄沙城成了程大雷的大本营。但其实黄沙城并没有屯多少兵，大部分人手还是在草原上，继续进行这次报复行动。
<br/>　　“大当家……”有一名手下汇报：“高将军从青牛山回来了。”
<br/>　　“喔！”程大雷眼睛一亮：“带回来什么东西没有？”
<br/>　　“呃……好像没有。”
<br/>　　咦！这是怎么会事。
<br/>　　青牛山山谷中，有一批戎族的宝藏，这笔宝藏藏了很久，这次正好在青牛山附近，程大雷便派高飞虎去取出来。
<br/>　　他和刘悲、徐神机一起来到军营，高飞虎的人正在休整。一见程大雷过来，高飞虎就立刻叫苦道：“大当家，那里没有你说的东西啊。”
<br/>　　“不应该啊。”程大雷心里琢磨着，难道是这件事情发生了意外。
<br/>　　“大当家，我们去的时候，发现山谷有被挖掘的痕迹。你说得那笔宝藏，估计是被别人挖走了。”
<br/>　　“别人……”程大雷皱起眉头：“会是哪个别人呢？”
<br/>　　程大雷也猜不到是谁半路截胡，抢走了本该是自己的宝藏。不过这件事程大雷猜也猜不到，暂时也只能把这困惑压在心底。
<br/>　　“回来就好，先歇一歇，之后我们还有别的事情要做。”程大雷。
<br/>　　高飞虎却没有疲惫的意思，道：“大当家，我刚回来便听说了，这几天打得可是很热闹。”
<br/>　　“是呐，打得很热闹。”程大雷轻轻叹了口气。
<br/>　　短短时间内，蛤蟆军扫荡过草原，一时间风声鹤唳，那副狰狞丑陋的***，如今成了草原上最恐怖的标志。

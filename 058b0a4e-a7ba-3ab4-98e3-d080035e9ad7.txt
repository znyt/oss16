　　第703章
<br/>　　田七打量着面前的汉子，汉子顶天立地，身上一股英武勃勃之气。但若说他有万人敌的本事，田七还是不太相信。
<br/>　　卢俊义读懂了田七的眼神，他微微点头，脚下挪了半寸。田七的眼睛倏忽睁大，只见青石板铺成的地面，出现半个脚印。
<br/>　　他再也坐不住了，腾地站起来，冲卢俊义竖起大拇指。
<br/>　　“好汉子。”
<br/>　　卢俊义表情不变，只是向田七拱了拱手，似乎这对他来说并不算什么。。
<br/>　　田七复又坐下，双手摁着膝盖，长长叹了口气：“便是有卢英雄这样的好汉，事情怕也不太好做。”
<br/>　　难，难在缺少人手，最难的还是如何让这八百人神不知鬼不觉的潜入长安城。相府势力极大，耳目众多，一旦走漏消息被那位老人知道，有危险的不仅仅是百里胜一行人，更会威胁到那位高高在上天子的安全。
<br/>　　但事情也已经到了没有办法的地步，陷入绝地，也逼的当今天子也不得不铤而走险。
<br/>　　凉州城内，程大雷手扣在桌面上，长长叹了口气。
<br/>　　长安城的局势变化，将会影响到程大雷这边的情况。假若李乐天胜了，虽然这种可能性很小，那么自己就会有一段时间的安稳日子；但假若胜的是崔相，那么，不仅仅是凉州，整个帝国都将因这场变故而陷入混乱中。
<br/>　　虽然说浑水摸鱼，火中取栗，乱了未必对程大雷没有好处。但程大雷目前更需要安稳，他需要时间稳住目前的局势，消化刚刚吞下的凉州。
<br/>　　如果能帮，程大雷倒是不介意帮一帮李乐天。对于这位帝国天子，程大雷并不缺乏好感，但若让自己手下弟兄进入长安，搅入这场注定没有胜算的对决中，也是程大雷不想的。
<br/>　　说来说去，自己目前能做的，也仅仅是静观其变，袖手旁观，等着自己所希望的事情发生。
<br/>　　整理一下思绪，程大雷还是将注意力放在自己目前正着手去做的事情。目前凉州城没有太大的状况，正忙碌的一件事便是整顿军纪，扩充兵员。
<br/>　　目前算上降兵，程大雷手中的总兵力不过四万出头。这样的兵力守住一座城是没有问题的，但要说守住整个凉州，目前的兵力还严重不足。
<br/>　　程大雷透过脑海中的江山图，查看各城的情况。蛤蟆城的发展已经处于停滞状态，商业几乎为零，工业和农业却都不错。而朔方城则属于刚起步阶段，距离达成任务需要的十万人口，还差三万余。
<br/>　　至于凉州城，商业工业农业却都不错，只是军事力量有些不足。
<br/>　　要做的事情还有许多，却也只能一件一件去做。
<br/>　　程大雷想起自己还有三次抽奖机会，一直没舍得用，或者说不太敢用。今日正是今日，程大雷索性将抽奖次数用掉，看会不会有什么意外之喜。
<br/>　　嘟，恭喜获得顶级麦种，成熟期三个月。
<br/>　　目前蛤蟆城、朔方城、凉州城总的农田数量是多少，其实程大雷也并不太清楚，但想也知道是个天文数字。有了这顶级麦种，能为程大雷解决一部分粮食问题。三军未动，粮草先行，对于程大雷这种扯旗造反的主儿，粮食永远是重中之重。
<br/>　　嘟，恭喜获得大旗一面，附带技能：神圣力量。
<br/>　　神圣力量：提升士气，提高城中部下忠诚度。
<br/>　　这个技能类似于『齐心』，但很明显覆盖范围更广，技能作用更强。程大雷手下现在招募了许多人，三教九流什么人都有，恰好用这面大旗刷刷忠诚度。
<br/>　　嘟，恭喜获得绝世文官宋慈。
<br/>　　程大雷眨了眨眼睛，今天的三次抽奖，总之是都没有浪费。目前自己整顿军纪，正好需要一个铁面无私的人物，这宋慈精通刑狱，恰恰能做自己的监察官。
<br/>　　嘟，暴恶属性生效，随机出世武将马超。
<br/>　　程大雷倒抽一口凉气，这下玩大发了。马超马孟起，估计应该也是传说潜质，出场应当也是绝世等级。
<br/>　　古往今来，这姓马的都不太好对付。估计日后在战场上遇见，自己手下只有关张赵能制住对方。
<br/>　　也不知道他会落到谁的手中，自己说不得有可能将他抢回来，也好凑齐五虎上将。
<br/>　　京州，长安城。
<br/>　　一汉子背枪佩剑，披头散发，背后牵一匹银龙驹。如此蛮人打扮，在长安城这天子脚下还是蛮显眼的。
<br/>　　汉子昂起头，深吸一口气，道：“这里就是长安城么，果然好气派。”
<br/>　　长安城的老百姓天生就有一种傲气，路边一卖瓜的老汉撇撇嘴，道：“这里算什么气派，要到皇宫看看，那才叫真气派。”
<br/>　　汉子也不恼，道：“皇宫，就是皇帝住的地方么？我去了那里就能见到皇帝么？”
<br/>　　老汉真有些受不了这乡巴佬，道：“当然了，皇帝自然住在最好的地方，有最好的女人，你去了皇宫就能见到皇帝。对了，你从哪里来的？”
<br/>　　“我……”汉子手指向西方，忽然有些茫然，最好手指向前面：“我要去皇宫。”
<br/>　　帝国皇宫还是蛮好找的，沿中轴线一直往前走就能走到皇宫。皇宫宫墙深深，外墙内墙，李乐天住在未央宫，平常也没理由走到皇宫外围。
<br/>　　今天偏偏是个例外，皇宫西北养着一群乌鸦，李乐天百无聊赖过来喂鸟。
<br/>　　就听得西门乱哄哄一片，李乐天皱眉道：“发生什么事了？”
<br/>　　天子一怒，浮尸千里，血流成河，身边跟随的小黄门立刻打探清楚，回来禀报：“启禀天子，有人闯宫，意图行刺。”
<br/>　　“啊？”李乐天不由得咧开嘴，虽然自己这皇帝没多大权力，可光天化日就敢过来行刺，也太不把自己当会事了吧。
<br/>　　“走，去看看。”
<br/>　　他丢开手中的稻谷，迈步向宫门走去，小黄门碎步跟在其身后，道：“陛下小心，陛下……”
<br/>　　宫门洞开，七八个侍卫东倒西歪，正有负责守卫皇宫的鱼龙卫集结。中间大门立着一个汉子，身上冒着热汗，一身生龙活虎的劲头。
<br/>　　“呔，哪里来的莽汉，擅闯宫门，不知死活，不知这里是什么地方么？”
<br/>(本章完)

　　“休伤我家宗主！！”
<br/>　　那两名云丹宗长老冲了上来，想要阻止叶尘。
<br/>　　可惜，以他们的实力，连慕容丹都远远不如，而且又没有镇天印那样的法宝，如何会是叶尘的对手？
<br/>　　轰！
<br/>　　轰！
<br/>　　叶尘直接随手两拳，两人同时发出一道惨叫，犹如炮弹般，倒飞了出去，狠狠坠落在地上，便再也爬不起来了。
<br/>　　远处那些云丹宗的其他高手，见叶尘如此威武，原本还想跃跃欲试之人，顿时吓得又缩了回去。
<br/>　　“死吧！”
<br/>　　叶尘没有理会周围那些人，再次一拳朝下方的慕容丹砸落下去。
<br/>　　然而就在这时，又响起了一道好听的女子声音。
<br/>　　“不要！”
<br/>　　眼看叶尘这一拳下去，慕容丹就算不死也得重伤，一道倩影忽然出现在他的前面，拦住了叶尘的去路。
<br/>　　此人自然正是冷蝉衣。
<br/>　　叶尘见状，来不及多想，只好将拳头朝一旁偏移。
<br/>　　轰隆！
<br/>　　百米外地面之上，顿时出现了一个巨大的深坑。
<br/>　　冷蝉衣也算是叶尘前世的朋友，而且刚刚自己危难之时，若非她开口阻止慕容丹，叶尘也不可能如此顺利突破到元婴境。
<br/>　　所以对于这个女人，叶尘自然无法痛下杀手。
<br/>　　冷蝉衣突然挡在慕容丹的面前，几乎是下意识的动作，并没有想到，叶尘会对她手下留情。
<br/>　　看到叶尘没有杀她，冷蝉衣眼眸中顿时透过一抹感激之色，双腿微微一弯，跪倒在了地上，哀声道：
<br/>　　“叶前辈，求求您，放过我师尊吧！他只是一时鬼迷心窍……”
<br/>　　叶尘眉头顿时微微一皱，
<br/>　　按理说，慕容丹对他起来杀心，还想要抢他的修炼功法和吞天鼎，此人非杀不可。
<br/>　　但他前世，曾受过云丹宗祖师的恩惠，也不想和冷蝉衣结仇，若是杀了云丹宗的宗主，今后和云丹宗以及冷蝉衣之间，这个梁子便算是彻底结下了！
<br/>　　一念及此，叶尘瞥了一眼冷蝉衣身后，已经气息萎靡的慕容丹，冷冷道：
<br/>　　“要我不杀他也行，他必须自废丹田，今后不得再担任云丹宗的宗主！”
<br/>　　“这……”
<br/>　　冷蝉衣顿时一愣，虽然她也知道，叶尘能够答应不杀她的师尊，就已经是天大的恩惠，可是让一向心高气傲的师尊，自废丹田，沦为废人，只怕他未必会同意。
<br/>　　果不其然，冷蝉衣身后，慕容丹拼尽了力气，勉强从地上坐了起来，反而哈哈大笑起来，状若癫狂，
<br/>　　“小子！成王败寇！本宗主实力不济，输给了你，你杀了我便是，想要我自废丹田，你休想！”
<br/>　　说完这话，慕容丹突然一咬牙，眼眸中闪过一抹决然之色，继而双腿在地面上狠狠一蹬，
<br/>　　砰！
<br/>　　下一刻，
<br/>　　慕容丹犹如炮弹般，自深坑中弹起，朝叶尘爆射而来，
<br/>　　“本宗主今日要和你同归于尽！！”
<br/>　　随着声音落下，慕容丹的身上，陡然爆发一团炙热的赤色火焰，仿佛整个人都要被火焰融化了一般，脸上则全是狰狞扭曲之色，眼眸中透着浓浓的杀意！
<br/>　　这慕容丹，竟然要自爆！！
<br/>　　“师尊！不要啊！”
<br/>　　下方的冷蝉衣顿时惊呼了一声，可惜以她的实力，根本没有能力去阻止。
<br/>　　轰隆隆！
<br/>　　慕容丹仿佛化身成为了一颗璀璨的明星，速度更是飙升到了极致，几乎转眼之间便奔袭到了叶尘跟前，
<br/>　　彭！
<br/>　　随着一道惊天巨响，慕容丹犹如一枚人形核弹，在九天之上爆炸开来！
<br/>　　轰隆隆隆隆！
<br/>　　方圆百里之内的虚空，瞬间全部破碎开来，形成了一个巨大的黑洞，璀璨夺目的光芒，久久不息。
<br/>　　远处的云丹宗一众高手，看到这一幕，顿时也全都纷纷握紧了拳头。
<br/>　　“宗主大人舍生取义，定能将那小子一举轰杀！”
<br/>　　“能够逼得一名化神天君自爆，那小子就算是死，也死得其所了！”
<br/>　　“宗主大人虽然死了，可至少为我们除掉了一个心腹大患！”
<br/>　　……
<br/>　　在众人看来，一名化神天君的自爆一击，便是返虚境的强者，也不敢正面撼其锋芒，眼前的这个神秘小子，就算再厉害，也不能活着。
<br/>　　过了许久，等那璀璨的光芒，渐渐消散下去，破碎的空间，也渐渐恢复如初。
<br/>　　只见那九天之上，空无一物，唯独剩下一道淡淡的虚影。
<br/>　　那是慕容丹的残魂！
<br/>　　“哈哈哈！小子，虽然本宗主马上就要魂飞魄散，可终究是我笑到了最后！”
<br/>　　慕容丹的残魂，站在九天之上，开始得意的狂笑起来。
<br/>　　而云丹宗的其他众人，顿时也纷纷松了一口长气，
<br/>　　虽然慕容丹也命不久矣，但至少在临死之前，为宗门除掉了一个心腹大患，也算是不幸中的万幸了。
<br/>　　然而，
<br/>　　就在这时，
<br/>　　忽然一道淡漠的声音，陡然在天地之间响起，
<br/>　　“慕容老儿，你真以为自爆，就能杀得了我吗？”
<br/>　　声音落下的同时，慕容丹前方千米外的虚空，忽然生生裂开一条缝隙，一道瘦削略显狼狈的身影，从那道裂缝中，缓步走了出来。
<br/>　　“什么！！”
<br/>　　不但慕容丹懵了，周围所有人，也全都懵了。
<br/>　　此时的叶尘，虽然略显狼狈，可是气息依旧浑厚如初，似乎只是受了一点轻伤而已。
<br/>　　“不！不可能！这怎么可能！！”
<br/>　　慕容丹看到这一幕，脸上的笑容瞬间凝固，取而代之的是满脸惊恐之色。
<br/>　　他一个化神境后期的强者，拼死自爆一击，居然依旧没能杀死这个只有元婴境的小子！
<br/>　　这世界是怎么了？疯了吗？
<br/>　　慕容丹又哪里知道，
<br/>　　作为前世修炼了八百年，经历过无数次战斗的狂帝叶尘，在慕容丹想要自爆的时候，他就已经察觉到，所以事先取出了逍遥仙珠。
<br/>　　等到慕容丹真的自爆，叶尘便第一时间操控着逍遥仙珠，破开空间，躲进了无尽虚空之内，成功过躲过了对方自爆一击，虽然依旧有所波及，但是受伤很轻，根本不足为道。
<br/>　　“这世上，没有什么事情是不可能！”
<br/>　　叶尘冷冷一笑，随手屈指一弹，
<br/>　　彭！
<br/>　　慕容丹最后一缕残魂，也破碎开来，彻底消散在世间。

　　第685章
<br/>　　崔白玉抬起头，她几日间水米未打牙，此刻完全是靠一腔恨意在支撑身体。
<br/>　　其实也谈不上恨，成王败寇，她比所有人都明白这个道理。她只不过是用自己的冷漠表达对程大雷的不屑，试图假装自己没有输。
<br/>　　然而心底最深处明白，自己已经输了。更可怕的是输掉之后的后果，以自己的美貌，以程大雷的残忍，定会百般折辱自己，对一个女人来说，那样的经历索性还不如死掉。
<br/>　　这几日数次求生，为的便是这一点。
<br/>　　本以为程大雷会极力阻止自己的死亡，毕竟自己活着要比死了更有价值，而她也明白，自己的美貌对世间男人来说是怎样的诱惑。只要上了这乱世牌局的人，都该知道怎么抉择。
<br/>　　可程大雷仿佛偏偏不懂，竟将剑抛在自己面前，等着自己自裁。现在自己距离死亡已经很进，只要一个弹指的功夫，就可以用剑割破喉咙，宣布在这场乱世中退场。
<br/>　　死代表永逝，表示一切为零。任你崔白玉是如何智慧，如何美貌，勉强在世间留下的一鳞半爪，也会在俗世浊浪下消失的无影无踪。
<br/>　　而这样的结局，是否真的是你想要的？
<br/>　　没有人不怕死。
<br/>　　在死亡触手可得时，崔白玉终于发现了自己内心深处的一丝胆怯。这发现令她沮丧，也令她愤怒。
<br/>　　她昂着头，不服气的盯着程大雷，的确是在认认真真打量对方。
<br/>　　不管是相府还是宋伯康，都透过各种渠道收集程大雷的资料。最后这些资料汇聚到崔白玉案前。她逐字逐句在孤灯长夜翻阅这些资料，为的就是搞懂程大雷是个怎样的人。
<br/>　　她还没来得及搞懂这一点，程大雷便掀翻了桌子，自己成了他的阶下囚。
<br/>　　如今，终于有个机会和程大雷面对面，她一定要弄明白这一点，否则死不瞑目。
<br/>　　目光死死盯着程大雷，试图透过表像，看穿程大雷的骨，看穿他的心肺。
<br/>　　的确是个平平无奇的男子，相貌粗鄙，身材魁梧，符合普通人对山贼的所有想象。那一双铜铃般的眼睛，也没什么所谓智慧的光芒。
<br/>　　当真看不透呐。
<br/>　　崔白玉更加苦恼，她自豪资本便是自己识人的绝技。普通人往她眼前一过，她就能判断得八九不离十。可面对程大雷，这绝技竟然失效了，他看上去当真太过简单。
<br/>　　可这样一个人，又怎能用简单两个字评价。
<br/>　　程大雷被盯得有些发毛，心道这姑娘在做什么。委实不怪崔白玉无能，实在程大雷本就很简单。可崔白玉偏偏要将他往复杂想，也只能是越想越复杂。
<br/>　　“谁，谁在帮你？”
<br/>　　崔白玉半晌不开口，开口便吓得程大雷一激灵。
<br/>　　第一时间他就领悟崔白玉在说什么，要不是有系统鬼鬼祟祟的帮忙，程大雷怎会有今天。可能看破这一点的，崔白玉还是第一个。
<br/>　　惊讶稍纵即逝，程大雷恢复冷酷，冷冷笑道：“本当家受命于天，仙人梦中传艺，这种事，难道会说给你听。”
<br/>　　程大雷身上稀奇古怪之处太多，如苏樱、徐神机这些人很早就发现了。程大雷往往以鬼神之说遮掩，他们也未必完全相信，只是没有深究。可蒙骗徐神机容易，蒙骗崔白玉的确有些勉强。
<br/>　　“天，哪个天，天子的天？”崔白玉隐隐觉得自己接近真相，不过，若说程大雷是明帝埋下的钉子，其实还有许多说不通之处。但这已经是最合理的解释。
<br/>　　程大雷蹲下身，两眼直勾勾盯着崔白玉，一手指天，斩钉截铁道：“老天的天。”
<br/>　　崔白玉有那么一刹那的愣神，她忽地扑过来，张开嘴咬住程大雷的耳朵。
<br/>　　程大雷赶忙挣开，耳垂还是被咬破流出血来，他气恼无比，将崔白玉推开。
<br/>　　“你属狗的么！”
<br/>　　程大雷郁闷得紧，刚才大家打哑谜不是玩得蛮开心么，怎么说着说着，动起口来。
<br/>　　这要是传出去，还不得说自己贪图她的美色，最后偷鸡不成蚀把米。可自己当真是什么都没干呐。
<br/>　　程大雷一指高力士：“你可要为我作证，本当家是清白的。”
<br/>　　高力士一阵无语：大当家还在乎清白这种东西。
<br/>　　崔白玉歪倒在地上，强撑着直起上半身，嘴角还渗着血。
<br/>　　“程大雷，我既然落到你手中，要杀要剐你便给我个痛快，若想侮辱践踏我，我死也不想。”
<br/>　　程大雷并不是没有这份心思，但也的确一直在克制。成王败寇不假，但也有另外一句话：士可杀不可辱。他是希望崔白玉自裁的，如此美人，生是骄傲的生，死也该骄傲的去死。
<br/>　　“崔姑娘，道不同不相为谋，我给了你路，为了自己的体面，早早上路吧。”
<br/>　　崔白玉恶狠狠盯着程大雷，二人的目光在空中交汇，在某一刹那，崔白玉读懂了程大雷眼神中的一丝怜悯。
<br/>　　为何自己会沦落到如今的地步，需要被一个山贼怜悯了。
<br/>　　崔白玉眼中的恨意越来越淡，她咬着牙，下唇被咬破，血流出来，一双美眸中，忽地啪嗒啪嗒落下泪来。
<br/>　　她没有哭出声，但的确哭了。
<br/>　　体面，体面……自己一直是个很体面的人。养尊处优，锦衣玉食的程度根本是程大雷这种乡巴佬能够相信的。崔白玉也一直认同这一点，她觉得自己不输男儿，生将从容淡定，赢得秋风扫落叶。死将慷慨悲歌，饮酒三杯，大笑而去。
<br/>　　不过，想是如此想，可真正遇到绝境时才发现，自己……其实没有那么体面。
<br/>　　她会暴走，就是想程大雷能在一怒之下挥剑斩了自己。可程大雷不知怎地怎么想的，偏偏打定主意要她自裁。
<br/>　　崔白玉这个时候忍不住想要爆粗口，妈的，我如果不是怕死，不干脆拿起剑自刎，何至于此。
<br/>　　怕死便是求生，为了求生，自己会慢慢献出自己所有的东西。体面、尊严、骄傲以及最后唯一的资本。
<br/>　　一个本来骄傲的人，会变得越来越不体面。
<br/>　　相比之下，这种惩罚比直接杀了她更加残忍，因为程大雷击碎了她心中最骄傲的东西。
<br/>(本章完)

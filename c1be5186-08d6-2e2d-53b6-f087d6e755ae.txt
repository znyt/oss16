　　　　盘古殿中，祖巫帝江听到烛九阴的话后，当即便下令屠杀人族。
<br/>　　“大哥，这样做不好吧，人族毕竟是我巫族的联盟，而且有好多巫族已经融入到了人族之中，这样做会不会……”
<br/>　　后土面带忧色道。她性子柔弱，是一个喜欢和平的人，不喜欢打打杀杀，她觉得人族不错，为了一个模糊的未来，就要屠杀人族，使得亿万生灵遭殃，她还做不到。
<br/>　　“如此屠戮人族，以魂魄炼制戮妖幡，怕是有伤天和。况且，我们与人族结盟，此举怕是……怕是有些不妥。”
<br/>　　“小妹，你太善良了，人族重要还是巫族重要？你也知道，人族潜力惊人，如今实力已经如此强大，若是再给他们一段时间，那还了得？人族必须削弱，要不然我巫族没有崛起之日。”帝江面不改色，威严道。
<br/>　　“可是，大哥……”后土祖巫急声道。
<br/>　　“好了，没什么可是的。若是小妹不想去，你就不要去了。各位兄弟，前去召集巫族儿郎们，准备杀向人族。”帝江打断后土的话，威严道。
<br/>　　“是！”
<br/>　　接着，他随着众祖巫一起出了盘古殿，准备召集人马之事。
<br/>　　“唉……”
<br/>　　后面，后土祖巫一声幽深的叹息，神色有些复杂。她知道一旦帝江做出决定，就不会改变。她隐隐觉得巫族不能这样做，要不然会后悔的，更何况，她对人族武祖玄天这人有些看不透，觉得此人不简单。
<br/>　　“小妹，不必担忧，我巫族不会有事的。”
<br/>　　十二祖巫中剩下的女祖巫玄冥留下来，看见后土的状态，以为她是在为巫族未来担忧，便对她轻声安慰道。
<br/>　　“但愿如此吧！”
<br/>　　后土感叹一声，便没有再多说。之后，随着众祖巫出去了。
<br/>　　……
<br/>　　东海之滨，一处隐匿虚空之中，永恒岛正漂浮在此处。
<br/>　　此时，永恒岛主宣来他的四位弟子。玄青、玉玲珑、白衣、风晓月站成一排，对着永恒岛主恭敬道：
<br/>　　“老师！”
<br/>　　行完礼后，大弟子玄青对着岛主问道：“不知老师唤我等前来，所谓何事？”
<br/>　　永恒岛主高坐云台之上，伟岸无比，道法玄妙，让人看不真实。他看着自己眼前的四位弟子，都满意的点了点头，当初四位弟子资质虽然一般，但有了自己的帮助，他们自己又足够努力，自龙汉劫一直修炼道如今，都已经成了准圣高手了，虽然只是准圣初期巅峰，但已经可以面对准圣中期了。
<br/>　　“嗯，不错！看来你们平时很努力，没有枉费吾之一番苦心。”永恒岛主点了点头，毫不吝啬的夸赞道。接着脸色一正，严肃道：
<br/>　　“洪荒如今三族****，巫妖两族大肆屠戮人族，人族道统不该灭。此次叫尔等前来，便是要让你等前去助人族一臂之力。更何况，你们在岛中待的时间已经够长了，是该出去走走了。”
<br/>　　“弟子知晓！”
<br/>　　四人相互对视一眼，眼中全都闪过一丝兴奋，同时对这岛主恭敬道。
<br/>　　“嗯，去吧！”岛主淡淡点了点头，道。
<br/>　　“是！”
<br/>　　四人再次对视一眼，同时转身离开，出了宫门，化作四道了流光，向着岛外飞去。
<br/>　　海面上虚空一阵波动，四人从虚空之中走了出来，看着一望无际的大海，玉玲珑对着玄青问道：
<br/>　　“师兄，我等如今该如何？”
<br/>　　玄青看了一眼大海，道：“既然老师让我等去助人族一臂之力，那么，我等也不该让老师失望，我们还是先去人族祖地看看，到时再说其他。”
<br/>　　“好。”其他三人都同意了。
<br/>　　顿时，四人化为四道流光，向着远处飞去，瞬间便已消失不见。
<br/>　　……
<br/>　　祖巫一下令，各地的巫族人员纷纷动手，将目标盯住周边的人族。人族因为是匆忙召集律令，大部分人族来到了首阳山，但依旧有不少部落还没有搬到首阳山，而这些部落，便是巫族与妖族下手的目标。
<br/>　　不到半月时间，在两族的屠戮之下，洪荒顿时就掀起一番腥风血雨。巫妖二族的屠戮，毫不收敛，就这样明目张胆，将一个个在外的人族部落连根拔起，或遭受屠戮，或沦落为奴隶，一个个昔日繁华的村寨，境界被付之一炬，人死了，楼也毁了。
<br/>　　随着屠巫剑与戮妖幡的炼制，对魂魄与精血的用量越来越多，巫族将目光投向首阳山的人族身上，不过首阳山毕竟是人族祖地，更何况如今人族正在与妖族开战，他们想要动手，就必须派遣更强的队伍前去杀戮。
<br/>　　祖巫们发现，屠戮人族后，有近两成的魂魄不知去向，欲炼制十二枚戮妖幡需要更多的人族，于是他们纷纷派遣手下的顶尖大巫前去首阳山屠杀人族，如刑天、相柳等。
<br/>　　而在此时，人族武祖玄天出关了。
<br/>　　当他看到巫、妖两族毫不收敛的肆意屠戮人族后，顿时大怒。
<br/>　　“啊！”
<br/>　　玄天口中猛然暴起一声惊天长啸，代表着这位人族武祖，彻底进入暴走的状态。
<br/>　　“巫族，背信弃义，背叛联盟，该死！妖族，屠杀人族，收魂炼剑，该杀！”
<br/>　　但随着玄天一声长啸，他的周身一股恐怖的波动彻底荡漾开来，一股不弱于天道圣人的庞大威压，彻底爆发！
<br/>　　乳白色、黑白色、殷红色，三道神轮宛如大日一般，悬浮于首阳山最高峰，如日中天，恢弘大气，一股岁月，生死，命运的气息，从三道神轮之上创出。神轮旋转，带动岁月的波动，一瞬间，便收去无数巫、妖两族强者的生命。
<br/>　　霎时之间，首阳山中巫、妖两族众人只觉得周身感到一阵冰冷，体内妖元、巫元都仿佛为之生涩，开始变得流转不通，甚至就连神识都看不清周围的情况，眼前闪过一片迷迷糊糊的三道光华，肆意流转之间，全都是涌动不休的杀气，他们已经被玄天的气机彻底封锁了。
<br/>　　愤怒，愤怒，极致的愤怒！杀戮，杀戮，疯狂的杀戮！
<br/>　　天地风云瞬息之间为之一变，人族武祖，玄天携着滔天之威，岿然降临，岁月之轮，生死之轮，命运之轮，三轮轮转，轰然之间，席卷寰宇乾坤，首阳山附近的数万巫妖大军，一瞬间便死伤过半。
<br/>　　洪荒世界就是如此，就算数量再大，也绝对没有可能威胁到顶尖的大神通者，因为这样的人，都是一群超脱天地规则的存在，天地之间的自然规则一般很难在给他们造成什么损伤，唯有同等级的存在，才能带给他们足够的威胁。
<br/>　　随着武祖玄天携着无尽神威出场，瞬息之间，形势逆转！
<br/>　　……

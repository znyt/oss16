　　“你们说起食物，我今天听不少人说了，军部已经开始安排工农会的人去种地了，专门开辟出了一个地方种植小麦和玉米。另外还有专门的鱼塘喂养鱼苗，今天刚从工农会那边要过去了三十多个有过养殖经验的人过去。”
<br/>　　苏云道。
<br/>　　“可是，这群东西要吃上，也得等好几个月了吧。”
<br/>　　陈勾道。
<br/>　　“那可不一定了，我听说军部好像合成出来了某种强化液，可以缩短种子和鱼苗生长周期的。”苏云道。
<br/>　　“哦？还有这种事情？那这可是大好事啊。”陈勾意外道。
<br/>　　林峰点了点头，“不过估计到时候就算长出来了，也不是一般人吃得起。”
<br/>　　“林峰，你去把李广那群人叫过来吧，今天下午我去买了一袋子米，然后今天做饭吃。”
<br/>　　苏云忽然道。
<br/>　　“饭？”
<br/>　　陈勾等人都愣住了。似乎对这个曾经非常熟悉的名字变得无比的陌生。
<br/>　　这几天。
<br/>　　他们吃的不是罐头就是面包饼干，早已经忘记了曾经吃腻了但是每天都不得不吃的白米饭的滋味。
<br/>　　“是啊，军部有粮仓，一斤要三十颗晶石，特别贵，我一口气买了三斤，然后我又去花一晶石买了个电饭煲，老板还送了我几副碗筷。到时候蒸饭吃。”苏云笑道。
<br/>　　林峰等人面面相觑，电饭煲，放在末世前那怎么也得好几百了，甚至一些高科技点的上千也不是问题，但大米，除去那些珍贵大米，寻常点的，一斤也就二三十块钱顶天了。
<br/>　　可是现在一斤米要足足三十颗晶石，而一个电饭煲只要一晶石。
<br/>　　李广等人一听有饭吃了，一群人都是冲了出来。
<br/>　　“我去，听说有饭吃，嘶，我已经闻到了饭香味。”
<br/>　　李广吴三一脸的陶醉。
<br/>　　一群人就这样眼巴巴的盯着电饭煲。
<br/>　　这一幕在末世前是绝对难以想象的。
<br/>　　谁会盯着一个电饭煲看个半天，谁会听到有米饭吃会流口水。
<br/>　　可是现在就是如此。
<br/>　　“把门关上，免得米香被别人闻到了，引起麻烦。”
<br/>　　林峰留了个心眼。
<br/>　　陈勾立即把门关了，甚至窗帘都拉上了。
<br/>　　很快。苏云就端着一碗碗白米饭出来了。
<br/>　　一群人立即都感觉肚子咕咕咕叫了。
<br/>　　“且慢！”
<br/>　　忽然林峰站起身道。
<br/>　　“干什么峰哥，大家都饿了。”
<br/>　　李广捂着肚子道。
<br/>　　“是啊，林峰有什么事情，吃完饭再说可不可以。”
<br/>　　陈勾说道。
<br/>　　他两百斤的胖子，如今已经饿得只剩下一百七十斤了。
<br/>　　其余人也都眼巴巴的看着林峰。
<br/>　　“别急啊，我也没有说不让你们吃啊，只是这干吃白米饭，是不是有点没有多少情趣。”
<br/>　　林峰嘿嘿一笑。
<br/>　　“得了吧峰哥，现在情况你也不是不知道，有米饭吃就不错了，毕竟如今要想吃碗熟米饭，首先他得有米，其次他得有电饭煲，然后他得有电，这可不简单。”李广道。
<br/>　　林峰也懒得废话，直接拿出了空投里搞到的牛肉罐头、带鱼罐头和老干妈。
<br/>　　“握草，老干妈？”
<br/>　　李广陈勾刷的一下站起来了。
<br/>　　“这特么可是好东西啊。”
<br/>　　“峰哥，你居然有这玩意？我爱死你了。”李广眼睛都冒光。
<br/>　　“行了，瞧你们这饿狼样，都分了吃。”
<br/>　　林峰全部给他打开，顿时牛肉罐头的肉香，带鱼罐头一种独特的腥味，外加老干妈的香辣味道混合着米香味是在这个不大的房间内充斥着。
<br/>　　而此时此刻。
<br/>　　在这个充满了老干妈和米香味的房间外面。
<br/>　　越来越多的幸存者涌入了这个代号为曙光的基地。
<br/>　　曙光代表的是希望，代表的是驱逐黑暗。
<br/>　　每个人幸存者眼中都充斥着兴奋。
<br/>　　然而这个兴奋很快就没了。
<br/>　　因为基地不提供任何的救济粮食。
<br/>　　就在昨天。
<br/>　　基地还提供免费的救济粥。东西不多，就一碗白米粥外加一些玉米粒。
<br/>　　可是。
<br/>　　这个免费的救济粥就持续了一日。
<br/>　　基地宣布取消了，日后不再提供。只是因为人太多，哪怕是一碗玉米粥，基地都承担不起了。
<br/>　　每个人都彻底的绝望了，当然那些有本事的或者自己是黑铁战士的，倒不至于吃不饱肚子，不过即使如此，要吃百米饭？还是奢侈的老干妈拌饭？一泡尿给你尿醒，做梦吧。
<br/>　　屋内。
<br/>　　“我陈勾就是从这里跳下去都不会吃这白米饭拌老干妈，嗯，真香。”陈勾笑嘻嘻道。
<br/>　　“对了，把这东西给忘了。”
<br/>　　林峰忽然拿出了烧烤架。
<br/>　　外加一些牛排，这是他上次没吃完的。
<br/>　　“牛排？我尼玛没有做梦吧？”
<br/>　　“我日哦，烤架都出来了？林峰，你是不是把哆啦A梦的口袋给偷来了？怎么这都有？这烤架内还有碳？”陈勾等人都震惊了。
<br/>　　苏云也是瞠目结舌，震惊道：“林峰，这些木炭哪里来的，这些东西价格现在很贵了。”
<br/>　　“这个就别管了，就说你们想不想吃吧。”林峰笑道。
<br/>　　“想，这特么谁不想，我都已经快忘记肉的滋味了。”
<br/>　　陈勾口水都快要流出来了。
<br/>　　“峰哥就是牛逼！”
<br/>　　李广等人个个都是由衷的佩服。
<br/>　　“跟着峰哥走，吃喝都不愁！”
<br/>　　“行了行了，别拍我马屁，谁有火？”
<br/>　　林峰的火焰符箓消耗完毕了。
<br/>　　“巧了！我有打火机！”
<br/>　　陈勾立即拿出打火机。
<br/>　　“老师，这个时候了，就别收我的打火机了嘿嘿。”
<br/>　　苏云翻了个白眼，然后拿了几张纸过来。
<br/>　　“这些纸现在不值钱，我随手买了一些，没事的时候寻思着写个日记之类的，就像是《安妮日记》。”
<br/>　　安妮日记是犹太人被屠杀时，一个小女孩藏身密室时的生活和情感的记载。
<br/>　　“老师放心吧，我们现在的境遇可没有那么糟糕，至少我们有可以反抗的力量，只要人类还会反抗，那么就代表人类还没有失败。”
<br/>　　林峰认真道。
<br/>　　苏云楞了一下，旋即笑道：“也是，毕竟有林峰同学在呢。”
<br/>　　一群人点了火，把木炭点燃后，开始烧烤牛排。
<br/>　　很快。屋内充斥着一股肉香。

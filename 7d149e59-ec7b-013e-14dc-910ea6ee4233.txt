　　第664章
<br/>　　程大雷不好打，这是自然，假若好打的话，宋伯康也不会容忍到现在。
<br/>　　一天又一天，程大雷成了尾大不掉的祸患，已经到不得不动手，并且要为之付出一定代价的地步。
<br/>　　凉州五座城，宋伯康占了四座，程大雷占了一座，另外程大雷又开拓出一座蛤蟆城。如今，宋伯康能够出动的兵力在十五万左右，单纯从兵力来讲，他是占了绝对优势的。
<br/>　　然而程大雷毕竟是令人忌惮的存在，全军出击，能否消灭程大雷，是一件谁也说不好的事。
<br/>　　戴白提议向京州借兵，便是也觉得，消灭掉程大雷是件极为困难的事情，以目前宋伯康手中的兵力怕是还做不到。
<br/>　　宋伯康沉思片刻道：“最后还是要崔相大人提供一些兵器支援，若能领兵而来，这是最好，但相府处怕也没有富裕兵力。”
<br/>　　以宋伯康的心思来讲，是不想有外地兵力进入凉州。他现在已经不想再被相府控制，当然，这还是他个人的小心思，现在还不敢公布于众。
<br/>　　如果能灭掉程大雷，自己彻底掌握凉州后，方才有向崔相提条件的资格。
<br/>　　宋伯康磨刀霍霍，大势已经是箭在弦上，不得不发。
<br/>　　而程大雷此刻自然也是没闲着。他交待完所有事情后，便与徐神机离开朔方城，快马加鞭返回蛤蟆城。
<br/>　　刚刚回城，程大雷根本没有歇息，便召集城中的主要人员，召开一场军事会议。
<br/>　　参加会议的有刘悲、徐神机、赵子龙、秦蛮、吴用、阿喜。因关鱼、张肥与和珅留在朔方城，没有参加这次会议。
<br/>　　开会就是这样，有时候人数越少，反而越加重要。
<br/>　　这次会议也并非是在城主府的大厅，而是在城主府的书房，几个人聚在一起，也没有端茶倒水的丫鬟。
<br/>　　阿喜将凉州地图在桌上摊开，图上标着五座城池的位置。
<br/>　　“聊聊吧，大家有什么想法？”程大雷抿了口茶道。
<br/>　　宋伯康控制了四座城池，都是极其重要的战略位置，论资源比程大雷好，论人口比程大雷多。
<br/>　　程大雷一直有和宋伯康讨价还价的资格，其实就凭着几个字：光脚的不怕穿鞋的。真如果实力对比，程大雷并不占任何优势。
<br/>　　可目前的形势，程大雷已没有决定战争是否开始的权力。他只能去想，如何能在这场战争中获取胜利。
<br/>　　书房中，诸人开始发表自己的看法，诸人议论纷纷，时而因一个问题争吵不休。
<br/>　　打是一定要打的，关键是如何去打？战略已经确定，大家讨论的其实是战术。
<br/>　　保守些的，决定坚壁清野，应对宋伯康的攻击。想要打下两座城，对宋伯康也并不容易，让他付出一些代价，只要他付出的代价足够多，他便会选择收手。
<br/>　　激进些的，是主动出击，金城，武威，浩甲，凉州一座城一座城的打，消灭掉宋伯康的军事力量，将他驱逐或者消灭，从而占领整座凉州。
<br/>　　支持激进策略的有秦蛮、赵子龙、吴用。支持保守派的有程大雷……
<br/>　　只有一个程大雷。
<br/>　　程大雷歪靠在墙上，心情有些无语。自己手下这帮谋士武将，就没有一个正常些的么，一听要和宋伯康开战，每个人都像打了鸡血，没有人说想安稳些，保守些。
<br/>　　“大当家，我们究竟该如何动手，您拿个主意？”秦蛮道。
<br/>　　秦蛮看着像个稳重人，何时也被沾染得如此暴躁，现在也有些按捺不住。
<br/>　　“打是一定要打的，决定权已经不在我们手中。但目的是不变的，以安稳胜利为第一目的，而不是扩张地盘。大家想一想，如何在这场战争中取得胜利。”
<br/>　　吴用领导的军策府有无数谋士，做的就是动脑子的事情。关于他们的训练工作，程大雷一直没有停止过。
<br/>　　今天只是第一场会议，终究是不可能拿出一个确切的法子。接下来应该是夜以继日的讨论，甚至争吵。最后如何行动，还是由程大雷来决定。
<br/>　　阿喜的喜字队这段时间没办法歇着，要时刻刺探凉州城的情报，自己做备战的准备，同样也需要看对方是怎样准备的。
<br/>　　会议持续到晚上，诸人散去之后，程大雷也走出书房。凉风一吹，也清醒过来。
<br/>　　忽然他想到，眼前还有一场麻烦需要面对，便是苏樱和柳芷。在朔方城躲了些日子，但终究是躲不过去的。
<br/>　　他无可奈何的叹口气，向后院走去。
<br/>　　今天回来得着急，还没与苏樱见过面，他来到苏樱的院子，忽然发现有些冷清。
<br/>　　“大当家，你回来了，我伺候你洗脚。”柳芷过来道。
<br/>　　程大雷点头应承，柳芷蹲着身子，帮他将靴子脱了。程大雷目光转了一圈，还是没看见苏樱。
<br/>　　“人呢，苏樱去了哪里？”
<br/>　　“夫人去了落玉寨，随着二夫人一起去的。”
<br/>　　“咦……”
<br/>　　程大雷有些无语，原先樊梨花的确觉得闷得慌，想要回落玉寨。但没料到，她竟然把苏樱一块儿拉走了，还未与程大雷支会一声。
<br/>　　当然，她们想告知程大雷也不容易，毕竟程大雷一直躲在朔方城。
<br/>　　原本看见她们两个还觉得有些麻烦，可眼前突然不见了二人，程大雷又觉得空空落落的。
<br/>　　“出去散散心也好。”程大雷自我安慰一句：“我今夜还是在书房歇了吧。”
<br/>　　“是，我伺候大当家。”
<br/>　　军策府此刻怕还是灯红通明，讨论目前的局势。程大雷虽然把这件事交给他们做，但自己也不可能不去想。
<br/>　　毕竟，这件事已经关乎蛤蟆城未来存亡。
<br/>　　他点灯将自己关在书房内，一面翻看着喜字队递过来的情报，一面盘算蛤蟆城目前能利用的力量。
<br/>　　这并不是一件轻松的工作，枯燥到足以耗尽所有激情。程大雷忍着无聊，表面看上去安坐在那里，内心却是波涛翻滚。
<br/>　　书房分为里间外间，程大雷偶尔歇在书房，外间有一张小床，是供柳芷用的。有时她会走进书房，为程大雷端茶递水，给将要耗尽的油灯续上灯油，让它持续的燃烧下去。
<br/>(本章完)

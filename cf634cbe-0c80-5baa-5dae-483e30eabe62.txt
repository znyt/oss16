　　天尊殿中，元始天尊高坐于九龙宝局之上，看着下方大殿中央上的四人，心情不知为何，舒畅了几分。
<br/>　　下方玄青等四人也同样看着元始，他们神色间有些疑惑，不知元始留他们下来想要干什么。
<br/>　　之前自己等人还在东海龙宫做客，身为邻居，他们四人经常去龙宫游玩，一来二去地便也熟悉了。可下一瞬，他们便被移送道昆仑山玉虚大殿之中，见是元始在讲道，他们也明白了原因。可如今都已经讲道完毕了，为何还留下他们。
<br/>　　他们不解，但不多也不会多问，毕竟元始是他们的恩师。四人也不多想，便对着元始恭敬道：“拜见老师！”
<br/>　　“嗯。”元始点头应道，接着又说道：“此次将尔等四人留下，便是要与尔等一项任务，尔等需尽心去完成。”
<br/>　　任务？四人对视了一眼，同时说道：“老师请放心，我等必将尽力完成。只是……不知此任务是……”
<br/>　　“哦，你们也不比紧张，这件事虽说是任务，但只需尽力，也有你们一份好处。”元始笑了笑，接着道：
<br/>　　“人族需经历三皇治世，五帝定伦，确立三皇五帝，方可气运漫长，万劫不衰，坐稳洪荒主角之位。而如今三皇已成，五帝尚缺一人，本座与你师伯等人商议，这最后一人之教化便落在了我阐教之中。吾之本意，是在尔等四人之中，选出一人，作为人族大帝之师，助其成功登临帝位，治世天下，功德圆满。不知，你们可愿意？”
<br/>　　“我们愿意。”
<br/>　　四人一听，原来是这事，教化帝王，这事说难不难，说容易也不容易，只要好好教导，最后还有功德拿，这事好事啊，哪有不愿意的，于是连忙应道。
<br/>　　元始微微一笑，也不意外，接着道：“我收你们为徒，也有不少年了，你们也成长了，玄青身为大弟子，更是突破到了准圣，而其余三人也不远了，玲珑亦是半只脚踏入了准圣，此次人帝之师便给你吧。”
<br/>　　元始看了看玉玲珑，之后又对着另外三人道：“至于你们，便也随着去吧，相当于帮助玲珑，这对于你们没有坏处。”
<br/>　　“是，老师。”
<br/>　　四人自然是没有意见，答应下来。
<br/>　　“玲珑，尔等去人族都成，收夏禹为弟子，他便是下一任大帝，顺便将这个给他！”
<br/>　　元始对着四人说道，最后似乎是想到了什么，拿出一块玉简，刻录一些东西在里面，便扔给了玉玲珑，顺便说道：
<br/>　　“这是他修炼的功法。你们去吧！”
<br/>　　“是，老师。”
<br/>　　四人行了一礼，便躬身退下了。
<br/>　　在四人走后，元始摇了摇头，轻声道：“你们都走了，这玉虚宫倒是清净，不过，昆仑山似乎不是很不清净啊。”
<br/>　　说到这里，元始看向昆仑东方山脉，那里是截教的地盘。这些年的事情，元始都看在眼里，通天主张有教无类，凡是有毅力，求道之心坚固之人，都收入教中，来多少，他收多少。
<br/>　　这样的事情，短时间是好，可是时间长了，也就出问题了。截教中，三教九流，什么样的人都有，一个教派之中，更是还有数十个团体，各人都不服对方，弄得昆仑山乌烟瘴气的。有时甚至都常来阐教之中挑衅，双方都摩擦不断，必竟是两个教派，教义更是不同，出现如此情况，那是正常事情。
<br/>　　一山不容二虎。
<br/>　　而通天作为元始的弟弟，元始也不好多说他什么，所以他才会决定放自己的徒弟们下山。一来是因为他们的确合格了，再待在昆仑山上，也学不到多少，只有下山历练，才能进步。二来未必没有截教的原因。
<br/>　　元始暗自摇了摇头，叹了一口气，希望通天能够明白吧。封神的结局，其实自从截教出现开始，便早已注定。不是因为阐教的原因，也不是因为封神量劫的原因，而是因为天道。
<br/>　　天道最重平衡，截教号称万仙来朝，其教中仙人数量众多，遍布洪荒，已经隐隐有洪荒第一大教的气派了，门下弟子又良莠不齐，有好有坏，而又恰逢天庭确缺人，天道为了制衡，不拿通天开刀又拿谁呢？
<br/>　　希望通天能有所改变，元始本想劝说一番，可是又怕兄弟间闹出隔阂，一想到通天那犟牛般的脾气，元始便无奈一笑。
<br/>　　“罢了，未来的事，谁有说得准呢？该走的人都走了，本座也要出去走走了。”
<br/>　　语罢，元始便一挥手，将玉虚宫隐去，自己却向着昆仑外走去。一步便是数万丈，数十步便走出昆仑山，同时，一边走着，元始的形象也开始变化起来。
<br/>　　第一步走出，元始的衣服变成了一件宽松白袍，第二步踏出，元始的腰间多了一柄普通木剑，第三步踏出，他右边腰间多了一个葫芦，里面装着美酒，再一步踏出，他手中多了一把美人扇……
<br/>　　走出昆仑地界后，他的形象已经彻底变了，变成一个游戏于山水间的浪子形象，他模样潇洒，全身散发着独特的气质，那是一种自由的气息。任谁也认不出来，这就是鼎鼎大名的元始天尊。
<br/>　　元始抓起要变葫芦，仰头便是豪饮一番，同时狂笑道：“哈哈哈，好酒！好酒！”
<br/>　　“天尊有天尊的威严，浪子有浪子的潇洒，酒鬼有酒鬼的狂放，果然，每一重身份，便是一种不一样的体验……”元始喃喃低语道，任谁也没能听清他说些什么。
<br/>　　两三步走出，元始便消失于山野之间。
<br/>　　昆仑山，太清宫中，老子那盘坐了仿佛万古不变的姿势，终于动了，他睁开眼睛，疑惑地看了一眼昆仑之外，没有看到什么。之后又看向玉虚宫方向，瞬间便发现了玉虚宫的变化。他的脸色微微一变，喃喃道：“刚才那是……大兄？”
<br/>　　“大兄这是去干什么了？”尽管疑惑，但老子并没有多想，他也不但心，这洪荒之中，也没有几个能够奈何元始之人。
<br/>　　之后老子又看向截教方向，暗自摇了摇头，没有多说，便继续闭目悟道。
<br/>　　……
<br/>　　却说玉玲珑四人自出了昆仑山，便一直向着人族都城而去，根据元始给他们的消息，夏禹就在人族都城之中。
<br/>　　夏禹又名大禹，他的父亲鲧在皇城中担任官职，帮助帝舜治理水患问题，所以他被留在了都城之中。
<br/>　　玉玲珑四人来到都成，也没有耽搁，径直来到夏禹家，向他母亲表明身份后，要求收夏禹为徒。
<br/>　　夏禹母亲听是圣人门徒，大喜，自然是答应了这件事。于是，四人就这样简单的收了十岁的夏禹为徒。
<br/>　　四人以都城为红尘之地，不易修行为由，带着夏禹出了城门，找了一处灵山，随意打了个屋舍，四人就这样简单的住了下来。
<br/>　　拜师之后，玉玲珑将元始给他的玉简给了小夏禹，夏禹自由聪明，自然知晓此为何物，他将玉简紧贴额头处，瞬间，一段神秘玄妙的道诀传入夏禹脑海之中：
<br/>　　“九鼎镇世诀，修九鼎，聚九州，拢气运，镇世间……”
<br/>(本章完)

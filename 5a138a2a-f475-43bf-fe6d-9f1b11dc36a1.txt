        接下来，贺思思拉完一首表白曲后，又拿起了扩音器，对着楼上大声喊道：“明司寒，你给我出来，我要当你现在的女朋友，未来的老婆！”
<br/>
<br/>        她这一声吼完后，倒是仍旧未见明司寒的身影，只是兜头迎来一盆水，将她从头到肩泼得个狼狈不堪。
<br/>
<br/>        顿时，周遭惹来一阵嘲笑声和唏嘘声。
<br/>
<br/>        连星儿却并未笑，只是惊愕地抬起头，想知道是谁这么缺德，竟然往外泼脏水。
<br/>
<br/>        而就在这个时候，贺思思本能地看向连星儿，见连星儿仰头望着楼上，心里莫名其妙地感到窝火，直接大步流星地朝连星儿走去。
<br/>
<br/>        当围在前面的人，看到气势汹汹的贺思思走过来时，一个个纷纷向两边让道。
<br/>
<br/>        而道路的尽头，直达连星儿的面前。
<br/>
<br/>        连星儿回眸之际，只见贺思思已经扬起手，一巴掌掴了过来。
<br/>
<br/>        一阵冷风扑面。
<br/>
<br/>        连星儿还未缓过神来时，贺思思的手掌离她的脸颊只有那么几厘米远。
<br/>
<br/>        但是，她的手腕，此刻却被郎安给紧紧地抓住。
<br/>
<br/>        郎安依旧面无表情。
<br/>
<br/>        但随着他手下的力度加重，贺思思感觉到痛意后，眼泪瞬间在眼眶里打转。
<br/>
<br/>        刚刚还气势汹汹的她，此刻却哭丧着脸，呜咽着向郎安求饶：“痛！好痛！你快放开我！”
<br/>
<br/>        郎安这才一脸冷酷地甩开了贺思思的手。
<br/>
<br/>        贺思思见状，趁机推了连星儿一下。
<br/>
<br/>        郎安眼疾手快地将连星儿揽入怀中。
<br/>
<br/>        而这个时候，连星儿手中的那袋礼物被贺思思夺了过去。
<br/>
<br/>        他们还未来得及阻止，贺思思便把所有的火气发在了那袋礼物上。
<br/>
<br/>        贺思思将礼物袋狠狠地往地上砸去。
<br/>
<br/>        下一个瞬间，连星儿便隐约听到了瓷器破碎的声音。
<br/>
<br/>        紧接着，贺思思破口大骂：“连星儿，你这个女人真是犯贱！我跟司寒哥哥表白，关你什么事？你犯得着找人拿脏水泼我吗？还有！你要脚踏两条船到什么时候？一边跟我哥暧昧不清，一边还跟司寒哥哥纠缠。”
<br/>
<br/>        贺思思说着说着，目光下意识地落在郎安身上，见这个男生还护着连星儿，不禁咧嘴讥笑道：“呦！这不是脚踏两条船啊！原来，你是脚踏三条船！这世上，怎么会有你这么贱、这么渣的女人噢！”
<br/>
<br/>        楼上，男生寝室里，仲正浩成了“实时报道记者”，楼下一有点风吹草动，他就立马向明司寒、张子晋和邢乐游报告。
<br/>
<br/>        “我靠！那个女生似乎要跟连星儿动手了！”仲正浩激动不已地说。
<br/>
<br/>        一提起“连星儿”，明司寒连忙起身，直接从床上跳了下来，踏上拖鞋便径直冲出了寝室。
<br/>
<br/>        他的这种行为，倒是让留在寝室里的邢乐游、张子晋和仲正浩面面相觑。
<br/>
<br/>        明司寒几乎是一口气冲到楼下，等到他达楼下的前坪里时，只见连星儿扬起手便给了贺思思一巴掌。
<br/>
<br/>        贺思思被连星儿打得偏过了头去，她微微颤抖着抬起手来捂着被打的脸颊，眼中满是委屈。
<br/>
<br/>        而连星儿则跟郎安贴身站在了一起，趾高气扬的样子，像极了专横跋扈的千金大小姐。
<br/>
<br/>        就在这时，贺思思无意间看到明司寒时，立马开哭腔，疾步过去，扑入明司寒的怀中，抽泣起来：“司寒哥哥，连星儿她打我！不仅如此，她还找人从楼上泼我一身脏水！我到底做错了什么？她要这样对我？还有，司寒哥哥，我是来给你庆祝生日的！我……呜呜——”
<br/>
<br/>        连星儿听着贺思思的这些话，心里更是恼火，刚想反驳什么的时候，明司寒却在此刻开口了。
<br/>
<br/>        “连星儿，谁给你的胆子，打我的朋友的？”
<br/>
<br/>        明司寒虽然推开了贺思思的投怀送抱，但是却毫不客气地对连星儿怒吼了起来。
<br/>
<br/>        连星儿惊愕地瞪着明司寒，半晌后，她直接将刚刚从地上捡起来的礼品袋，直接塞到明司寒的手中，没好气地祝福：“祝你生日快乐！托你这位朋友的福，顺道祝你‘碎碎’平安！”
<br/>
<br/>        “？！”明司寒双手下意识地抱住礼品袋，瞬间一脸懵然地看着连星儿。
<br/>
<br/>        连星儿二话不说，径直往那边女生宿舍的大门口走去。
<br/>
<br/>        今天是星期六，寝室里顾千凝她们都不在，十有八九是出去逛街了。
<br/>
<br/>        连星儿一个人闷闷不乐地坐在书桌前，回想起明司寒刚刚那护着贺思思的样子，就一肚子的火气。
<br/>
<br/>        楼下，连星儿回寝室后，郎安也转身离开了。
<br/>
<br/>        贺思思依旧就像狗皮膏药般粘着明司寒不放：“司寒哥哥，我可是千里迢迢来为你庆祝生日的！无论如何，你今天都不能赶我走哦！”

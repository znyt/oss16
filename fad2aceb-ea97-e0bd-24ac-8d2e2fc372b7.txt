　　小九娇羞地笑了笑，眼睛却很明亮。
<br/>　　下楼的时候，又路过电影院，叶少阳见小九有些好奇，就带她进去，正好有一场电影要放映，叫《我不是药神》，叶少阳之前在浏览器推送里看到过这个电影，口碑好像不错，就买了票跟小九一起看。
<br/>　　看完出来，小九有些沉默，半晌说道：“看来人间也有很多不公平的事情。”
<br/>　　叶少阳叹道：“我是法师，只能管法术界的事，别的什么都管不了，不对，就连法术界的事很多我也管不了。”
<br/>　　小九挽着他的胳膊，安慰道：“不，少阳，你不要妄自菲薄，你现在做的事，非常了不起，如果平了太阴山，等于拯救了三界。”
<br/>　　她看着路上拥塞的行人，做什么的都有，来来往往，人生百态，尽在眼前，接着说道：“如果人间大乱，经历巨变，今天人间的这一切文明，可能都将不复存在，你能拯救这一切，虽然他们永远不会知道，但你仍然是英雄啊。”
<br/>　　英雄……
<br/>　　这个词，叶少阳从来想都没想过，自嘲地笑了笑，拉着小九的手往前走去。
<br/>　　回去之后，叶少阳翻出山海印，跟小九一起研究，两人一起摆弄了半天，小九也是一无所知，只好作罢。
<br/>　　“你休息吧，我去隔壁房间修炼去。”小九看了他一眼，起身开门，叶少阳有点尴尬，不知道是该挽留还是放任她走，小九已经关门出去了。
<br/>　　正好这两天四宝和吴嘉伟都没回来，房间空了很多。
<br/>　　瓜瓜去了空界，也没是回来，家里就他们两个了。
<br/>　　叶少阳听见小九脚步声在门外走远，内心好像被蚂蚁啃着似的，说不出的躁动，只好坐起来吐纳，两个周天之后，躁动倒是平息了，但困意一点也没了，翻了一会朋友圈就放下了——他至今都不太喜欢玩手机，无聊之下，他又拿出山海印在手中研究，突然眼前一黑，停电了。
<br/>　　停电了倒也没啥，他也没什么事非要点灯去做，但他想知道是都停电了还是自己家线路出了问题，想到床头柜里还有蜡烛，找了两根出来点上，一根放在床头柜上，一根拿在手里，出门到走廊上，看到应急灯都亮了，声控灯没反应，就知道是统一停电了，于是捧着蜡烛又回到卧室，吹掉一根蜡烛，爬上床去。
<br/>　　点着一根蜡烛，他又把山海印拿出来，对着蜡烛端详起来，本来也只是不死心，想要再看一看，并没抱什么希望。
<br/>　　然而，对着烛光这么一看，他立刻就发现了问题：
<br/>　　之前对着阳光或灯光看，根本看不出什么，就是一块青绿色的玉石，但现在在火光的映照下，这块半透明的玉石内部却显示出了之前没有看到过的纹路。
<br/>　　叶少阳心下一惊，更加凑近火光，仔细观察。
<br/>　　在这块玉石的中间，有一个小块的东西，就像琥珀里的小动物，被玉石包裹在中间，形状像沙漏，有个五号电池那么大。
<br/>　　这是啥玩意？
<br/>　　叶少阳把山海印拿的远了点，中间的东西立刻就看不见了，这才明白，原来这东西要在火光下才能看得通透。
<br/>　　沙漏……叶少阳顺手把山海印翻转过来，沙漏里似乎有一抹金色的流光，也跟着流淌起来。
<br/>　　叶少阳盯着看，不知不觉中，整个人仿佛钻了进去，飞过了一片汪洋大海，来到一座岛屿上，岛屿有篮球场那么大，说正方形，是一个泛着金光的轮盘，说是岛屿，还不如说是浮在水面上的一块物件，这轮盘就像一个魔方的表面，被分割成了九块，叶少阳是悬浮在魔方的上方，试了试，脚下能动，但他不敢乱踩。
<br/>　　朝左右看去，四周是一望无际的水面，颜色却是青绿色，像黄龙沟的某些地质特殊的湖泊，水面平静，整个看上去像是一大块蓝绿色的镜子，不，应该是玉石。
<br/>　　玉石？
<br/>　　叶少阳蓦地一惊，再看看四周，猛然明白过来，这不就是山海印的中间吗，四周那青绿色的水域，就是沙漏周围的玉石，自己所在的这个九宫格一样的东西，应该就是沙漏所化。
<br/>　　这么说，自己是来到山海印的内部了？叶少阳感觉到自己轻盈的身体漂浮在空中，想着应该是自己的神识吧。
<br/>　　想到自己意外之中破解了山海印的秘密，不禁有些激动，但是……下一步呢？需要怎么做才能操控山海印？
<br/>　　放眼望去，这个虚幻的空间之中，除了自己脚下这个九宫格一样的轮盘外什么都没有，叶少阳不由自主地把注意集中在轮盘上。
<br/>　　一共九个方块格子，彼此间有方块缝隙，透着亮光出来，看上去有点像是键盘的感觉。
<br/>　　叶少阳心中陡然一个灵感蹦出来：莫非，这是一个类似密码的东西，需要自己踩上去？
<br/>　　九宫格，本来就出自河图洛书，据说当年皇帝就是通过这个推演出的易数八卦，九个格子，从一到九这九个数，每个格子代表一个数，要求是不管横竖还是斜边，加起来都是十五。
<br/>　　唯一正确的排列方式，就是：
<br/>　　六、七、二；
<br/>　　一、五、九；
<br/>　　八、三、四。
<br/>　　看上去就是个数字游戏，但其中透着深奥的易数，不过这些都跟叶少阳没关系，他想的是，如果这真的是九宫格，那么是不是就像一个数字盘一样，只有输入正确的“密码”，才能触及山海印真正的核心机密。问题是，密码是什么呢？
<br/>　　叶少阳看着身下的九宫格，越看越觉得是要输入密码，否则这里什么都没有，而且自己也来了半天了，这里压根没有一丝变化。
<br/>　　密码……
<br/>　　叶少阳绞尽脑汁，觉得这个太难了，九个数字都是一样的，又没任何提示，连密码是几位的都不知道，想试试也不可能。
<br/>　　他都想要放弃了，但是转念一想，既然徐福把山海印给自己，那必然会给出什么提示吧，不然把自己困在这里，守着山海印不能用，又是个什么意思呢？
<br/>(本章完)

　　整个拍卖会的气氛，再次静了下来。
<br/>　　一层的宾客中，无人敢语，等待着二楼贵宾席中顾家家主的抉择。
<br/>　　拍卖师，也没有因为要多等时间而多言。
<br/>　　最后一件拍卖品，星芒石，乃重宝，自然不能按照普通拍卖品的三锤定音，需等到无人竞价为止。
<br/>　　顾家主，确实在考虑着。
<br/>　　要竞价，就必须有银两。
<br/>　　完美丹药，作为硬通货，可直接充当银两。
<br/>　　可功法、武技、灵器，则必须先变卖出去，得了银两，再继续竞价。
<br/>　　若真的能拍下星芒石，顾家自然可以不惜一切代价。
<br/>　　可是，若到时候星芒石没拍下，自己家族的底蕴则都变卖出去了，那可就麻烦了。
<br/>　　一个家族，若底蕴没了，那么离消亡，也就不远了。
<br/>　　作为一个家主，他不敢轻易冒险。
<br/>　　最重要的是，他无法确定易霄身上到底有多少完美丹药。
<br/>　　而且，就算论宝贝，他顾家和钟家联合起来，可能也比不过白家。
<br/>　　白家毕竟是有破玄境武者的家族，底蕴可不是他们这些只有洞玄九重武者的家族能比的。
<br/>　　十数分钟后，顾家主的拳头握得劈啪作响，但最后，还是拳头一松，脸色难看地说道，“顾家，放弃竞价。”
<br/>　　拍卖师点点头，王家也早就放弃竞价了，那么，便没人与白家竞价了。
<br/>　　“星芒石，白家530万两拍得。”拍卖师高声宣布。
<br/>　　拍卖会的工作人员，将星芒石拿来，白家主付了450万两，加上萧逸的100粒完美生灵丹，100粒完美巨力丹，交易算是完成了。
<br/>　　.....
<br/>　　历时数个时辰的拍卖会，便在这不断高潮此起彼伏的过程中结束。
<br/>　　拍得心仪宝物的，自然心中欣喜；没拍得的，也只能怪自己没钱，同时也当来这里开眼界了。
<br/>　　总的来说，这次的拍卖会，算得上宾至如归，完美结束。
<br/>　　.....
<br/>　　拍卖会外，一众宾客，陆续离开。
<br/>　　白家主与萧逸并肩走着。
<br/>　　身后，跟着白管家和一众白家武者。
<br/>　　此刻，白管家看萧逸的眼神，完全不是在看一个少年，而是将萧逸放到了和自己一个层次的高度。
<br/>　　他怎么都想不到，数月前，他甚至想撵出白家的少年，今日竟是和家主并肩走着。
<br/>　　“易霄小兄弟，这次多亏了你的相助，我才能拍下星芒石，这下又得欠你一个人情了。”
<br/>　　白家主说着，感兴趣地问道，“不过，你真这么厉害，随便都能拿出几千粒三品的完美丹药？”
<br/>　　萧逸耸耸肩，笑道，“唬人罢了，几千粒丹药，别说是完美了，就算是劣品，也要炼几年，我怎么可能有那么多。”
<br/>　　“傻瓜才会信我的话。”
<br/>　　萧逸丝毫没有掩饰自己的话音。
<br/>　　此言一出，周围的跟随离开的宾客不禁一愣，随即便是满脸怪异表情。
<br/>　　因为，顾家家主，还有钟家家主，赫然也在附近，也听到了萧逸的话。
<br/>　　“咔咔”一阵拳头猛捏的声响，顾家主双眼冒火，杀人般的眼神，紧紧地盯向萧逸。
<br/>　　“紫炎，我顾家与你无怨无仇，你为何要针对我？”顾家主冷声问道。
<br/>　　萧逸淡淡道，“何来的针对？我与白家主相识一场，帮他就是与你作对？”
<br/>　　白家主一身气势澎湃而出，猛烈地压了过去，“顾家主，最好收起你的杀意。易霄小兄弟是我白家的朋友。”
<br/>　　顾家以及钟家众人，被压得几乎喘不过气来，这才猛地想起来，面前之人，可是破玄境武者，挥手便可杀了自己。
<br/>　　而且，白家主得了星芒石，很快便能拥有上品灵器。
<br/>　　白家，必将跻身北山郡最顶尖的大势力之一，远不是他们能招惹的。
<br/>　　“白家主息怒，在下刚才只是胡言乱语罢了，还请不要放在心上。”顾家主立刻放低了姿态。
<br/>　　“哼。”白家主冷哼一声，收回了气势，与萧逸转身离开。
<br/>　　在场宾客愣愣看着，而他们的目光，大部分都聚集在萧逸的背影上。
<br/>　　他们知道，从今日起，紫炎易霄，除了是名声在外的超级天才外，更将是百武城附近十数城范围内，再无人敢招惹的存在。
<br/>　　因为，他是白家的朋友。
<br/>　　......
<br/>　　事实上，萧逸确实没有那么多的丹药。
<br/>　　离开萧家以来，他得的最多东西，除了银两外，便是天材地宝了。
<br/>　　萧家长老们赠予的，狂血玄君洞府内的。
<br/>　　还有击杀山贼首领，四大杀手的乾坤袋里得到的。
<br/>　　虽然没有什么太过珍贵稀罕的天材地宝，但普通天材地宝，数量却极多。
<br/>　　这些材料，他几乎都是每得到一些，就会顺手炼成丹药。
<br/>　　毕竟他那时要看大长老萧离火的炼药心得，边看，自然也边练手。
<br/>　　而他之所以早就知道三品以上的完美丹药能成为硬通货，是因为他以前就卖过了，正是卖给猎妖殿。
<br/>　　准确来说，是收购。
<br/>　　猎妖殿，无限收购这些丹药；当然，只给钱，没有任务点。
<br/>　　他那二百多万两，除了清剿山贼匪窝所得外，有很多是来自于这些丹药出售给猎妖殿所得。
<br/>　　后来，抢夺了北山慕容家和暗影楼那次交易的商物，又得了一大批天材地宝。
<br/>　　那100粒生灵丹、巨力丹，就是用这些材料炼制成的。
<br/>　　当然，他现在还有很多天材地宝，没来得及炼制。
<br/>　　进入裂天剑派后，时间都放在修炼上了，也没机会炼药。
<br/>　　至于今天，他之所以帮助白家主。
<br/>　　并非是因为顾长风和顾长空。
<br/>　　武者间的恩怨，与家人朋友无关。萧逸还不屑做那种报复顾家的小人行径。
<br/>　　只是因为，白家主与自己有一定的交情；而之前竞拍八宝玲珑参时，顾家的人太过霸道了些，让得他很是不爽。
<br/>　　这才有了今天的相助白家。
<br/>　　......
<br/>　　言归正传。
<br/>　　这时，萧逸与白家众人，刚准备离开。
<br/>　　一群身穿统一制服的武者，却挡在了他们面前。
<br/>　　看这些制服的样式，显然是盛宝商行的护卫队，清一色的先天九重武者。
<br/>　　为首的，正是慕容杀和慕容胤。
<br/>　　“紫炎易霄，终于肯露面了吗？”慕容胤冷声说道。
<br/>　　“你以为来这拍卖会，能瞒得住我们的眼睛？偷偷来，又偷偷离开？”
<br/>　　“之前不动你，只是因为你是客人身份；如今拍卖会已经结束，你则是敌人身份。”
<br/>　　慕容杀也冷声道，“识相地便把那批商物交还出来，否则，休怪我们对你不客气。”
<br/>　　慕容杀说罢，两个壮汉一左一右，封锁了去路。
<br/>　　这两人，正是之前在拍卖会中负责守护展览台拍卖品的两个洞玄境武者。
<br/>　　他们本身就是盛宝商行的武者。
<br/>　　也就是说，慕容杀、慕容胤、两个壮汉，四大洞玄境武者，将四面的方向都包围了。
<br/>　　“放肆。”白管家冷喝一声，“你们敢拦我们白家去路？”
<br/>　　慕容杀上前一步，道，“白家主、白管事，盛宝商行无意与你们为敌。你们可随意离去，我们绝不阻挠半分。”
<br/>　　“只是紫炎易霄，却绝不可能离开。”
<br/>　　“哦？”白家主冷笑一声，“莫非你们数人，觉得能在我白某人手下抢人不成？”
<br/>　　慕容杀语气强硬，道，“我们区区洞玄境，自然无法在白家主您这位破玄境手上抢人。”
<br/>　　“只是，这易霄，是我盛宝商行通缉的人。白家若庇护此人，等同与北山慕容家为敌。该如何做，我想白家主不用在下多说吧。”
<br/>　　“你威胁我？”白家主冷声说着，就要出手。
<br/>　　“慢。”萧逸站前一步，拦住白家主，道，“白家主好意，易某心领了。这事，我自己解决足矣。”
<br/>　　说罢，萧逸看向慕容杀，道，“不知慕容杀执事，要易某交什么东西？”
<br/>　　“还在装蒜？”慕容杀冷声道，“数月前，我盛宝商行与暗影楼的那批交易商物，你敢说不是你抢的？”
<br/>　　“证据呢？”萧逸反问道。
<br/>　　“还敢狡辩。”慕容杀冷喝道，“百武城十数城范围内，谁不知道你就是凶手。”
<br/>　　“哦，是吗？谁能证明？要不，你将那次运送商物的执事或者护卫叫出来，亲自与易某对质。”萧逸冷笑道。
<br/>　　“混账。”慕容杀怒道，“那些人全被你杀了，如何能与你对质？”
<br/>　　萧逸冷笑道，“那就是没有人证，没有物证，没有任何证据了。”
<br/>　　“哼。”慕容杀冷哼道，“无忧公子亲眼看到是你所为，那便是铁证。”
<br/>　　“那好。”萧逸怡然不惧，道，“便请无忧公子来与我对质一番。”
<br/>　　“放肆。”慕容杀怒不可遏，“无忧公子，我们如何能请得动，你这是在胡搅蛮缠，强行开脱罢了。”
<br/>　　“说来说去，你们根本没有证据证明是易某所为。”萧逸的声音渐渐变得冰冷起来。
<br/>　　“既然没有证据，却非要留下易某，便休怪易某对你们不客气了。”萧逸忽然笑了。
<br/>　　他狡辩，不是怕慕容杀等人，只是找一个出手的借口罢了。
<br/>　　当初，若不是慕容杀要去紫云城对付萧家；萧逸是万万不会抢他们货物，自寻祸端的。
<br/>　　慕容杀胆敢对付萧家，萧逸就做好了不惜一切对付他的准备。
<br/>　　“不客气？”慕容杀冷笑一声，“我倒要看看，你如何不客气。”
<br/>　　说着，慕容杀大手一挥，喝道，“立刻拿下易霄，若敢反抗，格杀勿论。”
<br/>　　一众盛宝商行护卫瞬间出手。
<br/>　　白家主脸色一怒，刚准备帮萧逸，却被萧逸一个眼神止了下来。
<br/>　　萧逸的眼神告诉他，自己有把握对付，他不要出手。
<br/>　　“哼。”同一时间，顾家、钟家的人，得意地看着，暗道，“紫炎，有白家庇护你又如何，得罪了北山慕容家，你不过死路一条。”
<br/>　　但，他们得意的想法，并没有持续多久。
<br/>　　因为，嘭的一声，漫天紫炎倾泻而出。
<br/>　　那十数盛宝商行护卫，瞬间被吐血击飞。
<br/>　　而后易霄瞬间出手，直朝慕容杀而去。
<br/>　　“来得好。”慕容杀冷喝一声，气势逼人。
<br/>　　但，下一秒。
<br/>　　又是嘭的一声。
<br/>　　在所有人不可思议的目光中，慕容杀，竟被萧逸一拳轰飞，口吐鲜血。
<br/>(本章完)

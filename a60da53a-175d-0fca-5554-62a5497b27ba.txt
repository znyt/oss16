　　萧家，房内。
<br/>　　萧逸席地而坐，放好鎏金炉和材料，准备炼药。
<br/>　　在离开药材店后，萧逸没有立刻回萧家，而是找了间规模较小的药材店，将炼制淬体丹的其他辅助药材买到手。
<br/>　　这些辅助药材价格并不贵，仅仅数十两。
<br/>　　萧逸不知道那间药材店的伙计为什么给自己打折，而且是七折，这个价格已经等同于药材店没有赚钱，甚至还亏一点儿。
<br/>　　但，无论他们是出于善意和恶意，自己在没有足够的自保实力之前，应该都不会再踏足那间药材店了。
<br/>　　萧逸一向谨慎。
<br/>　　言归正传，炼药是一门非常深奥的学问，即便材料已经备好了，但炼药的过程才是重中之重。
<br/>　　炼药存在着成功率，像淬体丹，一些普通的一品炼药师，炼制的成功率不足五成。
<br/>　　哪怕是萧家那几位经验丰富的一品炼药师，成功率也不过是八成左右。
<br/>　　恐怕就只有大长老萧离火这位三品炼药师，才能做到对一品丹药百分百成功炼制。
<br/>　　炼药过程中，对火候的把控，药材的投放顺序，投放时间，都有极高的要求。
<br/>　　很可能，火候不对，时间差了几秒，就会导致毁丹。
<br/>　　而投放顺序的不同，则很可能会炼制成其他丹药。
<br/>　　材料的不同顺序组合，会衍变出许多种不同的功效。
<br/>　　所以说，哪怕炼药师拥有丹方，也必须将这个丹方钻研透彻，才不至于在炼药过程中浪费材料。
<br/>　　毕竟材料还是很贵的。
<br/>　　萧逸将脑海中的炼药知识整理了一遍，所谓艺高人胆大，他立刻开始了炼丹。
<br/>　　一团猛烈的火焰陡然出现在他掌心。
<br/>　　在控火兽武魂的赋予能力下，他将火焰投入鎏金炉。
<br/>　　待鎏金炉被完全烧热，控制好火候，他立马投入了第一株药材，无心藤。
<br/>　　火焰的燃烧之下，无心藤渐渐融化，化为了一股碧绿色的液体。
<br/>　　然后是八叶花。
<br/>　　.......
<br/>　　一个个步骤，一次次投入材料，萧逸小心翼翼。
<br/>　　大概在半小时后，萧逸收回了火焰。
<br/>　　而此时，鎏金炉中，传来一阵诱人而扑鼻的丹香，让人闻之舒爽。
<br/>　　打开炉鼎，四粒完美的淬体丹呈现于眼中。
<br/>　　一份材料，大概能炼制出四粒淬体丹。
<br/>　　“一次就成功了。”萧逸大喜。
<br/>　　第一次炼药就成功，哪怕是当年被誉为萧家天才炼药师的萧离火都做不到。
<br/>　　而萧逸之所以能做到，完全是因为冰鸾剑。
<br/>　　炼药师，需要一次次积累经验，一次次尝试失败，最后才换来驾轻就熟、如臂指使以及炼制成功。
<br/>　　可萧逸不同，冰鸾剑完全将这个世界的炼药知识如同醍醐灌顶般灌输到他的脑海中，让他的炼药技术达到了如教科书一般的完美。
<br/>　　看着四粒完美的淬体丹，萧逸不禁感慨，这个世界的炼药师简直就是会印钱的机器。
<br/>　　一份材料，价格不过是数十两。可炼制出来的淬体丹，一颗就能卖五十两，四颗就是二百两，足足赚了几倍。
<br/>　　二百两，看似不多，但这放到普通人的家庭，足够他们用几年了。
<br/>　　而且，这还只是最低等的一品丹药。那么二品、三品，乃至更高品的丹药呢，价格又该高到何等地步。
<br/>　　萧逸将四粒淬体丹装好，但没有服用。
<br/>　　武者服用丹药虽然能提升修炼速度和修为，但这是有限制的。
<br/>　　毕竟，淬体丹中除了药力外，还有不少的杂质。
<br/>　　这些杂质，会堵塞丹田。之后再服用更多的淬体丹也不会有任何效果。
<br/>　　萧家之所以一个月只提供三颗淬体丹给家族子弟，就是因为，一般的橙阶武魂，一个月服用三颗已经是极限了。
<br/>　　武者必须将这些杂质完全排出，才能继续服用淬体丹。
<br/>　　至于萧逸，他的冰鸾剑武魂吸收药力的速度太快了，淬体丹的杂质甚至还没能完全堵塞他的丹田，药力就已经完全被吸收，并转化为真气了。
<br/>　　这才让他在短时间内一次性吸收了几十颗淬体丹。
<br/>　　当然，现在那些杂质同样存在于他的体内。在排出那些杂质之前，他再服用淬体丹也于事无补。
<br/>　　既然淬体丹暂时没用了，他自然不会再炼制，而是选择了炼制九转炼体丹。
<br/>　　根据脑海中的信息，九转炼体丹是冰鸾剑衍生并创造出来的丹药，功效远超淬体丹。这种丹药完全由精纯的药力组成，没有杂质。
<br/>　　武者服之，可不受限制。
<br/>　　刚才炼制淬体丹，只不过是用来练手，现在炼制九转炼体丹，才是主要目的。
<br/>　　炼制九转炼体丹，同样需要用到无心藤等药材，只不过多了灵参罢了。
<br/>　　当然，两者的药材投放顺序，以及投放时间都不同。
<br/>　　萧逸没有浪费时间，立刻开始炼药。
<br/>　　.....
<br/>　　又是一次次的步骤，一份份的药材投放，每分每秒的小心翼翼。
<br/>　　半小时后，鎏金炉内，出现了四粒洁白无瑕的丹药。
<br/>　　如果说淬体丹的丹香是让人闻之舒爽，那么九转炼体丹的丹香，则是让人闻之四肢百骸通畅无比。
<br/>　　全身无数的毛孔仿佛都在一瞬间张开，贪婪地吸收着这些香气。
<br/>　　每一点香气闻入，身体就变得浑身充满了力气般。刚才炼制丹药时消耗的体力，一下子完全补充；身体的疲劳感，顷刻间消散须弥。
<br/>　　就算没有吞服，萧逸也知道这四粒九转炼体丹的功效绝对不凡。
<br/>　　掌心中再次冒出火焰，萧逸再次开始了炼药。
<br/>　　......
<br/>　　月明星稀，夜幕降临，萧逸才结束了炼药。
<br/>　　所有材料几乎都用光了，总共炼制了二十粒九转炼体丹。
<br/>　　剩下还有一些材料，萧逸顺手炼制了些强身丹和养气丹。
<br/>　　“辛苦了一整天，该是品尝劳动果实的时候了。”萧逸满意地笑了笑，感到一阵莫名的喜悦和成就感。
<br/>　　随手一抛，一粒珍贵的九转炼体丹抛入口中，转瞬间化为精纯的药力进入体内。
<br/>　　药力中没有任何杂质，宛如一股通彻的清流，直达丹田。
<br/>　　丹田吸收后，又转化为真气。
<br/>　　真气的越来越浓郁，也代表着自身的修为越来越高。
<br/>　　同时，肉体明显被淬炼得强了数分，起码比以往的萧逸那孱弱的身体强上许多。
<br/>　　如果现在使用形意五绝的话，再也不会出现身体支撑不住的情况。
<br/>　　论效力和药力，九转炼体丹，比之淬体丹强了数倍都不止。
<br/>　　没有任何犹豫，萧逸立刻将剩下的19粒九转炼体丹一一吞服。
<br/>　　一粒粒丹药入腹，一股股精纯药力被吸收，丹田渐渐膨胀，越来越坚实，肉体越来越强。
<br/>　　当第十五粒九转炼体丹被吞服，萧逸的修为完全突破到凡境五重。
<br/>　　剩下的四粒，仅仅是让他的修为精进了数分，却没能让他突破。
<br/>　　值得一提的是，武者的修炼，果然是越到后期越是困难。
<br/>　　以九转炼体丹的功效，萧逸从凡境四重到凡境五重，都足足用了15粒；若是换成普通的淬体丹，需要的数量肯定极多。
<br/>　　加上淬体丹拥有杂质，无法连续服用。也难怪萧家子弟中，有很多都卡在凡境四重这个坎，一直没能突破。
<br/>　　凡境五重以上的萧家子弟，不足二十人，六重以上，更是不足十人。
<br/>　　“咔、咔、咔。”萧逸捏了捏拳头，发出一阵咔咔的声音。
<br/>　　明显能感觉到，此刻自己的肉体，拥有着爆炸性的威力。
<br/>　　“虽然只是凡境五重，但加上我的形意五绝，足以轻松应付半月后的家族比武。”萧逸笑了，自从来到这个世界后，头一次出现了可以轻微放松一些的心态。
<br/>　　“这半个月我再勤加修炼，希望半个月后能突破到凡境六重，到时候，我便有了与萧若寒好好算账的底气了。”萧逸眼中闪过一道杀意。
<br/>　　....
<br/>　　“嗯....”忽然，旁边的床上传来一阵微弱的呢喃声。
<br/>　　昏迷了一天一夜的依依终于醒了。
<br/>　　“少..少爷。”依依意识到自己正睡在萧逸的床上，有些局促和不安，挣扎着虚弱的身体，打算起来。
<br/>　　“躺下吧。”萧逸将她按回床上。
<br/>　　“少爷，依依是个下人，会弄脏您的床。”依依满脸不安。
<br/>　　“让你睡就睡，我昨晚都打了一晚的地铺，也不差这一晚了。”萧逸淡淡道。
<br/>　　“阿，晚上了吗？我睡多久了？”
<br/>　　“一天一夜了。”
<br/>　　“一天一夜？”依依眼睛瞪得大大的，很是惊讶，道，“那少爷你岂不是饿坏了，我立刻去给您准备晚饭。”
<br/>　　依依说着，立刻准备起来。
<br/>　　“不用了，我吃过了。”萧逸将依依按回床上。
<br/>　　“吃过了？少爷您会做饭吗？是三长老给您准备的吧？”依依先是惊讶，随即露出了恍然的表情。
<br/>　　在她心中，萧逸少爷几乎是个没手没脚的家伙，衣来伸手，饭来张口。别说自己照顾自己了，如果她不煮饭给他吃的话，他饿死都有可能。
<br/>　　萧逸看依依的表情，大致猜出她内心想法，顿时满头黑线。一把将她按下，帮她盖好被子后，呵斥道，“闭嘴，闭眼，睡觉。”
<br/>(本章完)

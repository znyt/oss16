　　叶少阳认出他就是白天自己看到的跟在伏明子身边的那个孩子，应该也是茅山弟子，好奇地走了过去，那男孩似乎没发现他，仍然望着溪水发呆，样子很是痴迷。
<br/>　　叶少阳看了好半天，实在好奇，于是咳嗽了一声，问道：“你在干什么？”
<br/>　　“看水。”男孩头也不抬。
<br/>　　“我知道，问你看水干什么。”
<br/>　　“我在格物。”
<br/>　　“格物？”叶少阳不懂。
<br/>　　男孩这才抬起头来，像是突然发现叶少阳，皱眉盯着他，说道：“你是什么人？”
<br/>　　呃……跟自己聊了好几句，才想起来问自己是谁，这孩子，可是真够呆的。
<br/>　　没等叶少阳开口，男孩认出他来，说道：“哦，你是白天见过的那个人。”
<br/>　　叶少阳隔着小溪，在他面前蹲下，说道：“格物是什么？”
<br/>　　“格物致知，你没听说过吗？”
<br/>　　“额……好像听说过这个词。”叶少阳有点尴尬，对这个词，他也只是听说过而已，好像是古代的跟儒家有关的一种东西，具体是什么他还真不知道。
<br/>　　“格物致知，是朱圣人从儒家经典中总结出的一种了解万事万物的法则……这个说起来就远了，不过在法术界，却是一种内修的法门，跟冥想类似，不过冥想是静坐感悟周围的天地万物，格物致知，却是从一方事物中感知万物变化，从而悟出道之所在……”男孩回答得十分老实。
<br/>　　叶少阳看这个小男孩一本正经的样子，觉得很好玩，伸手摸了摸他的脑袋，说道：“我怎么不知道有格物致知这种修炼法门？”
<br/>　　“因为，早就失传了。我是从上也没有说怎么格物，我只能自己琢磨……”
<br/>　　“哪怎么格物呢？”
<br/>　　“就是盯着看。”
<br/>　　“没了？”
<br/>　　“没了。”
<br/>　　叶少阳嘴角抽搐了几下，这特么算哪门子修行法门！
<br/>　　小男孩道：“格物致知，也不是你想的这么容易，只有富有生机的物事，才有‘格’的必要，这溪水湍急，私有道在其中，所以我就来这里了。”
<br/>　　叶少阳跟着他一起盯着溪水，实在没看出道在什么地方，笑了笑道：“你师父不反对你这么做？”
<br/>　　“我天资愚钝，怎么修炼都赶不上别人，而且我对那些心法没什么兴趣，我觉得，古人既然记载了这个修行法门，一定不会是假的，而且，我能够感觉到道的存在，这就证明我适合这么修行啊。”
<br/>　　叶少阳不知道该怎么说，在他看来，这种坑爹的修行方式简直就是浪费时间，消耗生命，无奈地摇头说道：“你这么干多久了？”
<br/>　　“有两三年了吧。”
<br/>　　两三年……叶少阳无语。“那你格出什么没有？”
<br/>　　“我格出了天地大道，但是……我却无法归纳总结，不明白道之原理……我说给他们听，他们都不相信，就连我师父，每天也只允许我格物两个时辰……但是，我真的格出了一些大道，只是无法用言语说出来，因为道可道，非常道。”
<br/>　　叶少阳心中一动，看着男孩稚嫩的面孔上显出的认真模样，也算是明白他为什么要跟自己说这么多了……平时没人听他这么说，相反，像他的这种行为，可以想见，一定是受人嘲讽，很不受待见，他找不到人倾诉。
<br/>　　“你呢，你也觉得我是在浪费时间吗？”
<br/>　　叶少阳拍了拍他的肩膀，道：“你如果真的从这格物致知里感受到大道的存在，你坚持做就是了，管别人做什么。”
<br/>　　小男孩眼睛一亮，道：“这么说，你是支持我这么做的了？”
<br/>　　叶少阳点点头，“我师父告诉过我，任何人，都有他自己的路，只要不危害社会、不伤害他人，那就没人有资格对你指东道西，坚持自己就好了。”
<br/>　　小男孩咀嚼着他的话，脸上现出动容的神色，高兴地说道：“谢谢你，你师父说的对！”
<br/>　　“你继续吧。”叶少阳站起身，跟他道了别，小男孩盘膝坐下，又开始格物致知了。
<br/>　　叶少阳笑了笑，转身要走，突然想起什么，说道：“对了啊，你是茅山外门弟子是吧？”
<br/>　　“我是内门弟子。”小男孩头也不抬。
<br/>　　内门？
<br/>　　叶少阳皱起眉头，问道：“你师父是伏明子？”
<br/>　　见小男孩点头，叶少阳心中更加疑惑了，从来没听自己师父说过有什么师兄弟啊？
<br/>　　“你叫什么？”
<br/>　　“我俗家姓蒋，道号叫青云子。”
<br/>　　青云子！！！！
<br/>　　叶少阳好像被雷劈了一下，浑身一颤，跌倒在地上。
<br/>　　小男孩听见动静，抬起头，诧异地看着他，道：“你怎么了？”
<br/>　　叶少阳傻傻地看着他，突然醒过神来，也顾不得身上被弄湿，直接从溪流里爬了过去，一把抓住小男孩的手，失声道：“不会吧，你真的是……青云子？”
<br/>　　“这……我当然是了，我为什么要骗你！”小男孩很不解，“这位大哥，你……你为什么涉水过来啊，你快上来啊。”
<br/>　　叶少阳爬上岸边，坐在他身边，魂不守舍地看着他。
<br/>　　青云子……自己的师父，在1922年的这个时候，已经出世了，看上去十来岁……他在心里头死算了一下，照这么算，自己的师父，在自己那个时代，的确是有一百岁出头了，以前一直以为他是吹牛，没想到……他真的有一百岁。
<br/>　　师父……
<br/>　　叶少阳看着面前这个小男孩，仔细端详他的五官，别说……跟自己师父还真有点像。
<br/>　　“青云子……”叶少阳恍然想到了师父的经历：前半生都默默无闻，资质愚钝，四十岁的时候，一遭在水潭边观看鱼龟争食，突然开窍，悟出了天地大道，从此以后，实力突飞猛进，虽然一向玩世不恭，不爱显山露水，但他的真实实力，绝对是冠绝人间，能火拼人间道神无极天师，便是明证……
<br/>　　这一切都是那么的吻合，原来，师父靠的是格物致知……而且这么一格，就格到了四十岁，他能面对别人的质疑和对自己的怀疑，坚持到四十岁，方才大彻大悟，这其中的辛酸和寂寞，没有人能懂，也几乎没有人能坚持下来。

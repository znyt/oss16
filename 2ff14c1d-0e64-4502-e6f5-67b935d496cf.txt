　　楚风眠的手心中，灵力猛然凝聚，这陆雀，陆画二人，几乎是在下一刻，就要被捏死。
<br/>　　在楚风眠的手心之中，这陆雀，陆画二人的脸色，都是极为惨白，一个个吓得半死。
<br/>　　他们二人，只怕是做梦都不会想到，居然会有着距离死亡如此之近的一天。
<br/>　　就在楚风眠出手，灵力猛然凝聚的一刻。
<br/>　　从天空之上，突然传来一道声音。
<br/>　　“小辈，这事做的过了吧！”
<br/>　　这声音，虚无缥缈，令人根本不清楚，到底是从何传来的声音。
<br/>　　“谁？”
<br/>　　楚风眠猛然抬起头，他的脸色都是骤然一变。
<br/>　　似乎也是没有料想到，在这个时候，居然又有人来插手。
<br/>　　“小辈，你敢如此对付我们陆家之人，莫非真的是欺我们陆家无人吗？”
<br/>　　从空中，再次传来这几位威严的声音，这声音的主人，其中话语之中的意思，便是真正的怒火中烧，显然是愤怒到了极致。
<br/>　　陆家！
<br/>　　陆家的人！
<br/>　　楚风眠心中吃了一惊。
<br/>　　这声音的主人，就连楚风眠，似乎都无法察觉到这其中的真面目。
<br/>　　显然这是一尊陆家的大人物，他的实力，还要超越陆雀许多，就连楚风眠都看穿不透身形，这至少也是要超越生死境巅峰的存在了。
<br/>　　“陆千刃老祖！”
<br/>　　听到这声音的瞬间，那陆雀，陆画二人，都是脸色大喜，大声喊道。
<br/>　　“陆千刃老祖，快要救我们！此人的实力，绝不会是我们北邙学院的人，很有可能是地摩门派来的奸细，还请老祖出手，降服此人！”
<br/>　　老祖！
<br/>　　听到这个称呼，在场没有任何一人，不是为之色变的。
<br/>　　这称呼，可不单单是代表着陆家的老祖。
<br/>　　在北邙学院之中，可都是只有超越了生死境巅峰，踏上生死台阶的老古董，才有资格被称之为老祖。
<br/>　　能够被称为老祖的存在，都是将来可以被放入到众圣堂中，被人瞻仰的存在。
<br/>　　这样的老古董，猛然出手了，目标，正是楚风眠。
<br/>　　“哼！”
<br/>　　天空之上，传来一声冷哼，便是将如今这老祖的愤怒，展现的淋漓尽致。
<br/>　　“镇！”
<br/>　　这天空之上，单单是吐出这一个值，只见一座巨大无比的山峰，突然从天而降。
<br/>　　山峰？
<br/>　　等到楚风眠一眼望过去，仔细一看，这一座山峰，其实却只是一根手指。
<br/>　　一指突然向着楚风眠，轰击过来。
<br/>　　这简简单单的一指，其中蕴含的灵力，都是将这一指，凝聚的如此一座千米高山一般，突然从天落下，要将楚风眠镇压。
<br/>　　“请老祖息怒！此人是我们学院高层要培养的天才，还请老祖高抬贵手！”
<br/>　　一尊长老，猛然出手，对着天空之上劝解道。
<br/>　　但是他的话音未落，从天上却是落下来一个字。
<br/>　　“滚！”
<br/>　　这一个“滚”字，如同一道巨大无比的拳风，直接是落在了这长老的胸口上，令这长老这一下子被打飞出去。
<br/>　　这长老，也是生死境高阶的强者，放在北邙学院都是高层，在外面，更是雄霸一方的大人物。
<br/>　　可如今在这陆千刃老祖面前，却是就连一个字都接不下。
<br/>　　这是何等的实力，生死境高阶的强者，在他的面前，就连一个字都接不下。
<br/>　　以现在楚风眠的实力，虽然面对生死境高阶的强者，想要轻易击败，也是足以做到。
<br/>　　但是要说让楚风眠单单一个字，就将生死境高阶的强者击败，那楚风眠可就做不到，就算是楚风眠动用天龙八音的力量，都不一定可以震伤生死境高阶的人物。
<br/>　　更何况，楚风眠也看得出来，如今这陆千刃老祖的一句话，其实根本就不是什么秘术，就是简简单单的一个字，就可以做到如此地步。
<br/>　　这种实力，让一百个，一千个生死境高阶的人物，同时出手，只怕都是远远不是这陆千刃老祖的对手。
<br/>　　毫无疑问，这便是超越了生死境巅峰，踏入了生死台阶的人物。
<br/>　　“不知道这陆千刃老祖，到底是踏上了第几层台阶，生死台阶，一共九层，每一层之间的差距，都是宛如天沟，无法跨越。”
<br/>　　“只是刚刚一个字，只是将这长老重伤，却是没有将他灭杀，这陆千刃老祖的实力，应该还没有达到太高的地步，若是踏上了第三层，第四层，只怕是一个字之下，这长老都要陨落了！”
<br/>　　楚风眠心中，暗自分析起来这陆千刃老祖的实力。
<br/>　　只是就算如此，楚风眠的心中也是极为不安，这可是一尊超越了生死境巅峰的大人物。
<br/>　　他若是出手，以现在的楚风眠，还真不一定足以对抗。
<br/>　　楚风眠也没有料到，他将陆雀，陆画逼到如此地步，这陆家的老古董，居然都不顾一切，出手了。
<br/>　　这种老古董，往日里都在参悟境界，想要踏入更高的阶梯之中，往日里学院有着无数大事，他们都不会参与。
<br/>　　现在被楚风眠逼的苏醒过来，看到楚风眠追杀陆家之人，一下子怒火冲天。
<br/>　　“陆千刃老祖！请您息怒！”
<br/>　　又是一尊长老急忙的飞了过来，劝解到。
<br/>　　“此人得到了众生堂的传承，不能杀啊！”
<br/>　　“滚开！一群废物，也敢跟我们陆家对坐，再不滚，你得死！”
<br/>　　天空之上，陆千刃的声音，简直是霸道到了极点。
<br/>　　如今不管是谁敢出手阻拦，他都丝毫不给这个面子。
<br/>　　“小辈！给老夫去死！”
<br/>　　轰隆！
<br/>　　这陆千刃的一指，猛然从天落下，隔空而下，向着楚风眠轰击过来。
<br/>　　这一指的边缘，周围的空间，都开始一层层的破碎，在这北邙圣域中的空间法则，居然都挡不住这如此可怕的力量。
<br/>　　不止是力量，这一指的速度，也是快到了一种截然不同的地步。
<br/>　　看似缓慢，但是却是瞬息之间，便是已经落在了楚风眠的头顶之上。
<br/>　　这种速度，根本便是绝无可以逃走的可能。
<br/>　　“糟了！”
<br/>　　楚风眠脸色猛然骤变。
<br/>　　这一指的速度，要超乎他的想象。
<br/>　　这种速度，想要对开，根本没有可能。
<br/>　　如今唯一的办法，只能硬抗。

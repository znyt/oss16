        “别开玩笑了，怎么可能！”童筱颖不以为然地摆了摆手。
<br/>
<br/>        夏凌焰随即从童筱颖的头上扯了一根头发下来，痛得童筱颖还手拍了一下夏凌焰的肩头。
<br/>
<br/>        “我把你头发交给夏凌焕那小子，让他带回去跟九叔做个亲子鉴定。”夏凌焰洋洋得意地说道。
<br/>
<br/>        童筱颖连忙将他手里的头发给夺了回来。
<br/>
<br/>        “诶？你这是做什么啊！”夏凌焰不禁抱怨道。
<br/>
<br/>        童筱颖蹙眉道：“你就不要无中生有了！”
<br/>
<br/>        “好吧！”夏凌焰见童筱颖一脸不开心，就不跟她较劲了，“我先走了，你就安心在这里住吧！有项目做，我会电子邮件通知你。”
<br/>
<br/>        “好！”
<br/>
<br/>        “这是提前支付给你的工资，去买台配置高的笔记本电脑回来制图。”夏凌焰接着有板有眼地说。
<br/>
<br/>        童筱颖如小鸡啄米般点了点头。
<br/>
<br/>        而接下来的这些天，童筱颖过起了宅女一般的生活。
<br/>
<br/>        每天除了去楼下的生活超市里买菜以外，她几乎不会踏出这里的方圆五里。
<br/>
<br/>        虽然，空暇之余，她很想念大哥……
<br/>
<br/>        但是……
<br/>
<br/>        童筱颖工作完后，合上了笔记本电脑，起身去了客厅，拿起茶几上的遥控器，一屁股坐在沙发上，打算看一会儿电视。
<br/>
<br/>        频道被她轮流按了一遍，有好几个台都在重播昨天的国际电影节。
<br/>
<br/>        童筱颖看到的时候，刚好是开幕红毯环节。
<br/>
<br/>        许多记者扛着“长枪短炮”，堵在了红毯外围，拍着一个又一个来参加电影节的各大明星。
<br/>
<br/>        只可惜，她童筱颖不追星，对电视里的明星都不大感兴趣，正准备关电视的时候，屏幕上出现在赞助商的字幕。
<br/>
<br/>        傅夜沉这三个字，瞬间吸引了她的目光。
<br/>
<br/>        接下来的画面，苏漫雪便挽着傅夜沉出现在了红毯入口，镜头也随之一路跟拍。
<br/>
<br/>        屏幕上，只见傅夜沉一身剪裁合体的手工白色西装，双腿修长，恍若步下生风。那张俊美的脸庞上浮泛着似笑非笑，一双黑曜石般的眸子发出摄人心魄的微光，足矣让在场前前后后的男明星们自惭形秽。
<br/>
<br/>        他不是明星，却比明星更为夺目。
<br/>
<br/>        而苏漫雪则挽着傅夜沉的胳膊，一身银色亮片鱼尾长裙，发髻上别着云状的发饰，飘逸又华丽，加上精心修饰过的妆容，让她更显得娇俏动人。
<br/>
<br/>        童筱颖看到苏漫雪时，就好像是看到了自己。
<br/>
<br/>        因为苏漫雪穿着十公分的细跟高跟鞋，傅夜沉还特意为苏漫雪放慢了步伐。
<br/>
<br/>        看到这里，童筱颖忽然间觉得自己活得更像是一个笑话。
<br/>
<br/>        明明苏漫雪跟她拥有着一张一模一样的脸，但是苏漫雪从不把任何男人放眼里，也从不对任何男人付出真心，她善于利用别的弱点，去追求的自己想要的物质生活，所以她现在活成了光鲜亮丽的大明星。
<br/>
<br/>        那么她童筱颖呢？
<br/>
<br/>        被情所伤，还被伤了一次又一次。
<br/>
<br/>        忽然间，童筱颖不禁怀疑，大哥爱的，只不过是她这张脸，至于这张脸安在谁身上都无所谓吧！
<br/>
<br/>        关了电视，童筱颖往沙发上一躺，心痛不已地看着天花板发呆。
<br/>
<br/>        她这辈子，不后悔遇见何明旭，不后悔遇见傅昼景，却后悔再遇见傅夜沉。
<br/>
<br/>        他跟她其实真的是两个世界的人。
<br/>
<br/>        她不了解他，他也不了解她。
<br/>
<br/>        他身边没有了她，以他那优越的自身条件，还可以拥有其他光鲜亮丽的女人，继续当他那高高在上的大总裁。
<br/>
<br/>        而她，身边没有了他，就真的只剩下她孤家寡人了。
<br/>
<br/>        当初，他为什么要来招惹她啊！
<br/>
<br/>        招惹了她，又不信任她，还要抛弃她……
<br/>
<br/>        就在童筱颖陷入昏天地暗的情伤中时，家里的门铃突兀地响起，将她从这悲伤里拉了出来。
<br/>
<br/>        她从沙发上起身，走到门口，透过猫眼探看门外的来访者。
<br/>
<br/>        见对方是个陌生男子，戴着眼镜，面目清秀，穿着白衬衫，不像是坏人，于是才将门打开。
<br/>
<br/>        “你好，我是住你对面的夏凌焕，是夏凌焰的堂弟，他有跟我说过要我多多关照你一下。听说你离婚了，但是，你住在这里有一段时间了，我一直没见你出过门，还以为你……所以就……”夏凌焕腼腆又含蓄地说。
<br/>
<br/>        他以为她想不开，在家轻生了吗？
<br/>
<br/>        童筱颖明白了他的来意后，微笑着谢道：“我没事。”
<br/>
<br/>        “你叫童筱颖吧！要不，一起出去吃个晚饭。现在，也到饭点了。就当我们俩初次见面，彼此交个朋友。”夏凌焕又接着落落大方地邀请。
<br/>
<br/>        童筱颖抿了抿唇，微笑着点了点头。

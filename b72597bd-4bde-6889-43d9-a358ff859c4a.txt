
<br/>　　<br/>　　    墓室里的水量正在急剧增加，当金字塔的底部全部被湖水填充满了以后，正墓的积水开始往上蔓延，逐渐往顶部升去。<br/>　　<br/>　　    “噗通！”石棺飘过来后，王惊蛰翻身就从棺材里跳了下来，扶着李菲儿的胳膊，跟林汶骐说道：“给她送进去，我们把着棺材边缘”<br/>　　<br/>　　    李菲儿顿时一愣，林汶骐和王惊蛰托着她就把人抬进了石棺里，里面空间有限也就只能放下一个人了，这时候的石棺就跟一叶小舟差不多，论安全度来讲，肯定比在水里要好上不少。<br/>　　<br/>　　    “谢谢！”李菲儿神情复杂的说道。<br/>　　<br/>　　    王惊蛰抹了把脸的水渍，说道：“没啥，绅士风度，女士优先……”<br/>　　<br/>　　    水位越来越高，渐渐的离墓室的棚顶也越来越近，片刻之后，三人加上石棺就已经被湖水推到了棚顶，先前掉落下来的石砖，刚好能够让棺材转过去。<br/>　　<br/>　　    “一，二，三”眼看着接近上面棚顶后，林汶骐喊了一下，三人顿时深吸了口气沉到水中，水位逐渐接近，几乎不到一分钟的工夫，石棺和他们三人就被水流顶出了墓室。<br/>　　<br/>　　    这里应该算是金字塔的上半部分了，距离塔尖不过几十米的距离，湖水正从顶端倾泄而下，由于金字塔的底部和中部是面积最大的区域，当这两处地方被填满了以后，水位上升的速度就越来越快了。<br/>　　<br/>　　    四个人仰着脑袋，望着金字塔顶，此时他们心里惦记着的，祈祷的，就是湖面的黄九郎和午桥等人，能够看到现在抚仙湖的异样，然后准备着过来接应，如果一段时间后，湖面上都没有人的话，恐怕他们几个就算逃出金字塔也得葬身在湖下了。<br/>　　<br/>　　    等待是漫长也煎熬的，特别是在这种生死攸关的时刻。<br/>　　<br/>　　    无疑，他们四人只能把自己的性命寄托于外界了。<br/>　　<br/>　　    “你说午桥和菜刀文还有陈三岁他们都能靠谱吧？”王令歌惆怅的问道。<br/>　　<br/>　　    “他们肯定靠谱，这点智商还是能有的，所以这也是我为什么探古城时非得要把你们都给找来的原因，真要是出了事的话，还得是自己人可靠，换成别人的话，我们恐怕就得提心吊胆了”王惊蛰叹了口气，忽然看着棺材里的李菲儿，问道：“你想好了么？”<br/>　　<br/>　　    “啊，什么想好了？”李菲儿茫然问道。<br/>　　<br/>　　    “说辞啊，当然是我们在水下和金字塔里的经历，见闻了，从这出去后不管是你们研究所，还是黄九郎都得要询问我们这段时间的经历，你得准备好个说辞吧？”<br/>　　<br/>　　    李菲儿瞬间顿悟了，这个典型的高智商和情商的女人，当然明白了他们这一番遭遇，有些事情可以说，有些事情就得睁一只眼闭一只眼了。<br/>　　<br/>　　    李菲儿略微寻思了下，表情复杂的说道：“当祭坛裂开以后，猝不及防下我和你们都被吸进了一条水道中，进入了古城中金字塔的内部……后来我们发现了滇国国主党羌的墓室，有他的棺木和陪葬品，同时我们也找到了开启金字塔墓葬的方式……”<br/>　　<br/>　　    李菲儿很聪明，这一段经历她绝大部分都如实说了，但唯独关于金字塔下面那篇国殇文里的内容，还有林汶骐取走的四十九张人皮她给抹了下去，李菲儿其实也明白了，王惊蛰等人没有灭她口的心思。<br/>　　<br/>　　    金字塔里的经历，随便他们怎么去编造外人也不会发现其中的隐秘，毕竟当金字塔上方裂开湖水倒灌进来，从此以后恐怕没有人能够在进入金字塔的内部了，除非等到哪一天抚仙湖的湖水全部消失了，当这座古城全部显露在阳光下，才可能会有人能够进得去，但这可能么？<br/>　　<br/>　　    也许可能，但绝对已经是千年以后了，金字塔里的所有秘密和不解之谜，势必得要就此深埋在湖底了。<br/>　　<br/>　　    “你这么说的话，乍一听是没啥错误，但是呢稍微一琢磨的话，也有想不通的地方”林汶骐皱眉说道：“党羌为啥要费尽心思的把整个滇国都给淹了，总得能有个说得过去的原因吧？所以，你在交代的时候，金字塔下方第一层的事如实去说，该怎么讲究怎么讲，把后面略过去就行了”<br/>　　<br/>　　    “可是如果他们怀疑了呢？”<br/>　　<br/>　　    “怀疑啥啊？呵呵，谁怀疑谁就来查啊，金字塔就在水下呢，你要是不信自己下来看呗！”<br/>　　<br/>　　    李菲儿：“……”<br/>　　<br/>　　    于此同时，湖面上，凭空出现的漩涡开始慢慢缩小，直径急剧收缩，没过多久水面就渐渐的平静了下来。<br/>　　<br/>　　    “快，下水！”眼见湖水已经平静了，午桥当即率先就跃入了湖中，紧接着其他人纷纷跳水，迅速下潜。<br/>　　<br/>　　    当蛙人和午桥他们都下潜后，黄詹感叹着说道：“李菲儿和王惊蛰他们真要是能被救上来的话，无疑会出现让人震惊的两方面，一是滇国古城的秘密，二就是他们如何在水下存活下来的。”<br/>　　<br/>　　    张生利皱眉说道：“可能么，一天一夜过去了，他们还能活着？”<br/>　　<br/>　　    黄九郎摇头说道：“未必不能，假如他们真的进入了金字塔内部的话，这个希望还是很大的，从先前的探查和发现上看，金字塔的密封性非常好，也许过去了这么多年，内部都没有渗透进湖水，如果里面依旧保持原样的话，在湖水的挤压下，金字塔里是会存储着大量的空气的”<br/>　　<br/>　　    张生利顿时愣住了，一脸惊异，黄詹接着说道：“外部的气压压着水压，金字塔内部的气压也压着水压，二者平衡，所以金字塔里既不会进水也会有空气，如果他们真的能在里面的话……”<br/>　　<br/>　　    黄詹和黄九郎说的这种情况很常见，以前也发生过多次，几年前两名洞穴潜水爱好者在马来群岛海域的水下进行洞穴潜水探险时，进入了一片复杂的洞穴区域，后来因为意外导引绳断掉了，他们两个被困在了洞穴中没能出去，当两人的氧气瓶已经耗尽时，眼看着就要绝望的葬身海底后，他们发现了一个庞大的洞穴。<br/>　　<br/>　　    洞穴上方是空的，下面是海水，这个洞穴里因为空气和水压的原因，储存了大量的空气，两人从水中钻出来进入到洞穴里，在里面足足被困了六十个小时，最后才获救。<br/>　　<br/>　　    这种状况俗称气穴，抚仙湖水下的金字塔无疑就是个庞大的气穴，从这一点上来看，你不得不佩服两千多年前的古人，他们的智慧和经验，一点都不比两千年后的现代人差。<br/>　　<br/>　　    当午桥和蛙人进入湖水中开始下潜以后，就迅速朝着古城方向前进，当头顶的强光电筒隐约能看见水下的那座金字塔时，在顶部方向，这时突然冲出几道人影。

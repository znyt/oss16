　　代码小镇，教堂。
<br/>　　这是小镇上唯一一座教堂，这座教堂也是有着很长久的岁月，它存在的时间应该跟歌剧院差不多。
<br/>　　小镇上的居民每周都会到教堂中来祷告，而来小镇上来旅游的游客也是会来到这里参观，这种有着上百年岁月，带着浓厚欧洲风格的教堂可并不多见。
<br/>　　教堂四周用了五颜六色的彩色玻璃作为一种装饰，天晴的时候，阳光穿过这些彩色玻璃，会在教堂地面和桌椅上留下斑斓的色彩。
<br/>　　教堂中没有灯，使用蜡烛作为照明工具，年过半百的老教父坐在耶稣像前，手里捧着一本圣经，嘴里念念有词，只是他的声音很轻，根本听不清楚他在说些什么。
<br/>　　哒，哒，哒，哒……
<br/>　　教堂门口传来一串脚步声，几个身穿带有红色镶边银白兜帽衫，蒙着口鼻的刺客走进教堂，他们脚上的皮靴踩在地面上，发出“哒哒”的声响。
<br/>　　坐在耶稣像前的老教父合拢手里的圣经，对那个用袖剑杀死了Henry的领头刺客问道：“事情做完了？”
<br/>　　“一命换一命。”刺客头领用拉丁语回答道：“我需要跟他联系。”
<br/>　　“去吧。”
<br/>　　老教父重新打开圣经，嘴里继续念念有词起来。
<br/>　　刺客头领对着跟在自己身后的几个刺客挥了挥手，几名刺客立刻消失在了黑暗中，而他自己则是走向竖立在教堂正前方的耶稣像，他耶稣像前站定脚步，轻轻一低头，随后绕过耶稣像，走到耶稣像背后一个暗门处，打开大门走了下去。
<br/>　　刺客头领走到一张桌子前，唤醒桌子上进入休眠状态的笔记本电脑，发起一个视频聊天。
<br/>　　一个身穿带有浓厚中世纪风格服饰的刺客坐在一部笔记本电脑前，并且进行熟练的操作，实在是有着说不清楚的违和感。
<br/>　　视频聊天很快就建立起来，蛰龙出现在笔记本电脑中，而在蛰龙身边，还坐着一个面目苍老，头发银白，身上同样穿着一件银白色兜帽衫的老人，只是他身上兜帽衫的镶边却是金色的，这是一种身份的象征。这个老人叫安东尼奥斯，是教廷“刺客会”权力最大的存在，被称为“鹰皇”。
<br/>　　蛰龙右手腕下，一把袖剑的剑刃抵在老人的脖子上，蛰龙看着电脑屏幕中的刺客头领，问道：“事情完成了？”
<br/>　　“完成了。”刺客头领说道：“按照之前的约定，龙鹰你现在可以放人了。”
<br/>　　Henry的死，可以说是刺客头领刻意为之，其实当时Henry并没有完全踏入镇口红线内，但是没有人注意到，即便是Henry自己也是，刺客头领抓住机会从背后给他一剑，结束了他的性命，而这一切都是因为蛰龙的关系。
<br/>　　蛰龙把袖剑从安东尼奥斯的脖子上移开，把袖剑收起，说道：“我可不像某些人，说话像放屁，既然你完成了我要求的事情，我自然会放人，不过在我离开之前，我觉得我应该跟‘鹰皇’好好聊聊。”
<br/>　　“龙鹰，你到底想要干什么！？”刺客头领从椅子上站起来，盯着笔记本电脑中的蛰龙，恨不得自己有穿越功能，直接从笔记本电脑中穿过去，回到梵蒂冈的“刺客会”总部。
<br/>　　“放心，我只是要跟‘鹰皇’聊聊。”
<br/>　　蛰龙移动鼠标，把箭头停留在窗口右上角的红叉上，眼中带着轻蔑之色看着刺客头领，说道：“如果我真的想要杀他，你觉得包括你在内，有什么人可以阻拦我？”
<br/>　　刺客头领沉默了，尽管他非常不愿意承认，但他很清楚蛰龙所说的是事实。要不然，他也不能如入无人之境，把“鹰皇”安东尼奥斯给绑了，并且来要挟自己。
<br/>　　蛰龙关闭了视频聊天的窗口，移动了一下椅子，坐在安东尼奥斯对面，说道：“鹰皇，我们有多少时间没见了？”
<br/>　　“差不多有十年了。”安东尼奥斯回忆了一下，说道。
<br/>　　“准确的说，是十年三个月零九天。”蛰龙说道：“当年，你还能飞檐走壁，执行刺杀任务，但是现在你必须承认，你老了。你已经跑不动，跳不高了。”
<br/>　　“是啊，我老了。”
<br/>　　安东尼奥斯叹了口气，似乎是在感叹岁月的无情，那双苍老但是却依旧明亮的眼睛看着蛰龙，说道：“我现在真的有些后悔，让你成为我的学生。”
<br/>　　“不，你后悔的不是让我成为你的学生。”
<br/>　　蛰龙眼睛中闪过一道厉色，右手一握，他右手腕下的袖剑突然弹出，剑刃在空气中闪过道道寒光，盯着坐在自己面前的安东尼奥斯说道：“你后悔的，没有杀掉我的老师，没有能让我按照你的计划成为一把你手里的利剑。当年老师找到你，让你教我刺杀之术的时候，你在发现了我的天赋和能力就暗中打定主意，要让我成为你的人，成为你手里的杀人之剑，唯你的命令是从。只是你没有想到……”
<br/>　　“没有想到他居然会用自己的命换你的命。”安东尼奥斯显然是承认了蛰龙的话。
<br/>　　“你那一剑距离老师的心脏不到一公分，只差一点，他就死在了你这个‘朋友’手里。”
<br/>　　蛰龙的脸色越发冷冽起来，比他手中的袖剑还要冷。
<br/>　　“所以，你现在是想要杀了我吗？”安东尼奥斯对蛰龙问道。
<br/>　　“不，我不会杀你。”蛰龙回答道：“不是我不想杀你，而是我不会在这里杀你，‘刺客会’毕竟是从中世纪就存在，并且一直延续到现在，就算是我也没有充足的把握在杀了你之后，能安然无恙的走出这里。”
<br/>　　蛰龙站起身，走到安东尼奥斯身边，左手用力的按在他肩膀上，说道：“鹰皇，如果你想安然度过自己剩下的岁月，而不是死于非命的话，最好就一直待在这里不要乱走。”
<br/>　　“龙鹰！”
<br/>　　安东尼奥斯叫住蛰龙，问道：“你我毕竟师生一场，‘龙鹰’这个名字是我给你的，我能知道你原来的名字吗？”
<br/>　　“一个刺客是没有名字的，这是你教给我的，难道你忘了吗？”蛰龙瞥了眼自己身后的安东尼奥斯，说道：“外面的人都喜欢叫我‘幽灵刺客’，或者是‘蛰龙’。至于我原来的名字……，你没有资格知道，因为你不配！”
<br/>(本章完)

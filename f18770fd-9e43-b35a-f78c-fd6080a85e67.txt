　　离开主宫殿。
<br/>　　萧逸一行人，往其它宫殿而去。
<br/>　　这个洞府遗迹，毫无疑问，便是当年的暴雪山庄。
<br/>　　方圆百里，共计数百座宫殿，每座宫殿，皆有宝物。
<br/>　　虽说都远远不如主宫殿中的宝物。
<br/>　　但所有宫殿的修炼之物加起来，乃是一笔相当惊人的财富。
<br/>　　当年的暴雪山庄，盛极一时，其底蕴，自然是极其惊人的。
<br/>　　一路走着，不时还能见到少量武者。
<br/>　　这些武者，之前虽知晓主宫殿出世，却并未前往。
<br/>　　自然也没有参加围攻萧逸之事。
<br/>　　至于之前在主宫殿中围攻萧逸的人，包括韩苍、风花月等人，因怕萧逸报复，早已悉数离开了这片洞府遗迹。
<br/>　　边走着，白冰雪、钟无忧等四人，各自离去。
<br/>　　只剩柳嫣然和林劲，与萧逸并肩走着。
<br/>　　“抱歉。”林劲忽然停下了脚步，说道。
<br/>　　柳嫣然同样停下了脚步，面带歉意。
<br/>　　“抱歉什么？”萧逸笑笑，道。
<br/>　　“王都的事，我们都知道了。”林劲的脸色，有些复杂。
<br/>　　当日萧逸在王都大战，林家，可是站在了萧逸的对立面。
<br/>　　柳家，虽并未参与其中，却选择了袖手旁观。
<br/>　　两人当日虽不在场，却也能想象到萧逸当时孤身一人，面对整个王都所有强者的那种孤独、无助。
<br/>　　以及面对各种不公之下的愤怒。
<br/>　　特别是易老之事。
<br/>　　他们很清楚，易老与萧逸间的关系，以及那种无法言喻的情谊。
<br/>　　那等打击，怕是常人难以接受的吧。
<br/>　　萧逸笑笑，他知道两人在想些什么，摇了摇头，轻声道，“当日之事，与你们无关。”
<br/>　　“再者，以我们间的情谊，需要说那些吗？”
<br/>　　事实上，早在第一轮冰海之缘时。
<br/>　　萧逸便已经发现了林劲和柳嫣然。
<br/>　　不过，萧逸并未去找他们。
<br/>　　他们虽分别是林家与柳家的下一代接班人。
<br/>　　但，林柳两家做出的选择，与他们无关。
<br/>　　萧逸并未怪责过他们。
<br/>　　只不过，自己如今在王都的身份太过敏感。
<br/>　　他并不想为这二人带去什么不必要的麻烦。
<br/>　　“对了，你们怎地出现得这么及时？”萧逸岔开话题，问道。
<br/>　　林劲回答道，“主宫殿出世时，我们自知不可能有机会得到里头重宝的。”
<br/>　　“故并未前往，而是在其它宫殿继续寻找宝物。”
<br/>　　“就在那时，我们恰好发现了血无伤和血焰。”
<br/>　　“这二人，明明参加了冰缘大会，却藏头露尾。”
<br/>　　“我们猜测着，他们肯定是冲着你去的。”
<br/>　　“于是，便一直跟着他们。”
<br/>　　柳嫣然笑道，“幸好我们留了个心眼，否则这两个小人就得逞了。”
<br/>　　冰缘大会，是整个炎龙大陆的盛事，各方天骄，皆可参加。
<br/>　　炎武王国方面，许多大势力都是有名额的。
<br/>　　药王谷、血雾谷这个层次，有三个名额。
<br/>　　不过，血雾谷能派出的人，只有血无伤和血焰二人。
<br/>　　而王都四大家族方面，则各有两个名额。
<br/>　　说起来，除了极寒之地本身的少数大势力，拥有四个名额外。
<br/>　　其余各方，几乎所有势力，都只有二至三个名额。
<br/>　　裂天剑宗，算是这里头唯一拥有5个名额的。
<br/>　　血无伤和血焰，两人其实早就来了，只不过一直戴着面具，隐藏了身份。
<br/>　　在第一轮时，萧逸也未曾发现他们。
<br/>　　毕竟，参加的人数，足有一千五百多人。
<br/>　　萧逸不可能个个都去感知一番气息。
<br/>　　再者，武者在外历练，戴个面具，并非什么稀奇之事。
<br/>　　参加者中，身戴面具，或披宽大袍子，隐藏身份的，不在少数。
<br/>　　毕竟是整个大陆的盛事，各方势力都有参加。
<br/>　　言归正传。
<br/>　　这时，林劲愤愤说道，“那群小人，如此卑鄙，此事可不能就这样算了。”
<br/>　　萧逸点点头，道，“不急，待得冰缘大会结束，我再慢慢找他们算账。”
<br/>　　……
<br/>　　第二轮遗迹之缘，持续时间为10天。
<br/>　　10天内，除了本身参加冰缘大会的武者外，各势力长辈，不得进入。
<br/>　　甚至包括主办方方面，也不能派遣裁判之类的进入。
<br/>　　第二轮，不需要裁判，也没有排名之分。
<br/>　　这是只属于各位参加者自己本身的机缘。
<br/>　　10天后，各势力的长辈和强者，才会进入。
<br/>　　此举，只是以防参加者们在危险的洞府遗迹内若出现意外，他们能及时救援。
<br/>　　若10天后，所有人都安全出来，便无需多此一举。
<br/>　　10天时间，萧逸一行人，横扫了整片洞府遗迹的所有禁制。
<br/>　　数百座宫殿，所有宝物，被收取一空。
<br/>　　这些宝物，萧逸没拿半点儿，全归白冰雪、凌羽、林劲等六人。
<br/>　　10天后。
<br/>　　所有人，悉数离开了这个洞府遗迹。
<br/>　　当然，也有少数一些人，永远地留在了这里。
<br/>　　那些或因冒失不够谨慎，或因贪婪、实力不足的武者，成了这里漫天暴雪之下的一具具冰冷尸体。
<br/>　　……
<br/>　　哗啦一声。
<br/>　　萧逸一行人，穿过了屏障，回到了雪林当中。
<br/>　　各大势力强者，早已等候在此。
<br/>　　所有目光，在一刹那，全部聚集到了萧逸身上。
<br/>　　目光各异，或怨毒、或愤怒、或妒忌，但更多的，却是狂热。
<br/>　　怨毒的目光，来自于韩苍等一千多位武者。
<br/>　　他们早在刚逃离主宫殿时，便出了洞府遗迹。
<br/>　　第二轮中，可大打出手、可杀人夺宝。
<br/>　　他们自然不敢在洞府内多留。
<br/>　　萧逸一剑击杀三头雪地金刚猿那一幕，他们历历在目。
<br/>　　简单来说，他们在第二轮中的收获，几乎为零。
<br/>　　而狂热的目光，则来自于各大势力的强者。
<br/>　　不夜宫主、雪刀门主等极寒之地赫赫有名的强者，走到了萧逸面前。
<br/>　　“萧逸小友，出个价吧。”不夜宫主率先说道。
<br/>　　第二轮，遗迹之缘，不分排名。
<br/>　　但参加者，其实也能得到额外的‘奖励’。
<br/>　　那便是交易。
<br/>　　在洞府里头得到的任何宝物、功法武技、武道传承等等。
<br/>　　只要各势力强者有兴趣，皆可交易。
<br/>　　当然，卖不卖，全凭个人意愿。
<br/>　　现今，不夜宫主等人，面露狂热之色。
<br/>　　自然是知晓了萧逸得到暴雪庄主传承之事。
<br/>　　暴雪庄主，乃是八百年前声名极盛的至强者。
<br/>　　其成名时间，甚至比裂天剑魔前辈更早，那是另一个时代的人物。
<br/>　　但，在极寒之地这边，暴雪庄主，便是最强的传奇。
<br/>　　他的传承，自然也让人趋之若鹜，无比狂热。
<br/>　　......
<br/>　　第一更。

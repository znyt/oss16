　　青年被小医仙拒绝后，脸庞上没有丝毫的恼怒，淡淡一笑，手掌一挥，大喝道：“狼头的人，都给我注意点，现在快要进入魔兽山脉了，可别在阴沟里翻了船。”
<br/>　　“是，少团长！”
<br/>　　听到青年的喝声，周围地几十名大汉顿时齐声迎合，整齐的声调仿佛是专门训练过似得，引得众人侧目。
<br/>　　非常满意这种反应，青年微微一笑，逼气十足，加快了步伐，与小医仙并肩而行，极为殷切的贴身护持着。
<br/>　　“妈的，不就是仗着他父亲是狼头佣兵团的团长嘛，难道他以为凭这样就能得到小医仙的芳心啊？”
<br/>　　看到能够近身与小医仙聊天的青年，慕清霄身旁一名佣兵汉子，顿时低声骂骂咧咧起来，语气中满是浓郁的酸气味。
<br/>　　面无表情，慕清霄目光在那名青年身上扫过，最后停留在其胸口上的一枚徽章上，徽章上雕刻着一只独眼的狼头。
<br/>　　视线跃过青年，望着他身后三十几名胸口佩戴同样徽章的佣兵，慕清霄微微一笑，按照现在看来，自己拿到了骨灵冷火，剧情并没有产生变化。
<br/>　　在魔兽山脉外面，一切倒还算平静，然而当队伍刚刚进入魔兽山脉不久后，便引来一次小规模的魔兽袭击。
<br/>　　三头一阶魔兽赤冰蛇，掉悬在树干上，撑趁着下方人员的不备，犹如闪电一般，非常轻易地将寒毒输入进了三名佣兵体内。
<br/>　　顿时，三名脸色惨白如冰雕的汉子，便是手脚冰凉的软了下来。
<br/>　　遭受到攻击，佣兵们顿时含怒出手，片刻时间，三头一阶魔兽便是不出意外的被群殴致死。
<br/>　　而三名中了寒毒的佣兵，在赤冰蛇被解决后，便是被迅速的送到了后方的采药队中，由小医仙亲自出手，替他们将体内地寒毒驱逐了出去。
<br/>　　随着用兵队伍的深入，小医仙清脆的声音，忽然在安静的队伍中响起：“各位，这里已经接近采药区了，请休息一下吧，走了这么久，恐怕大家也累了。”
<br/>　　前进的脚步微微一顿，整个队伍都是不约而同的停止了下来，回过头，望着那笑容中不含丝毫杂质的白裙女子，极为老实的点了下头。
<br/>　　在迅速商量过后，十几名佣兵分散到四周警戒着，至于其他人则原地而坐，回复着因为赶路而大量消耗的气力。
<br/>　　在众人休息的同时，在慕清霄的神识范围内，有多名身影却悄悄的离去，而小医仙的身影就在其中。
<br/>　　微微一笑，慕清霄隐匿气息，也是悄无声息的离开队伍。
<br/>　　跟在神识中的身影后面，越来越深入小密林中，再次行走了一段距离，昏暗的视线微微明亮起来。
<br/>　　慕清霄径直走了出去，出现在前面的是一面有些陡峭的悬崖，在悬崖的下面，布满着郁郁葱葱的绿林，颇为美丽。
<br/>　　视线在悬崖边缘扫了扫，慕清霄视线定格在悬崖边一处盛开着淡白色花朵的植物面前，低声道：“看来，没找错地方。”
<br/>　　这株植物，开放着淡白花朵，隐隐盛开的花朵中，有着赤红色的果实，若隐若现，一股淡淡的药香味，从中散发而出。
<br/>　　目光在这株植物上仔细的扫了扫，慕清霄微微蹲下，就在他要摘采植物的时候，一只如玉般的洁白小手，突然从面前悬崖中伸探而出。
<br/>　　目标也是这株植物，只是，由于慕清霄的关系，却是一把抓在了他的手上。
<br/>　　玉手刚刚触碰到慕清霄的手掌，在略微呆泄过后，犹如触电一般的收缩了回去。
<br/>　　下一秒，一张泛着空灵气息的柔美脸颊，便是从悬崖下露了出来，略微泛着点点慌张的望着蹲在面前，脸上挂着淡笑的青年。
<br/>　　悬崖之上，两人的目光，便是这样呆愣愣的对视着，画面显得有些诡异。
<br/>　　“那个...能拉我一下吗？”
<br/>　　对视片刻后，小医仙终于率先打破尴尬，声音极其柔软，听起来很舒服，望着面前的青年，美眸中闪过一丝惊诧。
<br/>　　慕清霄微微一笑，一把抓住小医仙伸出的玉手，轻轻一拉，后者便从悬崖下跃身而起，娇躯在半空中划起美妙的弧线，轻灵的跃上了山巅。
<br/>　　“谢谢。”
<br/>　　脚步轻轻落在地上，小医仙低声道了一句谢，快速挣脱慕清霄的手掌，旋即目光不着痕迹的扫过悬崖边缘，纤指捋过额前的青丝，视线落在慕清霄神骏的脸颊上。
<br/>　　呆泄片刻，小医仙自惊愕中回过神来，轻声道：“我记得，你是此次万药斋聘请的佣兵护卫吧？”
<br/>　　“嗯，居然能让小医仙姑娘记住，在下是不是应该荣幸呢？”慕清霄轻轻一笑，回味了一下先前手中的柔软。
<br/>　　小医仙盯着慕清霄的脸蛋，道：“你的容貌，别人想不记住都不行吧？对了，看你想摘这株药草，你难道认识？”
<br/>　　慕清霄闻言，语气平淡，道：“白兰果，在中级药材中也算稀少，这枚成熟的白兰果拿去药铺，估计能换取四千多枚金币。”
<br/>　　听慕清霄直接道出白兰果的信息，小医仙美眸中掠过一丝讶然，有些惊异道：“你也学过辨别药草？”
<br/>　　“学过一年吧。”
<br/>　　跟药尘学了一年的时间，他仅仅几个月的时间，就将基础药草，中级药草，高级药草都学了个遍，底蕴远不是他人能比的。
<br/>　　“虽说见者有份，不过这枚白兰果毕竟是你先看到的，那我就不夺人所好了。”
<br/>　　微微一笑，小医仙蹲下身子，小心翼翼地将赤红色的果实从花朵中取出，然后递向慕清霄。
<br/>　　慕清霄接过果实，古怪的说道：“白兰果价值不菲，姑娘竟白白让给我，真是菩萨心肠。”
<br/>　　或许是被慕清霄一直盯着，小医仙心虚的移开目光，但美眸中却有些窃喜，道：“行了，队伍恐怕已经要歇息完毕了，我们快些回去吧。”
<br/>　　手中把玩着冰凉地白兰果，慕清霄嘴角牵起一丝古怪的笑容，真是鬼精鬼精的小妮子，悬崖下面的山洞又怎么可能瞒得过他。
<br/>　　他本就是冲着山洞里的东西来的，不过，也不急于一时，陪她玩耍一番好了。

　　第745章
<br/>　　程大雷和百里胜相携走入礼部大堂，一路遇到的官员纷纷侧目。一方面，是大家对这程大雷多少有些好奇，只闻其人，却从未见过他的庐山真面目。另外一方面，大家也震惊百里胜对程大雷的态度。
<br/>　　原来二人关系这么好。
<br/>　　百里胜的态度，就代表了当今天子的态度。如此，大小官员不敢对程大雷有任何慢待，谨小慎微的满足他提出的要求。
<br/>　　很利落的便为程大雷安排好了住处，百里胜还提出要送程大雷十名美眷，随时伺候程大雷。
<br/>　　“嗳，带姑娘来的，况且公务在身，还是为陛下分忧要紧。”谁知道你送来的人里面有没有奸细，或者十个人都是奸细。
<br/>　　“程当家做事果然令人佩服，如果多几个你这样的臣子，陛下就可以安枕无忧了。”如果多几个程大雷，帝国更不知会乱到什么地步。
<br/>　　“一样，一样。”程大雷拱拱手：“那宋伯康如何了？”
<br/>　　百里胜抬起头：“八月十五，将在天下藩王前问斩，陛下亲自监斩。”
<br/>　　程大雷读懂了百里胜眼神中的杀意，这杀意不仅是对宋伯康，也有对自己的。看来凉州那道梁子，他心里并没有搁下。
<br/>　　“那可真是大快人心呐。”程大雷呵呵笑着，杀宋伯康可以，想要我的脑袋你试试看。
<br/>　　程大雷第一次入京时无人问津，还真就是百里胜对他格外青眼。今时不同往日，百里胜特意为他挑了一座大宅子，在所有藩王中也算是数得上的。虽然那十名美妾未曾送到，但同样有下人仆役。
<br/>　　这一双双眼睛，随时盯着程大雷的一举一动，然后报告给他们背后的人。
<br/>　　如今距离八月十五还有些时日，各地藩王纷纷入京，人还没有到齐。程大雷等着八月十五的正日子，到时候先斩宋伯康，再封王。
<br/>　　徐神机待着有些无聊：“大当家，要不我和刘发财出去逛逛，好不容易来一次长安。”
<br/>　　“去吧。”程大雷在屋檐下晒着太阳：“别人若是绑了你，我可不管去赎。”
<br/>　　“呃，天子脚下，应该不至于吧。”
<br/>　　“天子脚下，为何就不至于？”程大雷笑笑，现在帝国还有几人把当今天子当回事。
<br/>　　丫鬟服侍着给程大雷添水，程大雷抬起头冲她笑笑，才发现她脸色苍白，手腕不停颤抖。
<br/>　　程大雷摸了摸鼻子，心道：我有那么吓人么。
<br/>　　终究是恶名在外，尤其是天子脚下的文明人。小孩听到程大雷的名字便不敢大声哭泣，都以为他是吃人的魔王，色中的饿鬼。这批丫鬟下人被派遣到此地时，人人如丧考妣，当场还吓晕了一人。
<br/>　　长安城最近很是热闹，在程大雷入住不久，城门口来了两人。
<br/>　　陈梦仍旧身背双斧，牵着斑点马，在他身旁还有一人，四十岁出头年纪，黑紫脸膛，马背上搭着一柄大斧。
<br/>　　城门守军立刻迎了上来，普通人是没有携带兵器的资格，所谓寸铁为凶，尤其长安城又是天子脚下。任谁敢私带兵器，城门守军都有立刻缉拿的权力。
<br/>　　当然，这仅仅是对一般人而言。看二人就将兵器带在身上，丝毫遮掩也不做，想也知道不是一般人呐。
<br/>　　守军迎上前去，先抱拳道了个喏。陈梦掏出随身的令牌在他面前晃了晃，根本没与他招呼。守军一见，原来是平野城城主的公子，他们立刻让开一条道路，二人大摇大摆的入城。
<br/>　　紧跟其后，有一江湖豪客，知道这次藩王入京，长安城会很热闹。便想着来撞撞运气，习得文武艺，货与帝王家，说不得就能走了狗屎运，被某家藩王看上，从此后便出将入相。
<br/>　　他本身也知道长安城的规矩，特意将自己的朴刀藏了起来。眼见陈梦二人明目张胆的入城，便以为这规矩不甚严，就大咧咧的往前走。
<br/>　　谁料一眼便被城门守军发现，对方将脸一沉：“来人，抓了。”
<br/>　　“喂，凭什么呢？”
<br/>　　这好汉一激灵，本想与守军分辨个道理，或者拿起刀一战，结果发现十余名守军已经将自己团团围住。
<br/>　　“去你娘的，老子不干了。”
<br/>　　大汉咆哮一声，抓出刀拼着命打开一条路，闯出了守军的包围圈，打定主意，这辈子都不入长安城了。
<br/>　　那紫脸汉子在路边一茶水摊歇脚，陈梦并不在他身边，不多时，陈梦回来，也在他身边坐下。
<br/>　　“打听清楚了，已经知道程大雷的住处。”
<br/>　　紫脸汉子惜字如金：“喝茶，等下过去看看。”
<br/>　　“嗯。”
<br/>　　陈梦重重的点点头，这紫脸汉子是他的授业恩师。年少时在江湖上闯荡，后来被仇人追杀，是陈霸祖出手相救，于是他便留在平野城。关于他的来历，无人知晓，甚至都没人知道他的名字，陈梦对他也一直是以师父相称。
<br/>　　“师父，那姓程的功夫了得，徒儿与他交过手，并不是徒有虚名之辈。”
<br/>　　无名汉子点点头，却又摇摇头：“你生来神力，这是你的造化，学到今日，用斧头能胜过你的人屈指可数。不过，你从未与人真正动作手，吃亏在没有经验。经此一败，对你也未必不是好事。”
<br/>　　“这次还要仰仗师父出手，给程大雷一个厉害瞧瞧，师父在府中隐姓埋名这么多年，也该是让世人知道知道了。”
<br/>　　紫脸汉子长长叹口气，他一身武艺，自量罕逢敌手，只不过为了报恩，才十年不曾出世。这次来长安挑战程大雷，一为提徒弟出口气，二就是为了扬名。
<br/>　　能做战场上领军百万的一军之帅，谁愿只做一座城主府的小小教师。
<br/>　　“不过，那程大雷的住处有官兵看守，想让他应战也并不容易呐。”紫脸人皱起眉头。
<br/>　　“嗨，我说这是谁，真巧，就在这里碰见了。”
<br/>　　二人正说着话，忽听一道刺耳的声音响起。陈梦抬起头，就见程大雷一张大脸杵在自己面前。
<br/>　　程大雷丝毫不客气，领着徐神机和刘发财坐下，自顾自招呼老板。
<br/>　　“来壶好茶，再来几个小菜。”
<br/>(本章完)

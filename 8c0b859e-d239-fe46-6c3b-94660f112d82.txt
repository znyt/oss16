
<br/>　　<br/>　　    路上，王惊蛰和常文书往京城走，这时候已经都到下午了，上车离京的时间已经错过了，自然就得改期才能再走了。<br/>　　<br/>　　    王惊蛰手里拿着那条白蛇从洞里带出来的盒子，翻来覆去看了几遍但也没打开，这东西从外表来看显得特别古朴和沧桑，根本就考量不出是什么年代的，质地是木头的，但什么木料他也没看出来，可这就挺有意思的了，因为一个不知道是什么年代的盒子，到了如今不知隔了多久都没有风化，还依旧保持着原样。<br/>　　<br/>　　    盒子顶端表面，刻着一个鼎的图案，除此以外就啥也没有了，显得非常简单和粗糙，工艺明显不咋样。<br/>　　<br/>　　    常文书开着车，转头看了两眼后，问道：“那条蛇带出来的，什么玩意儿啊，你怎么不打开看看？”<br/>　　<br/>　　    “就是不知道是什么，我才不敢打开，这盒子的密闭性不错，把里面的东西和空气都给隔开了，我怕一打开话容易让这里的东西氧化了”王惊蛰说的是个常识性的问题，如果这里的东西因为和空气隔开而一直保持原样的话，可能一打开就会快速的变质了。<br/>　　<br/>　　    常文书诧异的说道：“打开看一眼，不行，你就再扣上，要不然能咋办，就因为这玩意儿你还得整个真空密室啊？”<br/>　　<br/>　　    “你要是这么说的话，那还真有点道理……”王惊蛰无语的点来了点头，然后谨慎的慢慢的开了盒盖，看了眼里面的东西后，他就有点愣住了。<br/>　　<br/>　　    这里面放着的应该一块胛骨，大概一巴掌左右大小，骨头有些发黑，但没有出现骨头疏松后的洞眼，胛骨上面刻着一段文字，有点类似于甲骨文那种，王惊蛰粗略的扫了几眼后，一个字都没认出来。<br/>　　<br/>　　    “我还以为是什么宝贝呢，白好奇了……”常文书顿时不感兴趣了。<br/>　　<br/>　　    王惊蛰从盒子里拿出那块胛骨，翻过来看了几眼，然后举在眼前仔细端详着上面的文字，他几乎一个字一个字的从头看到尾，粗略的算了算，这上面差不多有过百个字，其中他差不多猜测出来有那么几个字大概是什么意思了。<br/>　　<br/>　　    这种类似于甲骨文的文字，很多都是象形字，就是根据物体的形象而演化出来的，他能瞧出来的那几个，一个是草字状，还有两个是一簇火苗和水波纹的意思。<br/>　　<br/>　　    “啪”王惊蛰把盖子又扣上了，然后看着表面刻出来的鼎，心里隐约有点念头，但也还惊疑不定。<br/>　　<br/>　　    “我送你去哪？回潘家园那边，还是在车站找个住的地方，等改签了之后再走？”常文书看王惊蛰一直没动静，就开口问道。<br/>　　<br/>　　    “先不走了，过两天再说”王惊蛰琢磨了一下，手指挺有节奏的敲着盒子，说道：“你能不能找到一些专门研究古文字的人，一般历史系的教授或者学生，再不就是考古的也行”<br/>　　<br/>　　    常文书瞄了眼他手里的盒子，顿时就明白了，点头说道：“找找关系应该不难，京城的大学挺多的，这个口的我有认识人”<br/>　　<br/>　　    王惊蛰说道：“那行，回头我把这上面的字全都抄下来，然后交给你，不过有个问题你得注意下，这些文字你不能交给一个人去辨认，多找几个来，再把这些文字打散了拿给他们”<br/>　　<br/>　　    “你太滑了，这是不想人知道，这里面的是什么啊？”<br/>　　<br/>　　    “有些东西，越少人知道就是对自己的保护，在我不知道这上面写的是什么之前，还是就我自己知道得了”<br/>　　<br/>　　    这天，本来要离京的王惊蛰，因为一件意外事选择了暂时继续留在京城，他已经隐约能猜出这盒子里胛骨上刻着的文字，大概是哪方面的意思了，并且这东西的真实性绝对极高。<br/>　　<br/>　　    白蛇通灵，在感知方面远比人要灵敏得多，传说人有六感，而成了精的畜生则有八感，多出来的那两个一是对危险的提前感知度，这一点绝大部分的动物都有这种本能，再有一种就是对于天才地宝在天性上的认知。<br/>　　<br/>　　    据山海经还有各种古书记载，许久以前各名山大川里，都生长着一些如千年何首乌，人参一类的药材，那时候人类对于这些天材地宝基本还处于慢慢摸索的时代，但一些灵慧的动物，却能察觉到这些难得的草药对自身的好处，所以有些还未成熟的药草旁边，多数都会有牲口守着，等着瓜熟蒂落的时候一口就给吃了。<br/>　　<br/>　　    所以早些年间的山中精怪还颇多，不少都是因此成精的，但到如今这些草药多数都已经绝迹了，自然精怪也少了很多。<br/>　　<br/>　　    这天晚上，王惊蛰仔仔细细的将那块胛骨上的百十来个文字全都抄了下来，又来来回回的对了几遍，保证一笔一划都不会错，不然差了一个字的话，那就可能又是另外一种意思了。<br/>　　<br/>　　    两天过后，常文书给他回了信，说是找了几个大学和研究所熟知古文的历史和考古学家，王惊蛰就将这些文字全都打乱了顺序交给了他，并且还告诉他，千万不要把这消息给漏了。<br/>　　<br/>　　    文字交了出去后，王惊蛰就满是憧憬的等着了，他觉得自己真要是猜准了的话，那恐怕得要涨涨不少的知识了。<br/>　　<br/>　　    其实到现在为止，王惊蛰都没有想到，那条白蛇因为报恩到底给他的是个什么样的天大好处，此时的他是好奇和兴趣占了不少，远没有意识到，陈臣在雾灵山上的遭遇，为他掀开了怎样的一副篇章。<br/>　　<br/>　　    有些事来的太巧，却也太好了。<br/>　　<br/>　　    一晃，多天过去了，足足能有一个星期左右，常文书告诉他，翻译古文有成效了，其中有三个专家都给予了回信。<br/>　　<br/>　　    王惊蛰当即就把他急巴巴的找了出来。

　　听到慕清霄的话语，少年身躯一颤，抬头看了他一眼，随后转身看了一眼，身后面露狠色的青年。
<br/>　　“谢谢，我叫陵越。”
<br/>　　少年向慕清霄道了一声谢，摇了摇头，与青年对峙，丝毫不惧。
<br/>　　“这里本来就是我的位置。”
<br/>　　“你......你们给我等着。”
<br/>　　瞥了一眼慕清霄，青年眼底闪过一抹狠色，显然是将两人给记恨上了。
<br/>　　此时，慕清霄完全不在意青年，反倒是打量起少年。
<br/>　　陵越这个名字，他自然不陌生，未来紫胤真人的首徒，亦是天墉城未来的掌门，只是没想到如此巧合。
<br/>　　若是天墉城没有第二个陵越，面前的少年就是本人了吧。
<br/>　　按照这样来看，距离古剑的剧情开始，恐怕还有很长一段时间。
<br/>　　毕竟，陵越现在还只是一名少年，看样子最多也就十岁，而古剑剧情开始，陵越最起码都有十五六岁了。
<br/>　　收回目光，慕清霄跟着队伍，向前走去，至于陵越如何，与他并没有关系。
<br/>　　刚才想要帮他一把，只是因为他的眼神，与当初的自己有些相似。
<br/>　　望着前面犹如长龙一般的队伍，按照等级步骤，最起码还需要十分钟的时间才到自己。
<br/>　　“下一位，名字，年龄，籍贯。”
<br/>　　在山门口一张桌前，一名身穿天墉城道服的弟子，边问边登记道。
<br/>　　“徐如龙，十七岁，江州人氏......”
<br/>　　随着时间的推移，转眼十分钟就过去，人群不断往前，终于轮到慕清霄。
<br/>　　“姓名，年龄，籍贯。”
<br/>　　“我并非来应征弟子，而是来应征天墉城长老，劳烦通报一声。”
<br/>　　闻言，登记的弟子抬起头来，看到慕清霄后，眼中充满了不可置信，问道：“你真的是来应征长老一职的？”
<br/>　　这名弟子的声音不小，瞬间就将周围的视线全都吸引了过来。
<br/>　　当这些视线落在慕清霄身上后，脸庞都浮现惊愕之色。
<br/>　　不在乎周围目瞪口呆吃瓜群众，慕清霄认真道：“我确实是来应征长老一职的。”
<br/>　　压下心中的震惊，登记信息的弟子努力使自己平静下来。
<br/>　　“应征长老的要求，年龄不得超过百岁，修行至练虚境，你已经是练虚境了？我读书少，你可不要骗我。”
<br/>　　慕清霄眼角抽了抽，道：“有劳通报一声。”
<br/>　　见他认真的模样，似乎不是在开玩笑，登记信息的弟子道：“你稍等一下，我这就去给你通报。”
<br/>　　说着，便找来另外一名弟子，顶替他的位置，至于本人犹如脚下生风，迅速向山顶掠去。
<br/>　　慕清霄则站在一旁，在周围惊诧的目光中，望着山巅等待起来。
<br/>　　陡然，有一道神识在他身上停留几秒后，便悄无声息的消失。
<br/>　　虽然隐晦，但慕清霄依旧清晰的察觉到，神识来自山巅，实力很强，最起码比他现在要强上许多。
<br/>　　半刻钟后，登记弟子返回，来到他面前，态度略微恭敬，道：“请跟我来，掌门真人已经在殿内等候。”
<br/>　　闻言，慕清霄微微点头，在周围目瞪口呆的视线中，跟在弟子身后，向山巅走去。
<br/>　　“卧槽，真的上去了，他的年纪也就二十岁吧，练虚境，这怎么可能！”
<br/>　　高昂的声音响起，顿时在人群中掀起一阵轩然大波，看向慕清霄的背影，眼中都生出了一抹敬意。
<br/>　　正在参加天赋测试的陵越，也是面露惊色，没想到刚才帮自己的，竟是如此强者。
<br/>　　至于先前的嚣张的青年，则面色惨白，腿肚子都在打颤，心中尽是后怕。
<br/>　　...
<br/>　　...
<br/>　　跟在弟子身后，一步步向山巅走去，慕清霄察觉到，越是往上，灵气就越浓郁。
<br/>　　而且，四周的灵气似乎都在向周围汇聚过来的趋势，若是猜的不错，山巅估计布置了某种聚灵的阵法。
<br/>　　五分钟后，终于踏上最后一层阶梯，映入眼中的是一座金碧辉煌的宫殿。
<br/>　　“掌门真人请您进去。”
<br/>　　登记信息的弟子将他送到殿外，通报一声后，微微行礼，便向山下走去。
<br/>　　微微点头，慕清霄就向殿内走去，刚踏进殿内，便有数十道视线投射过来，一齐汇聚在了他的身上。
<br/>　　甚至，站在前面一些穿着不一的老者，议论纷纷，眼中带有审视的意味。
<br/>　　“看来，想应征长老一职的，不止自己一个啊。”
<br/>　　瞥了他们一眼，慕清霄的视线就落在，坐在宫殿最上方的男子身上。
<br/>　　男子身穿一袭紫袍，头顶玉冠，满头白发，却是驻颜有术，仙风道骨。
<br/>　　慕清霄打量着男子，完全看不透，这是他有史以来，第一次看不透对方。
<br/>　　整个天墉城内，拥有这种实力的也就两人，紫胤真人和涵素真人。
<br/>　　那根据刚才弟子的话语，而且能坐在首座上，唯有天墉城掌门涵素真人了。
<br/>　　慕清霄打量涵素真人的同时，涵素真人也在打量着他，不时微微点头，面露赞赏之色。
<br/>　　“阁下可是来我天墉城应征长老职位的？”
<br/>　　“慕清霄见过涵素真人，听闻天墉城正在招募长老，特来应征。”
<br/>　　见他气质高贵，不卑不亢的模样，涵素真人更是满意，道：“听闻阁下已经达到练虚境，不知贵庚？”
<br/>　　见他谈吐，涵素真人根本没有把他当成青年，第一时间质疑他的年龄，说不定和自己一样，是个驻了颜的老怪物。
<br/>　　“在下年方二十，但修为却只有化神九重，不过，只需两年，在下就能突破化神，迈入练虚境。”
<br/>　　慕清霄面色如常，但语气中却充满了无与伦比的自信。
<br/>　　话语刚落，整个大殿都死寂一片，针落可闻。
<br/>　　“此话当真？”涵素真人不怒自威，再一次询问道。
<br/>　　“千真万确。”
<br/>　　慕清霄自然清楚，后者是在试探他，那又何妨，自从他融合凤凰血脉后，年龄这种东西，根本就不能用在他身上。
<br/>　　虽然有些强者有一些秘法，能够测试出一个人的年龄，但慕清霄丝毫不惧。
<br/>　　哪怕当场测试又如何，能查探出来就有鬼了。

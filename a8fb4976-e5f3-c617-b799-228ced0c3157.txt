　　第516章
<br/>　　孟青玄看着白衣老尼的样子，无可奈何的叹了口气。他心中其实也明白，白衣老尼现在完全是靠一口气撑着，所谓故国旧梦，不过是一场虚幻，但却是撑着她活下去的理由。若把这场梦打破，她也便失去了活下去的理由。
<br/>　　故国逝去一百年，自己亡家灭口，逃到这大雪山上，现如今不人不鬼，半妖半魔。而白衣老尼执迷不悟，都不过是梦中的可怜人而已。
<br/>　　“若是杀人这种事，应该是东海那位更擅长，公主殿下为何不去找他？”孟玄青忽然问。
<br/>　　“东海太远了，一来一去，还不知要浪费多少时间。”白衣尼道：“况我这些年隐居不出，也与他断了联系，想找他怕是不太容易。”
<br/>　　“想找总是能找到的。”孟青玄念叨了一句：“我如今这般样子，公主殿下已是看到了。让我随你下山，怕这崎岖山路也能要了我的性命。不过，我有一物，送给殿下，莫说杀一人，便是屠一城也是不难。”
<br/>　　白衣老尼抬起头：“毒？”
<br/>　　孟青玄点了点头，他转身回到茅屋，捧出一口陶罐，又从屋里拿出一堆瓶瓶罐罐。
<br/>　　蜀中巫医，之所以可以扬名天下，不仅仅是医术，更令人畏惧的是他们的巫术。巫医巫医，巫总是在医前。传言中，巫医有与鬼神沟通的能力，请神御鬼，赶尸引魂。当然，传言也只是传言，能不能沟通鬼神不好说，不过巫医一脉的传人，的确都有不错的用毒功夫。
<br/>　　川蜀多密林，多瘴气，多的是闻所未闻见所未见的毒物。而这孟青玄就是使毒用毒的大行家。
<br/>　　很久以前，对头之所以要他全家老小的性命，便是忌惮他的使毒功夫，以他挚爱之人的性命逼他就范而已。
<br/>　　白衣老尼见他打开一黑色竹筒，轻轻弹了两下，从里面窜出一条雪白的银蛇。虽然只有小指粗细，但令白衣老尼觉得，可怖不亚于深山老林的怪蟒。
<br/>　　“公主小心，此物甚毒。”
<br/>　　白衣老尼点点头，纵然她一身出神入化的功夫，但此时也不敢大意。
<br/>　　孟玄青的十指飞快，又打开四枚竹筒，看他眼花缭乱的手势，当真不会令人觉得他是一个行将就木的老人。
<br/>　　陶罐内多了五种毒物，白色银蛇，血红蝎子，黑色蟾蜍，五彩蜘蛛，筷子长短的蜈蚣。
<br/>　　白衣老尼眼睛眯了起来，道：“斗毒之术？”
<br/>　　孟玄青点点头，道：“这雪山银色，幽泉紫蝎，瀚海蛤蟆，五彩蛛，铁线蜈蚣，这五种毒物，是我三十年来跋涉各地辛苦找到的，一直用毒物养着，培育不易，传到这一代，也就只养活了这五只，今日是为了公主殿下全拿出来了。”
<br/>　　如孟玄青这种人，对世间毒物有种莫名其妙的嗜好。白衣老尼不理解他这种偏执，正如孟玄青不懂得她为何偏执得想要复国，明明做不到的事情嘛。
<br/>　　但白衣老尼也明白，今日肯将这五种剧毒之物拿出来，孟玄青也是下了血本。
<br/>　　孟玄青紧盯着陶罐内毒物的缠斗，白衣老尼也凑过一只眼睛看。比起陶罐内的厮杀，倒是孟玄青脸上的表情更堪玩味。
<br/>　　每有一种毒物被杀死，他就露出心痛的表情，可双眼又露出炙热的光芒。
<br/>　　这五种毒物都是他辛苦收集而来，一代又一代的繁育，还不知付出了多少辛苦，今日因为白衣老尼，却要它们自相残杀。但同样，他也期待看到它们自相残杀，养出真正的毒王。
<br/>　　到此时，陶罐内的五毒残杀已快结束，最后是瀚海蟾蜍活到最后。它蹲在陶罐底部，肚子鼓起，发出哞哞的叫声，本来声音就极大，这陶罐又成了一扩音器。白衣老尼就觉得耳边嗡嗡作响。
<br/>　　在获胜之后，它开始吞吃其他毒物的尸体，本来通体黝黑，但现在却是黑得发红。虽然不过是只手掌大小的蛤蟆，但蹲在陶罐底部却是瘆人胆寒。
<br/>　　砰！
<br/>　　孟玄青用塞子将陶罐口堵住了，在陶罐内，蛤蟆还一声声叫着，陶罐嗡嗡作响。
<br/>　　“成了。”孟玄青双手捧着陶罐，递到白衣尼姑面前：“公主殿下捧着此物去，多少人也杀了。”
<br/>　　白衣老尼倒是不太敢接，她凭直觉也能知道这毒物的危险。愣了楞神，她问道：“怎么用？”
<br/>　　“此物乃是吞食五毒的毒王，将其埋入土中，方圆十里五谷不生，口中流出的毒汁，见血封喉。公主想要对付程大雷，只需将其投入水源中，方圆百里，人兽不存。”
<br/>　　“五谷不生，人兽不存？”白衣老尼顿了顿道：“灵嘛？”
<br/>　　“呃……”孟玄青被噎了一下，道：“大概灵吧，我师父就这样对我说的，此毒可以屠城，不过他也没能养出这样的毒物。公主殿下完全可以试试么，方圆百里屠一城可能有些够呛，但杀百八十人应该是没问题的。”
<br/>　　白衣老尼这才把陶罐接过来，对于蜀中巫医的本事她还是相信的，不然，她也不会辛苦走这一遭，来寻孟玄青。
<br/>　　“公主可是要小心，这毒不仅伤人，也能伤己。”
<br/>　　“明白，我不会碰它的。”
<br/>　　“那个……这种毒物一旦放出来，不知多少人殒命，为杀一个程大雷值得么？”孟玄青还是问出了心中的担忧，蛤蟆毒如果释放出来，便不是程大雷一个人死，整座蛤蟆城怕都要成为一座死城。
<br/>　　“程大雷的手下也都是一丘之貉，杀多少也是结束他们的罪孽。”白衣老尼的目光冷下来：“没有霹雳手段，怎养菩萨心肠。”
<br/>　　孟玄青默然无语，他已看得出来，这位白衣公主，怕从执迷不悟开始，已经走火入魔。
<br/>　　当然，他也不会多说什么，一生经历多少事，早已给他一颗铁石心肠，死上几万人，对他来说根本不算什么。
<br/>　　“那我就祝公主马到功成，山路难行，我就不远送了。”
<br/>　　白衣老尼捧着陶罐起身，看着他的模样，重重的冷哼一声。无论自己说什么，他都铁了心要当缩头乌龟。
<br/>　　不过，得到这屠城毒物，自己的目的也算是达到了。
<br/>(本章完)

        当他只看到她趴在桌子上时，吓得立马从电脑桌前起身，拉开房门跑了出去。
<br/>
<br/>        回头才想起自己没带钥匙，又疾步跑回房间去拿自己房间的钥匙。
<br/>
<br/>        明司寒焦虑不安地打开了房门，气喘吁吁地跑到书桌前刚想摇动连星儿的肩膀，却见她睡颜恬静，呼吸均匀，顿时松了口气。
<br/>
<br/>        原来，这丫头只是玩累了，睡着了。
<br/>
<br/>        怪不得他拿钥匙开门这么大的动静，她没听到。
<br/>
<br/>        明司寒不知道自己是怎么着，就这样站在这丫头的身旁，莫名其妙地盯着她看了一会儿，才俯身弯腰，将她从椅子上横抱了起来。
<br/>
<br/>        他将她轻轻地放在了自己的大床上，欲要起身之际，他的臂膀从她的肩下抽出时，不经意间地带动了她睡裙的肩带往下滑落。
<br/>
<br/>        雪白的香肩与半露的酥胸之间被性感的锁骨相隔，他不由自主地滚动着喉结，只觉眼前的这丫头格外秀色可餐，惹得他一阵口干舌燥。
<br/>
<br/>        心跳加速时，身体莫名其妙地有些燥热。
<br/>
<br/>        明司寒猛然回神后，下意识地闭了闭眼睛，又深吸了口气，不让自己胡思乱想。
<br/>
<br/>        但是他此刻却一直保持着弯腰俯身的姿势，明明脑子里在快速运转着指使自己的身体要远离这丫头才对。
<br/>
<br/>        可结果，双脚却不听使唤，像是被石化了般，定格在原地无法动弹。
<br/>
<br/>        明明要直起来的身子，此时却在慢慢地往下沉。
<br/>
<br/>        他的身影已经完全将她笼罩，薄唇凑到了她的粉唇之上。
<br/>
<br/>        仅仅只隔树叶那么薄的距离，他便努力地让自己停了下来。
<br/>
<br/>        他这是在做什么？
<br/>
<br/>        偷吻她吗？
<br/>
<br/>        为什么要偷吻她？
<br/>
<br/>        偷吻她之后呢？
<br/>
<br/>        明司寒此刻的脑子里有些混乱。
<br/>
<br/>        闭了闭眼后，他斗过了住在心底的那只色魔，伸手拉过一旁的薄毯，替这丫头盖上后，便转身离开了。
<br/>
<br/>        走出房间，明司寒深呼了口气，又缓缓地吸了口气。
<br/>
<br/>        还是外面的空气新鲜啊！
<br/>
<br/>        可是，他还是觉得身体很热。
<br/>
<br/>        他替她轻轻地关上了房门，回到东厢房后，便去冲了个凉澡。
<br/>
<br/>        可是，脑海里，那丫头的倩影却挥之不去。
<br/>
<br/>        而这凉水澡，他莫名其妙地冲了一个小时。
<br/>
<br/>        结果，这一整夜，他还是辗转难眠。
<br/>
<br/>        翌日，清晨。
<br/>
<br/>        连星儿是被明宅里的老管家给叫醒的。
<br/>
<br/>        明宅和傅家大庄园里的规矩差不多，早上七点，得准时陪长辈一起吃早餐。
<br/>
<br/>        昨晚玩游戏玩得太晚，连星儿坐在餐桌前有些无精打采，想打哈欠又不敢打，怕对明护国不礼貌。
<br/>
<br/>        从昨天到现在，连星儿就只见着了明护国，明家的老管家和三个佣人，而其他亲戚她是一个也没见着。
<br/>
<br/>        由此可见，这偌大的宅子里，明爷爷平日里一个人和家佣住一起，是有多冷清。
<br/>
<br/>        “老太爷，小少爷说他没睡好，还想继续睡。”老管家走进堂屋后回了话。
<br/>
<br/>        明护国也没多说责备的话，而是和蔼可亲地对连星儿说道：“那星儿，我们先吃吧！”
<br/>
<br/>        “好！”连星儿微笑着点了下头。
<br/>
<br/>        见明护国拿起筷子先吃了起来后，她才拿起勺子喝碗里的皮蛋瘦肉粥。
<br/>
<br/>        明护国看得出，连星儿这小丫头到底是出身于豪门世家，自身修养不错。
<br/>
<br/>        他也知道，这小丫头不愿意嫁进他们明家，但是她没有发大小姐脾气，把这事闹得不可开交。
<br/>
<br/>        她很尊重长辈，知书达理，是不错的孙媳妇人选。
<br/>
<br/>        同时，他这老头子也知道，他们年轻人看重的是自由爱情。
<br/>
<br/>        但是在他老头子看来，爱情还得讲究门当户对。
<br/>
<br/>        连星儿安安静静地吃完早餐后，明护国放下了碗筷，一脸慈祥地问道：“既然，爷爷和你的父母还有你曾爷爷一起吃过婚约同意席了，那么接下来就是要为你和小寒举行一场对外公开的订婚宴。”
<br/>
<br/>        “还得办订婚宴？”顿时，连星儿傻眼了。
<br/>
<br/>        明护国微笑道：“当然，这是规矩。过几日，我带着小寒的父母，将订婚礼金和订婚日期亲自上门送到你父母和你的曾爷爷的手中。”
<br/>
<br/>        “爷爷，要不，等我大学毕业了之后再订婚也不迟啊！”连星儿慌忙找借口推辞。
<br/>
<br/>        哪知道明护国想得比她更远：“现在你跟小寒订婚，等你大学毕业之后，就直接跟小寒领证结婚。”
<br/>
<br/>        “……”连星儿无言以对，更是一脸不情愿。
<br/>
<br/>        “爷爷，是你娶星儿，还是我娶星儿啊？”就在此时，明司寒那慵懒的声音从身后响起。
<br/>
<br/>        明护国顿时眉头一皱，带着一丝怒腔呵斥：“浑话！没大没小了是不是？”

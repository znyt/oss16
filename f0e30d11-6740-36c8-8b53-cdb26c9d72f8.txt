　　“既然伏羲一事已定，那吾等便来商量一下三皇之师一事，如何？”
<br/>　　元始看着女娲等人将伏羲之事商定，便开口说道。
<br/>　　太上闭目不语，此时已经恢复了淡然无为的模样，好似其他事情不关自己一样。通天手抱青色长剑，却是饶有兴趣的看着众人。
<br/>　　“全凭大师兄做主！”
<br/>　　女娲娘娘看了众人一眼，知道自己的想法其他人也未必会听，更何况，她是一个人，也没有什么弟子，争夺什么圣皇之事也没什么意思，还不如将此事交给元始，而且这也是元始希望的。
<br/>　　元始淡淡的点了点头，说道：“本座适才捻指一算，得知三皇五帝之一丝未来。”
<br/>　　这话一出，顿时就惊住了几人，他们只知道人族当兴，三皇五帝治世，并不知道是哪三皇，哪五帝？而元始却知道，这怎能不让人吃惊，要知道，他们也是圣人啊，圣人一念便知天下事，更可知未来五百年之事，但却是算不出三皇五帝之事，由此可知此事的艰难。
<br/>　　这一句话，更让众圣对于元始的实力有了一个大概猜测，虽然并不详细，但至少比他们强多了。
<br/>　　“哦？不知是哪三皇，哪五帝？”通天饶有兴味的看了众人一眼，问道。
<br/>　　“三皇者，乃天、地、人三皇。天皇者，定礼仪，衍吉凶；地皇者，驱病疾，种谷物；人皇者，又称泰皇，平天下，定八方！五帝者，即东皇青帝，南方炎帝，北方黑帝，西方白帝，中央皇帝！”
<br/>　　“嗯？”
<br/>　　此话一出，顿时就让极为圣人惊讶了，紧接着，他们便各自思量着，这些人族圣皇所对应的事迹，应该如何分配。
<br/>　　不等他人再说，元始便开口接着说道：“三皇之中，地皇所行之事，最适合二弟，二弟的丹药一道，对他十分有利，是故，地皇便交由二弟了。”
<br/>　　“善！”
<br/>　　太上思索一番，觉得没有什么不妥，而且有一个能够成为圣皇的传承者，也没什么不好的，便答应了。
<br/>　　元始点了点头，继续说道：“人皇倒是与我玉虚一脉有些缘份，便交由我阐教来教导了。至于天皇，那便交给三弟你了。”
<br/>　　“可以！”
<br/>　　通天点了点头，觉得此事可行，三皇之师他们三兄弟包了，也没有侵犯道到其他人的利益。
<br/>　　“五帝之中，黑帝与我玉虚一脉有缘，至于其他的，你们两人各自选择一人，剩下两帝，我等也不必争了，便交由人族自己处理。如何？”
<br/>　　元始也不想为了争夺一个圣皇之师而闹得三兄弟不和，交由人族自己处理更好，毕竟，人族之中，也不缺少高手。更何况，这样还可以结人族一个善缘。
<br/>　　“善”
<br/>　　两人自然是没有意见，全部不同意了。至于女娲，全程都没有插上一句话，对于她来说，她只关心自己的兄长伏羲，她也没有什么传人，也不想找什么传人，也就没有什么兴趣来争抢什么圣皇之师了。
<br/>　　“可以。”
<br/>　　武祖玄天也是自无不可，更何况，就算他不同意，又能有什么其他的意见？圣人决定的事，也不是他一个转轮强者能够左右的。
<br/>　　“人族三皇五帝之事，乃是吾东方之事，西方不能插手，各位以为如何？”
<br/>　　元始突然轻声问道，说出这样一句话。他的意思就是，此事要将西方二圣排除在外。
<br/>　　“大善！”
<br/>　　对此，不管是三清，还是女娲、玄天，都没有什么意见，他们对于西方二圣也是没有好感，自然也不希望他们插手人族之事。
<br/>　　“若是接引与准提偏要插手呢？”通天教主突然问道。
<br/>　　此话一出，其他几人也将目光投向元始，想看他怎么回答，毕竟这是关系到圣人的事，关乎圣人的事情，那就不是小事。
<br/>　　“若真是发生那等事情，我等自是联手一致对外！到时候，就休怪本座手下不留情了。”
<br/>　　元始看了一眼通天，面色不变，淡淡回答道。声音虽然淡然，却透露着一股决绝之意。
<br/>　　众人心中一惊，之后顿时为西方两位圣人默哀，招惹到元始，他们就别想好过了。
<br/>　　……
<br/>　　这一日，女娲圣人一步来到幽冥界中，圣人圣威，顿时便惊动了幽冥界中的三生大帝与平心娘娘，平心娘娘眼神闪烁一阵，便没有动作，依旧坐在平心殿中，而三生大帝却是不能不出现。
<br/>　　“见过女娲道友，道友来此，可否是为了那伏羲之事？”
<br/>　　三生大帝一见女娲，便行了一礼，毕竟，他虽然为地府之主，但是地位还是要弱女娲这种天道圣人一筹。
<br/>　　“不错！”
<br/>　　女娲微微有点诧异，不知这三生道人是如何知道的，她圣念微微一探，便知道这三生道人是准圣圆满高手，而且，全身上下透露着一种时光命运的气息，让她看之不透，她顿时一惊。
<br/>　　要知道，她可是圣人，就连圣人都看不透一个人，要么就是有什么至宝遮掩住了天机，有么就是他本身有什么特殊的地方，不管是哪一种，都说明了此人的不凡。她顿时就收起了轻视的心态，重新面对三生大帝。
<br/>　　“道友请跟我来！”
<br/>　　三生大帝微微一笑，自是感应到了女娲的查探，不过他毫不在意。
<br/>　　接着，他便带着女娲走过向着六道轮回之处走去。突然，女娲停了下来，看着眼前不远处的一片妖异的鲜红，红光弥漫，有一种淡淡的哀伤，仿佛鲜血铺成一片。
<br/>　　那是一片花海，花如龙爪，具鳞茎，形如洋葱头，叶丛生，细长尖端，只有花，没有叶，花叶永不见。
<br/>　　彼岸花！又称死人花，地狱花，幽灵花，黄泉之花，死亡之花等，是一种只开在阴间的花朵。
<br/>　　彼岸花，花开一千年，花落一千年。
<br/>　　花叶生生相惜，世世永不相见。
<br/>　　彼岸花，开彼岸。
<br/>　　只见花，不见叶。
<br/>　　彼岸花开开彼岸，奈何桥前可奈何。
<br/>　　前生今世因果散，三生石上留姓氏。
<br/>　　愿殇心殇情亦殇，花叶飘零永不见。
<br/>　　“彼岸花，开彼岸，花叶飘零永不见……彼岸在哪里，哥哥，千年之后，你还是你吗？”女娲突然有些发愣，眼神之中闪过点点思念，心神一阵彷徨，她不知道未来的命运会如何，她只能随波逐流。
<br/>　　“我们走吧。”
<br/>　　不知过了多久，女娲恢复了常态，便自行向前走去，却是不愿在此停留片刻。
<br/>　　之后，女娲踏过黄泉，见过忘川河，踏上奈何桥，见到了三生石，又是一阵神伤，女娲将伏羲投入了六道轮回之人道。
<br/>　　当然，孟婆汤是一定要喝的，毕竟，孟婆可是平心娘娘的一道分身，谁也不能坏了这里的规矩，就算是圣人都不行。在幽冥界中，平心就相当于洪荒之中的道祖鸿钧，谁也不敢招惹。
<br/>　　“告辞！”
<br/>　　一番折腾后，女娲也不停留，直接消失在幽冥界中。
<br/>　　阴风吹过，寒气弥漫，虚空之中，仿佛传来一阵幽幽的叹息声：
<br/>　　“彼岸花，开彼岸……”
<br/>(本章完)

        一家浪漫又奢华的法国餐厅里，法国主厨现场烹饪美食，小提琴师拉起了一首《月亮代表我的心》。
<br/>
<br/>        刚巧，落地窗外的无垠天际，挂起了一弯皎月。
<br/>
<br/>        虽然是月明星稀的夜晚，但城市里的华灯初上，不似星光，却似众星捧月般华丽。
<br/>
<br/>        时隔十一年，两人再次见面时，李忆莎发现，傅夜沉早已不是她记忆中的那个翩翩少年。
<br/>
<br/>        是的，他初中毕业后，她就再也没有见过他了。
<br/>
<br/>        但是，她却一直喜欢着他，整整十四年。
<br/>
<br/>        她曾听冷爷爷提起过，傅夜沉在念初中之前，一直跟着他的父亲过军旅生活。
<br/>
<br/>        他从小就有好几个各科贴身私教，所以小学几乎是在部队里念的，没有去过一次正规的小学学校上学。
<br/>
<br/>        傅夜沉从小就会组装枪支，近身格斗，而且数理化和外语的成绩特别好。
<br/>
<br/>        到了初中，傅炎廷才安排傅夜沉在临海城念书，才与她李忆莎同一个学校。
<br/>
<br/>        后来，初中毕业后，他来了滨江城念高中，随后高中念到一半又去念了军校，军校毕业后又去当了兵。
<br/>
<br/>        他长得很帅，棱角分明的脸上五官深邃，一双斜飞入鬓的剑眉英气逼人，鼻梁笔直高挺、瞳眸漆黑如墨，整个人看起来温润俊逸，气质高雅中透着王者风范。
<br/>
<br/>        “李忆莎小姐，你好，我是傅夜沉。”
<br/>
<br/>        李忆莎朝这边走过来后，傅夜沉非常有绅士风度地替她拉开了椅子，待她坐下后，才回到自己的座位上，温文儒雅地向她自我介绍。
<br/>
<br/>        其实，这都不是他的本性。
<br/>
<br/>        本性上的他来说，不是自己爱的女人，他不会予以过多的理会。
<br/>
<br/>        但时至今日，他也已经到了没办法的地步。
<br/>
<br/>        筱颖和阿景已经结婚生子，爷爷也已经年事已高。
<br/>
<br/>        他必须学会忘记筱颖，重新开始自己的另一段新的感情。
<br/>
<br/>        所以，傅夜沉努力让自己迈出这第一步。
<br/>
<br/>        两人面对面坐着，话题并不多。
<br/>
<br/>        李忆莎一脸苦涩地提起了十四年前的往事：“情人节那天，我鼓足勇气向你告白，结果被你给无视掉了，真的是太伤我自尊心了。”
<br/>
<br/>        “我以为你是在向我弟弟表白。”傅夜沉云淡风轻地说。
<br/>
<br/>        他不大记得那天的情形了。
<br/>
<br/>        反正，他只知道，确实是有个女生，红着脸，低着头，将一封情书和一盒巧克力递到了他和傅昼景的中间。
<br/>
<br/>        傅夜沉以为这女孩是在向他弟弟傅昼景表白，自然是非常识趣地绕道离开，难道这也有错？
<br/>
<br/>        更何况，在他傅夜沉看来，当年在学校里，他的弟弟傅昼景好像比他更得女生的欢心，一到课后，就有一大堆的女生围绕在他弟弟傅昼景身边。
<br/>
<br/>        而殊不知，是他傅夜沉性格太冷，让那些爱慕他的女生不敢轻易靠近。
<br/>
<br/>        听到傅夜沉这样的解释，李忆莎险些哭笑不得。
<br/>
<br/>        “那么，我现在，还有机会吗？”李忆莎微笑着打趣道。
<br/>
<br/>        曾几何时，他也问过筱颖这类似的问题，但是筱颖当场就毫不留情地直接拒绝了他。
<br/>
<br/>        筱颖的事，他想通了后，傅夜沉也很希望自己能从筱颖的阴影里走出来，但是他却不想去利用其他人的感情来为自己疗伤。
<br/>
<br/>        “我已经心有所属。”傅夜沉非常坦白地回答道。
<br/>
<br/>        李忆莎却淡定地回答道：“但那个女孩不爱你，对吗？”
<br/>
<br/>        来之前，冷爷爷已经事先告诉过她，傅夜沉爱着一个不爱他的女人。
<br/>
<br/>        “我爷爷告诉你的？”傅夜沉明知故问，但间接地看来，爷爷真的是为他的感情操碎了心。
<br/>
<br/>        李忆莎点了点头，释然地微笑道：“我不介意。我们可以先试着相处一段时间，只希望你在这段时间里不要抗拒我。如果，在相处了一段时间后，我仍旧无法做到让你爱上我。那么，你就当这段时间，我们只是普通的朋友关系。”
<br/>
<br/>        这句话，也是他爷爷傅炎廷的原话。
<br/>
<br/>        爷爷说，让他去尝试着与别的女人相处，如果实在无法忘记筱颖，仍旧无法去接受别的女人，他这当爷爷的，就不再苦苦相逼，哪怕他打一辈子的光棍，他也不管了。
<br/>
<br/>        但前提条件是，他傅夜沉必须迈出第一步。
<br/>
<br/>        他得先从自己那封闭已久的空间里出来，才能尝试着去接受另一个新的空间。
<br/>
<br/>        爷爷的苦口婆心，用心良苦，傅夜沉终究还是答应了。
<br/>
<br/>        “我尽量。”傅夜沉淡淡地应了声。
<br/>
<br/>        除了这三个字，他没有其他话可说。
<br/>
<br/>        但他说的“我尽量”，不是指的尽量去爱上她李忆莎，而是尽量去忘了对筱颖的爱。

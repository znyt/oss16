　　嘭…
<br/>　　原地，一声剧烈爆响。
<br/>　　萧逸的身影，忽然被重重轰飞。
<br/>　　地面，被拖出一条长达百米的深陷痕迹。
<br/>　　百米之外，扬起一大片灰尘。
<br/>　　待得灰尘散尽，露出了里头萧逸极其狼狈的身影。
<br/>　　“咳咳。”萧逸咳嗽了两声，嘴角溢着的一丝鲜血，证明着他刚才受了不轻的伤势。
<br/>　　萧逸摸了摸胸膛处，那里，一是一片凹陷。
<br/>　　刚才与这头荆棘圣蜥的交锋瞬间，仅仅是被它重重一撞，他已被撞飞。
<br/>　　此刻，五脏六腑仿佛移位了一般。
<br/>　　荆棘圣蜥，看似身影不如夜影流光蟒那么庞大，但其蕴含的力量，却绝对是夜影流光蟒的十倍之上。
<br/>　　这等妖兽，本就擅长巨力。
<br/>　　加之荆棘圣蜥，又是土属性妖兽，脚踏于地面，便能源源不断借助大地之浑厚土属性力量，实力陡增。
<br/>　　若非这等妖兽比较笨拙，速度不快的话，恐怕已是圣皇境妖兽中有数的霸主级妖兽之一了。
<br/>　　再加上这些金光力量，怕是武道大能在此，都得忌惮三分。
<br/>　　“呼。”萧逸深呼吸一口气，一粒丹药吞入腹中。
<br/>　　丹药之力，快速在体内生效，修补着疼痛的五脏六腑。
<br/>　　手中冷焱剑，紧紧地握了握。
<br/>　　一双冷傲的双眸，直视面前两头妖兽。
<br/>　　两头妖兽，亦泛着凶光，凝视着妖兽。
<br/>　　在它们看来，它们联手之下，尚且奈何不了这个年轻人类武者，这个年轻人类武者，绝非善茬。
<br/>　　妖兽向来谨慎，特别是金光险地这里的妖兽，无比的谨慎中又带着狡猾。
<br/>　　故一时并未有异动。
<br/>　　萧逸也暂时没有动作。
<br/>　　如无意外，今晚，将是一场苦战。
<br/>　　不过，也恰合他意。
<br/>　　他正思索着如果应付。
<br/>　　双方，就这般对峙着。
<br/>　　半晌，夜影流光蟒与荆棘圣蜥，率先有了动作。
<br/>　　妖兽终究是妖兽，凶性十足。
<br/>　　嗖…一道流光，疾速朝萧逸袭来。
<br/>　　轰…轰…轰…
<br/>　　荆棘圣蜥的脚步，也已迈动；巨大的力量之下，每次异动，必是地动山摇。
<br/>　　当然，那般速度也极快。
<br/>　　所谓的看起来笨拙，速度不快，不过是对比萧逸而言罢了。
<br/>　　锵…
<br/>　　萧逸一剑劈出，星光长河倾泻而出，霎时挡下了率先而来的流光。
<br/>　　一挡过后，荆棘圣蜥的身影，堪堪而来。
<br/>　　巨大的力量，朝萧逸重重撞击而来。
<br/>　　萧逸未挡，这头孽畜的巨大力气，他已经感受过了。
<br/>　　脚步一退，瞬间拉开了距离。
<br/>　　萧逸的眉头，忽然皱了起来。
<br/>　　荆棘圣蜥的速度，看似缓慢，但它终究是圣皇境妖兽。
<br/>　　夜影流光蟒化作流光，速速极快地朝他而来，他必须挡之。
<br/>　　便是这一挡，已足够给荆棘圣蜥靠近的时间。
<br/>　　这两头妖兽，一为速度极快，二为速度慢却力量惊人。
<br/>　　二者的配合，竟是如此默契。
<br/>　　若是单打独斗之，萧逸尚且压力不小。
<br/>　　现今以一敌二，而且这二者配合无间下，压力更是抖增。
<br/>　　嗖…夜影流光蟒再次袭来。
<br/>　　萧逸一边仗剑挡之，一边脚步后退躲避荆棘圣蜥。
<br/>　　同时，他也暗暗思索着应对之法。
<br/>　　嗖…嗖…嗖…
<br/>　　萧逸的身影，不断变化着。
<br/>　　空气中的嗖嗖声，忽然变得剧烈。
<br/>　　或是流光之声，或是剑芒之声，或是萧逸脚步之声。
<br/>　　便是在这般僵持之下，萧逸眉头紧皱着。
<br/>　　半晌，萧逸眉头一松。
<br/>　　“夜影流光蟒，速度极快；荆棘圣蜥，力量惊人，又防御力强悍。”
<br/>　　再这般下去，怕是这两头妖兽耗也能耗死自己。
<br/>　　当然，如果放出雾妖，很轻松就能结束战斗。
<br/>　　可这是他的历练，他需得自己应付之。
<br/>　　战斗虽困难而危险，却也是他快速提升实力的途径之一。
<br/>　　“喝。”萧逸低喝一声。
<br/>　　原本的脚步快速移动，忽然一滞。
<br/>　　八万星光，瞬间降临，随后化作剑阵，困住了夜影流光蟒与荆棘圣蜥。
<br/>　　嗖…嗖…嗖…
<br/>　　天空中的星光之力，不断降临，源源无尽。
<br/>　　八万星光，一层接一层。
<br/>　　星幻剑阵，亦是一阵接一阵。
<br/>　　若是单单一道星幻剑阵，怕是这两头孽畜一个冲撞就能撞开了。
<br/>　　但十数道剑阵之下，它们即便能破开，但也绝非易事。
<br/>　　剑阵布下，萧逸手中冷焱剑脱手而出。
<br/>　　剑身，直直冲入剑阵中央。
<br/>　　以中品圣器冷焱剑为阵心，大阵之坚固，陡然剧增。
<br/>　　冷焱剑，可是接近于上品圣器，远超其它中品圣器之威力。
<br/>　　随后，萧逸双手齐出。
<br/>　　体内元力，极限调动。
<br/>　　双手吸摄之下，天空中，八万星光，一次又一次不断降临。
<br/>　　这时，降临的星光，未再化作剑阵，而是化作星光漩涡。
<br/>　　一个个由八万星光幻化而成的星光漩涡，散发着惊人的绞杀之力。
<br/>　　夜影流光蟒，虽表皮光滑，但滑得过无数星光？滑得开惊人绞杀之力？
<br/>　　不过片刻，夜影流光蟒已剧烈暴走，强行冲撞着星幻剑阵。
<br/>　　不过，由冷焱剑加持下的星幻剑阵，坚如磐石，岂是它能撞开的。
<br/>　　渐渐的，夜影流光蟒，发出声声痛呼嘶吼。
<br/>　　星光的绞杀之力，正不断绞杀着它的肉体，以及生机。
<br/>　　同一时间，剑阵之内的荆棘圣蜥，虽防御力惊人，绞杀之力一时奈何不得它。
<br/>　　但它同样破不了星幻剑阵。
<br/>　　剑阵，由十数个八万星光而成的剑阵叠加而成，巨大无比。
<br/>　　荆棘圣蜥想要冲撞之，却每每被之后凝聚幻化的星光漩涡所挡下。
<br/>　　星光漩涡之内，除了惊人的绞杀力外，还有恐怖的吸扯力。
<br/>　　一个星光漩涡，拉不住它的惊人力量。
<br/>　　那十个，二十个呢？
<br/>　　二十个星光漩涡，完全吞没了荆棘圣蜥。
<br/>　　一个时辰后，强悍的星光绞杀之力，渐渐将它的坚硬荆棘绞杀破开。
<br/>　　此时，萧逸的脸色，已然微微煞白。
<br/>　　他并未借用丹药或者灵脉元力的增幅。
<br/>　　此刻，是他实打实地极限操控着星光。
<br/>　　体内庞大气泉中的一身澎湃元力，在他手中，利用得淋漓尽致。
<br/>　　又是十数分钟后。
<br/>　　荆棘圣蜥的荆棘，已被绞杀殆尽。
<br/>　　剩下的，也不过是它在星光漩涡中的痛苦嘶吼。
<br/>　　“呼。”萧逸见状，擦了擦额头汗水，轻呼出一口气。
<br/>　　这一次，想必终于可以微微松口气了。

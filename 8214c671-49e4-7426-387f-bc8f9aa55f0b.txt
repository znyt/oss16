　　过了四季城，便是裂天剑派三百里范围之内。
<br/>　　萧逸早已摘掉了幽魂面具，也换掉了以往的一身黑袍，变回了以往在紫云城时的穿着。
<br/>　　一身贵公子服，样貌帅气不凡，看起来像极了放荡不羁的公子哥。
<br/>　　一路奔跑，三百里范围内，是一片辽阔无边的森林，野兽、鸟虫不少，妖兽却半只都没有。
<br/>　　路上，也不见有半个人烟。
<br/>　　萧逸顿觉奇怪，暗道，“今日不是裂天剑派开山收弟子之日吗？怎么半个鬼影都见不到。”
<br/>　　半晌，萧逸摇摇头，懒得多想。
<br/>　　数个时辰后，当他看到前方一座座恢宏非凡的宫殿时，才停下了脚步。
<br/>　　那些宫殿，远远看去，宛若自成阵势，威武不凡，气势逼人。
<br/>　　恍惚间，竟觉那是一把神兵利剑，剑身冲天而起，剑尖直插云霄，仿佛要将这天地裂开两半。
<br/>　　“不愧是裂天剑派，我们北山郡的武道圣地。单单远远观看，还未进入，已经感觉到它的惊天锋芒。”
<br/>　　“偌大个剑派，在这辽阔森林中，倒是山清水秀，人杰地灵。”
<br/>　　萧逸蓦地一笑，对于即将加入的裂天剑派充满了期待。
<br/>　　.....
<br/>　　裂天剑派山门外，早已聚集了数千前来参加考核的武者。
<br/>　　更前方，是数十个一身玄色衣服的老者，清一色洞玄修为。
<br/>　　这身衣服，只有裂天剑派内的执事才有资格穿。
<br/>　　萧逸顿时恍然，路途不见半个人烟，是因为其他武者早就来了。
<br/>　　裂天剑派开山收弟子之日，何其重要，等闲武者，岂敢怠慢，大多在昨日便已经在山门外等候了。
<br/>　　也就是他，姗姗来迟。
<br/>　　只见那些执事头也不抬，在桌案前翻翻写写，为前来参加考核的武者做着登记。
<br/>　　一个个武者做好登记，进入裂天剑派。
<br/>　　原本的数千人，很快只剩下寥寥数百人。
<br/>　　当轮到萧逸时，那些执事仍旧头也不抬，平淡而毫无感情地问道，“姓名、年龄、武魂、修为。”
<br/>　　萧逸一愣，如实回答道，“萧逸，16，控火兽，先天三重。”
<br/>　　“嗯。”执事机械性地点着头，做着记录，然而，当写到控火兽和先天三重时，却蓦然地顿了顿，更破天荒地抬起了头。
<br/>　　执事面露疑惑之色，当看到萧逸略显稚气的帅气脸庞时，眉头更是一皱。
<br/>　　萧逸问道，“执事为何如此看我？”
<br/>　　那执事疑惑道，“少年，你确实不是在耍老夫？”
<br/>　　“执事此话何意？”萧逸一愣。
<br/>　　“你16岁，我看出来了，你先天三重的修为，我也看出来了，但你控火兽的武魂...”
<br/>　　“少年，那种呆呆傻傻最垃圾的控火兽武魂，也能让拥有者修炼到先天三重？”
<br/>　　“额。”萧逸一愣，随后苦笑一声，凝聚出了自己的控火兽。
<br/>　　武魂，在武者体内，外人看不到，也感知不出。
<br/>　　只有武者自己凝聚出来，外人才能通过武魂的样貌，判断出是什么武魂。
<br/>　　当然了，别的武者，凝聚武魂时，就代表着实力才开，要借助武魂的增幅来增加战力。
<br/>　　而萧逸的控火兽，什么增幅都没有，只会控火。
<br/>　　所以一直以来，他从凝聚过控火兽出来。
<br/>　　这一次，到裂天剑派是学艺修炼的，尊师重道四字，萧逸还是懂的。
<br/>　　自然凝聚出了武魂，也用了真实姓名。
<br/>　　这时，那执事看着萧逸手中凝聚出来的，呆呆傻傻的控火兽，顿时一惊，“还真的是控火兽武魂。”
<br/>　　一旁，其他准备登记的武者‘噗哧’一声笑了出来。
<br/>　　“控火兽，那小子竟然是传闻中最垃圾的武魂。”
<br/>　　“别笑嘛，那好歹是最稀有的武魂。”
<br/>　　“哈哈，是阿，最稀有。正常人根本不会觉醒出那种武魂，当然稀有。只有万中无一的傻蛋才会觉醒出来。”
<br/>　　一旁的武者得意地笑着，赶来裂天剑派参加考核的武者，天赋都必然不弱，武魂也不差，自觉比萧逸高等许多。
<br/>　　萧逸眉头一皱，但也懒得计较。
<br/>　　反倒是那执事，重重地呵斥了一声，“放肆。”
<br/>　　“还未通过考核，便已经如此盛气凌人。若真正加入裂天剑派，他日岂不是目无尊长，肆意妄为？”
<br/>　　那执事瞬间发怒，让得萧逸包括那几个在嘲笑的武者顿时一愣。
<br/>　　这时，其他地方负责登记的执事，竟纷纷站了过来，躬身道，“易老，息怒。”
<br/>　　面前这老者，显然有别于其他执事，身份更高。
<br/>　　那执事摆摆手，道，“罢了。”
<br/>　　而后交给萧逸一个令牌，道，“少年，这是你的令牌，进去吧，等下会有其它派内执事安排你的考核。”
<br/>　　“是。”萧逸拱拱手，接过令牌，转身走入了裂天剑派。
<br/>　　.....
<br/>　　半个时辰后，山门外的数千武者都做好了登记，一一进入裂天剑派。
<br/>　　这时，数个执事出现，将一众人带到了一处宽阔的广场。
<br/>　　广场上，竟早有上万人在等待。
<br/>　　大部分来此参加考核的武者都一早就来了。
<br/>　　萧逸和那数千武者，算是最后一批。
<br/>　　不过，山门外的执事，还是一直等到了下午时分，太阳落山，才结束了这一天的开山收弟子之日。
<br/>　　.....
<br/>　　入夜时分，一众人却又再次被带到山门之外。
<br/>　　数个执事站在万多名武者前，高声道，“开山之日，已经结束。你们，虽然做了登记，却还不算裂天剑派的弟子。”
<br/>　　“现在，你们要经历第一个考核，名为适者生存。”
<br/>　　在真气的增幅下，声音清晰传入了众人耳中。
<br/>　　“适者生存？”不少武者都在议论纷纷，疑惑至极。
<br/>　　“静。”执事呵斥一声，顿时盖住了所有喧嚣声音。
<br/>　　“裂天剑派，只收天才，不要庸才。”
<br/>　　“由今夜开始，你们要在这方圆三百里的森林范围内，待上一个月。”
<br/>　　“你们的令牌，作为积分标准。每个令牌，内含一分，通过彼此抢夺，可增加分数。”
<br/>　　“一月之后，按分数作出排名，前五百者，方能加入裂天剑。”
<br/>　　执事声音铿锵，快速地说了一遍。
<br/>　　然而，万多名武者却同时脸色大变，并警惕地看着身边旁人。
<br/>　　“彼此抢夺？”
<br/>　　“分数前五百才能进入裂天剑派？”
<br/>　　所有人都懂了，难怪这第个考核名为适者生存。
<br/>　　萧逸也是眉头一皱，整个北山郡，人口数亿，如今聚集在这的万多名武者，便是来自全郡各地的青年才俊。
<br/>　　而最后真正有能力通过考核的，也必然是出色的天才，每一个都不简单。
<br/>　　万多名天才，就在这森林内进行一场为时一月的互相争夺。
<br/>　　毫无疑问，对于在场任何一人来说，这都将是一场激烈的比拼。
<br/>(本章完)

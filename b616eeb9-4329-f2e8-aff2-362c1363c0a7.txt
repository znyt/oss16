　　第264章
<br/>　　戎族大军压来，觉得谁都可以杀得，反正都是帝国人；马匪觉得谁都可以杀得，反正我们是土匪；鱼龙卫觉得谁都可以杀，反正都是蛮夷。江湖群雄觉得谁都可以杀，不过，先要保证自己活着。
<br/>　　而在如此复杂的战斗中，所有人都有一个共同的想法：程大雷是可以杀的，且是一定要杀的。
<br/>　　骑兵对冲，长枪铁刀，一时间方圆五里之内，成了一个杀戮场。
<br/>　　程大雷是谁也不想杀，他想要逃命。
<br/>　　而在乱战中，有一队人格外特殊些。他们人数很少，只有十几人而已，兵器也千奇百怪，有使狼牙棒的，有使月牙铲的，有使雷公锥的……看他们的兵器便知道，这些人既不属于戎族也不属于鱼龙卫。他们，当是属于江湖人。
<br/>　　这一队人斜着冲过来，毫无阵型可言，却也没人能挡住他们的前进的步伐。一路冲过来，别人也不问，只是迫向马车。
<br/>　　马车上是程大雷。
<br/>　　他们就是为杀程大雷而来。
<br/>　　虽然每个人都想杀掉程大雷，但至少也得顾及身边的敌人，如他们这般直愣愣的冲杀而来，就显得有些特殊了。
<br/>　　程大雷挥舞着斧头格杀，在这个时候，也无所谓什么招数，只要力气够大，速度够快，就是最强的招数。
<br/>　　好在程大雷对于这些事情也不算陌生，他已经习惯这个世界的生存方式，青叶佛说得不错，这就是一个人人争而相食的世道，没有舞文弄墨，颂佛念经之人的存身处，能够活下去的只有魔头。
<br/>　　嗵！
<br/>　　一柄方便铲向程大雷砸来，使铲的是一个满脸络腮胡子的汉子，阅读他的信息：卢浮生，顶级游侠。
<br/>　　程大雷挥斧头砸开，紧跟着一狼牙棒挥向程大雷，程大雷一斧将对方的马头占掉，骏马轰然倒塌。
<br/>　　张亿靼，顶级游侠
<br/>　　……
<br/>　　这些是崔相的人。
<br/>　　程大雷防得很狼狈，他现在不是自己要逃出去，还要护着马车上的李婉儿突围。这些人在见程大雷不好攻破时，已经开始攻打马车，程大雷不仅要保护自己，更大的精力还在保护马车上。
<br/>　　现在程大雷觉得自己就像在做《三国无双》中的马车任务，关二爷护送马车破无关斩六将，此情此景，何其相似。
<br/>　　可这难度……明显是地狱级的。
<br/>　　想到这里，程大雷心中一动，是啦，自己还有一个游戏道具没用。
<br/>　　上次杀掉鬼见愁时，系统奖励了一支战曲，如果不是今天情况危急，程大雷还真的很难想到。
<br/>　　只是……这战曲应该如何使用？
<br/>　　“喜羊羊，美羊羊，懒羊羊……”
<br/>　　咦，好像没什么用啊。
<br/>　　“你是我的小呀小苹果……”
<br/>　　好像……也没卵用。
<br/>　　其他人也都看懵了，这么残酷的战斗，他怎么打着打着哼起了小曲，而这曲调，感觉那里都不舒服。
<br/>　　“特么地，到底有得玩没得玩，你怎么还不播放。”
<br/>　　嘟，战曲使用，持续时间一炷香。
<br/>　　咦，原来是声控的。
<br/>　　紧跟着，悠扬宏大的声音在程大雷脑海中响起，那声音宛若九天风动，四海翻浪，山川崩塌。
<br/>　　程大雷体内宛若注入一股力量，瞬间热血技能双倍增益，体力增益50%，敏捷增益50%，回血能力增益45%。
<br/>　　程大雷立在马车上，忽然仰天大笑，铺天盖地的威压席卷而出，他看周围人都有一种尔等渣渣皆不可一战的感觉。
<br/>　　普通，优秀，顶级，绝世，传说是人物的四个等级，程大雷自从出世至今，见到的绝世人物不过一人而已，而起还是个戏子。至于绝世武者，程大雷一个也没碰过。
<br/>　　何谓绝世，绝世就是天下之大，寰宇之内，你都是站到山尖的人物，你已经走到这个世界可以达到的巅峰，再进一步，就可以留名青史，与那些传说中的名将平起平坐。
<br/>　　现在借助战曲，程大雷踏入绝世，虽然只有一炷香。
<br/>　　不是我多努力，也不是我多有天赋，就是开挂，就是强。
<br/>　　一个挥着大锤的汉子向程大雷砸来，他的速度不可谓不快，力量不可谓不大，但老子曰的好：故有无相生，难易相成，长短相形，高下相倾，音声相和，前后相随，恒也。
<br/>　　和程大雷相比，速度太慢，力气太小。
<br/>　　简单来说就是：和我比，你差得远呢。
<br/>　　程大雷用手接住了他的铜锤，大汉一怔，卖力挣扎，被程大雷握住的铜锤却不动分毫。
<br/>　　这铜锤总重一百八十斤，向来只有自己能使用，但现在，程大雷只用一只手就接住了。他惊愕的抬起头，目睹程大雷璀璨的笑容。
<br/>　　是强者的嘲笑，颤抖吧，你们这些渣渣，在强者面前只能颤抖。
<br/>　　别太放肆，只有一炷香。
<br/>　　程大雷暗自叮嘱自己，劈手将这铜锤夺了过来，单手在空中一丢，稳稳接住了锤柄。
<br/>　　马车周围的人都愣住了，一百八十斤的大锤，他可以单手抛在空中，还能轻松接住。
<br/>　　这是人能有的力气么！
<br/>　　看着马车上的程大雷，大家心底都升起一股惧意。
<br/>　　只见程大雷站在马车上，左手握着一百八十斤的铜锤，右手是一百单八斤的鬼面斧，在战曲的增益下，程大雷忽然有了大圣当初在龙宫的感慨：太轻了，不趁手啊。
<br/>　　两柄重武器轮开，程大雷就是一台人型教绞肉机，挨着死，擦着伤，无人敢靠近他的身周。
<br/>　　若仅仅是力气大也就罢了，关键是他的速度还很快，身体像矫健的白猿，穿上跃下，登高伏底，动作那个灵便，瞧瞧，好嘛，他在空中翻了个跟头。
<br/>　　以马车为中心，散开一个十米左右的空地，程大雷驾着马车向外冲。
<br/>　　“拦住他，拦住他！”
<br/>　　也不知道是哪家势力高声喊着，可这个时候谁敢靠前，大家都下意识往后退。
<br/>　　“放箭，放箭！”
<br/>　　“公主还在车上！”
<br/>　　“顾不得了。”
<br/>　　恰就在此时，只见前方突然杀出一哨人马，约莫有上千人，一同奔来，狼烟动地，这些人不要命般的往前奔，口中高呼着同样一句话。
<br/>　　“大当家在哪里，大当家在哪里！”
<br/>　　徐神机，秦蛮，高飞虎，高飞豹，熊大熊二，刘关张，赵子龙……
<br/>　　该来的，都来了。
<br/>
